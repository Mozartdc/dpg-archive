<p style="margin: 0 0 1em 0;">서양 음악에서는 A부터 G까지 일곱 개의 알파벳으로 음을 구분함. 음자리표를 솔(G) 음을 기준으로 그려서 G clef때 알파벳이 같으면 옥타브가 달라도 같은 음 이름으로 부름.</p>

<p style="margin: 0 0 1em 0;">음자리표와 음역</p>

<p style="margin: 0 0 1em 0;">음자리표는 오선의 각 줄과 칸이 어떤 음 이름에 해당하는지를 정해 주는 표시임. 피아노는 음역이 넓어서 높은음자리표와 낮은음자리표를 함께 사용함.</p>

<p style="margin: 0 0 1em 0;">높은음자리표
높은음자리표는 주로 오른손이 연주하는 높은 음역을 표기할 때 사용함. 음자리표를 솔(G) 음을 기준으로 그려서 G clef 라고도 함.</p>
