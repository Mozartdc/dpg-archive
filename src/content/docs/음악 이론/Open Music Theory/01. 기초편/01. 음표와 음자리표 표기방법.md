<p style="margin: 0 0 1em 0;">음높이와 리듬은 서양 음악 기보법에서 매우 중요한 요소임. 두 가음높이는 수직 방향인 Y축에 표기하고 리듬은 수평으로 X축에 표기함.</p>

<p style="margin: 0 0 1em 0;">악보를 읽을 때는 왼쪽에서 오른쪽 그리고 위에서 아래로 읽음.</p>


### <strong>음표의 표기 Notation of Notes</strong>

<p style="margin: 0 0 1em 0;">음의 높이와 리듬은 음표로 표기하며, 모든 음표는 Notehead(머리)를 가지고 있음.</p>

<p style="margin: 0 0 1em 0;">머리는 속이 비어있을 수도 있고(2분음표, 온음표 등)</p>

<p style="margin: 0 0 1em 0;">검게 칠해져 있을 수도 있음(4분음표 이하).</p>

<p style="margin: 0 0 1em 0;">그리고, 필요에 따라 Stem(기둥)이 붙거나, 리듬을 쪼개는 Flag(꼬리) 와 Beam(빔)이 추가되기도 함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>오선 Staff Notation</strong>

<p style="margin: 0 0 1em 0;">오선은 글자 그대로 간격이 일정한 5개의 가로줄이며 음의 높이를 표시하기 위해 꼭 필요함.</p>

<p style="margin: 0 0 1em 0;">각각의 음표는 그 음높이에 해당하는 줄 위나, 칸에 표시함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>음표 그리기 Placing Notes on a Staff</strong>

<p style="margin: 0 0 1em 0;">음표의 머리를 그릴 때는 위치가 매우 중요함.</p>

<p style="margin: 0 0 1em 0;">줄 위에 그릴 때는 줄이 머리 정중앙을 통과하도록 그리고,</p>

<p style="margin: 0 0 1em 0;">칸에 그릴 때는 줄을 넘어가지 않도록 위아래 줄에 딱 닿을 정도로 그림.</p>

<img src="/images/01.-음표와-음자리표-표기방법-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">음표 머리는 오른쪽 위를 향해 살짝 기울어진 타원형임.</p>

<img src="/images/01.-음표와-음자리표-표기방법-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>기둥과 빔 Stems and Beams</strong>

<p style="margin: 0 0 1em 0;">음표의 기둥은 위로 뻗을 수도 있고, 아래로 뻗을 수도 있음.</p>

<p style="margin: 0 0 1em 0;">기둥을 올릴 때는 음표 머리의 오른쪽에, 내릴 때는 왼쪽에 그림.</p>

<p style="margin: 0 0 1em 0;">방향 결정의 기준은 오선의 셋째 줄이며, 위에 있는 음은 기둥을 아래로 내리고, 오선의 셋째 줄보다 아래 음은 기둥을 위로 올림.</p>

<p style="margin: 0 0 1em 0;">셋째 줄에 걸린 음은 위 아래 둘 다 가능한데, 보통 주변 음표들의 방향을 따라감.</p>

<img src="/images/01.-음표와-음자리표-표기방법-5.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

### <strong>2도 화음 표시 Drawing Seconds</strong>

<p style="margin: 0 0 1em 0;">도와 레, 미와 파처럼 붙어 있는 음을 동시에 연주 할 때는, 기둥 하나를 공유해서, 음표 머리를 기둥의 왼쪽과 오른쪽으로 서로 빗겨서 그림.</p>

<p style="margin: 0 0 1em 0;">기둥이 위로 가든 아래로 가든 상관없이, 낮은 음이 항상 왼쪽에 와야 함.</p>


### <strong>음자리표 Clefs</strong>

<p style="margin: 0 0 1em 0;">음자리표는 오선의 각 줄과 칸이 정확히 어떤 음인지를 정하는 기준임.</p>

<p style="margin: 0 0 1em 0;">오선에 음표의 위치를 보고 높고 낮음을 말하긴 하지만, 음자리표가 없으면 그 위치가 어떤 음을 뜻하는지는 알 수 없음.</p>

<p style="margin: 0 0 1em 0;">이건 우리가 위에 있는 음을 높게, 아래에 있는 음을 낮게 인식하는 공간적 개념을 쓰기 때문이고, 물리적으로도 높은 음일수록 주파수가 높고 낮은 음은 주파수도 낮음.</p>

<p style="margin: 0 0 1em 0;">가장 많이 쓰이는 음자리표는 높은음자리표와 낮은음자리표임. 이 외에 알토와 테너 음자리표도 있지만, 거의 쓰지 않음.</p>

<p style="margin: 0 0 1em 0;">같은 음높이라도 어떤 음자리표를 쓰느냐에 따라 오선 위에서의 위치는 달라짐. 아래 그림은 동일한 음을 높은음자리표, 낮은음자리표, 알토 음자리표, 테너 음자리표로 각각 표기한 것임.</p>

<img src="/images/01.-음표와-음자리표-표기방법-6.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>음자리표 그리기 Drawing Clefs</strong>

<p style="margin: 0 0 1em 0;">높은음자리표는 아래 그림처럼 세 단계로 그림.</p>

<p style="margin: 0 0 1em 0;">먼저 오선 위아래로 약간 튀어나오게 기울어진 세로선을 그리고,</p>

<p style="margin: 0 0 1em 0;">위에서 두 번째 줄과 만나는 반원을 그린다음</p>

<p style="margin: 0 0 1em 0;">마지막으로 아래에서 두 번째 줄을 중심으로 원을 그리는데, 아래쪽 두 칸 안에 들어오도록 함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-7.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">낮은음자리표도 마찬가지로 세 단계로 그릴 수 있음.</p>

<img src="/images/01.-음표와-음자리표-표기방법-8.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">먼저 위에서 두 번째 줄에 점을 찍고,</p>

<p style="margin: 0 0 1em 0;">오선을 넘지 않도록 뒤집힌 C 모양을 그림.</p>

<p style="margin: 0 0 1em 0;">마지막으로 C의 오른쪽, 오선의 위쪽 두 칸 가운데에 점 두 개를 찍음.</p>

<p style="margin: 0 0 1em 0;">알토와 테너 음자리표는 잘 안쓰니까 간단히만 짚고 넘어감.</p>

<img src="/images/01.-음표와-음자리표-표기방법-9.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">알토 음자리표는 네 단계로 그리며, 기본 형태는 오선을 가로지르는 두 개의 세로선과 양쪽의 곡선을 가운데 줄에서 연결한 구조임.</p>

<img src="/images/01.-음표와-음자리표-표기방법-10.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">테너는 알토 음자리표와 그리는 방법이 같고, 전체를 오선에서 한 줄 위로 옮긴 거라고 보면 됨.</p>

<p style="margin: 0 0 1em 0;">실제로는 알토나 테너 음자리표를 그릴 때, 약식으로 알파벳 K처럼 쓰기도 함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-11.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>덧줄 Writing Ledger Lines</strong>

<p style="margin: 0 0 1em 0;">음표가 오선의 범위를 벗어나는 경우, 오선을 위나 아래로 확장해 표시하는 선을 덧줄이라고 함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-12.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">필요한 만큼만 선을 추가해 음높이를 나타내며, 오선을 대신해 임시로 사용함.</p>

<img src="/images/01.-음표와-음자리표-표기방법-13.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>덧줄 표기 시 주의사항</strong>

<p style="margin: 0 0 1em 0;">덧줄은 해당 음표가 위치한 곳까지만 그려야 하며, 음표와 관계없는 위쪽이나 아래쪽에 여분의 줄을 미리 그려 두거나 남겨 두면 안 됨.</p>
