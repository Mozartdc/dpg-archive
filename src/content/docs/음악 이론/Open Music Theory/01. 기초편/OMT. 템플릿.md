<p style="margin: 0 0 1em 0;"><strong>7화음 (Seventh Chords)</strong></p>

<p style="margin: 0 0 1em 0;">7화음은 3화음 위에 3도를 하나 더 쌓은 화음임.</p>

<p style="margin: 0 0 1em 0;">전부 3도씩 올라가니까 시작이 줄이면 전부 줄, 칸이면 전부 칸에 놓임.</p>
