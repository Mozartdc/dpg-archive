<p style="margin: 0 0 1em 0;"><strong>7화음 (Seventh Chords)</strong></p>

<p style="margin: 0 0 1em 0;">7화음은 3화음 위에 3도를 하나 더 쌓은 화음임.</p>

<p style="margin: 0 0 1em 0;">전부 3도씩 올라가니까 시작이 줄이면 전부 줄, 칸이면 전부 칸에 놓임.</p>

<p style="margin: 0 0 1em 0;"><strong>[Example 1. 선율형과 화성형으로 그려진 7화음]</strong></p>

<p style="margin: 0 0 1em 0;">3화음과 마찬가지로 7화음도 맨 아래 음이 근음임.</p>

<p style="margin: 0 0 1em 0;">나머지 음은 근음에서 몇 도 위인지에 따라 이름 붙임.</p>

- 근음
- 3음
- 5음
- 7음
<p style="margin: 0 0 1em 0;">Example 2. 근음, 3음, 5음, 7음 표시한 7화음.</p>

<p style="margin: 0 0 1em 0;">7화음의 종류와 이름</p>

<p style="margin: 0 0 1em 0;">자주 쓰는 7화음은 다섯 가지임.</p>

<p style="margin: 0 0 1em 0;">장7화음, 속7화음(장·단), 단7화음, 반감7화음, 감7화음.</p>

<p style="margin: 0 0 1em 0;">구분은</p>

1. 근음·3음·5음이 만드는 3화음이 장인지 단인지 감인지
1. 근음에서 7음까지가 장7도인지 단7도인지 감7도인지에 따라 나뉨
<p style="margin: 0 0 1em 0;">Example 3. 7화음 종류와 명칭 정리.</p>


| <strong>7화음 명칭</strong> | <strong>3화음</strong> | <strong>7음</strong> | <strong>코드 기호</strong> |
| --- | --- | --- | --- |
| 장-장 7화음 (장7화음) | 장3화음 | 장7도 | Fma7 |
| 장-단 7화음 (도미넌트 7화음) | 장3화음 | 단7도 | F7 |
| 단-단 7화음 (단7화음) | 단3화음 | 단7도 | Fmi7 |
| 반감 7화음 | 감3화음 | 단7도 | F∅7 |
| 완전감 7화음 (감7화음) | 감3화음 | 감7도 | Fo7 |

<p style="margin: 0 0 1em 0;">위 표에서 7화음의 명칭이 두 가지로 적힌 것은, 이론적 이름과 관용적 이름이 함께 쓰이기 때문임.</p>

<p style="margin: 0 0 1em 0;">처음 세 가지는 앞부분이 3화음의 성질, 뒷부분이 7도의 성질을 뜻함.</p>

- 장-장: 장3화음 + 장7도
- 장-단: 장3화음 + 단7도
- 단-단: 단3화음 + 단7도
<p style="margin: 0 0 1em 0;">반감 7화음과 감 7화음은 모두 <strong>감 3화음</strong>을 기반으로 하지만, <strong>7음의 간격</strong>이 다름.</p>

- 반감: 감3화음 + 단7도
- 감: 감3화음 + 감7도
<p style="margin: 0 0 1em 0;">Example 4는 이를 악보로 정리한 것임.</p>

<p style="margin: 0 0 1em 0;">코드 기호 표기</p>

<p style="margin: 0 0 1em 0;">코드 기호는 근음 이름 뒤에 화음의 종류를 표기함.</p>

- 근음에 ♯이나 ♭이 있으면 코드 이름에도 그대로 표기함. 예: B♭m7, G♯ma7
- 근음이 아닌 음이 베이스에 오면 슬래시(/) 뒤에 그 음을 표기함. (Example 5)
