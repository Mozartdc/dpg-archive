### 3화음의 전위


<p style="margin: 0 0 1em 0;">3화음에서는 베이스가 기준이 되고, 악기나 성부와 관계없이 가장 낮은 음이 베이스임.</p>


<iframe src="https://musescore.com/user/32728834/scores/6839459/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">A장3화음은 A, C♯, E 세 음으로 이루어짐. 기본형은 이 세 음을 3도 간격으로 쌓고 근음이 가장 아래에 놓여 베이스인 상태임.
만일, 근음이 아닌 다른 음이 베이스에 오면 전위 상태임.
- 3도가 베이스에 오면 제1전위,
- 5도가 베이스에 오면 제2전위임.
- 화음기호에 슬래시를 붙이면, 슬래시 뒤의 음이 베이스임.
베이스와 근음은 같은 개념이 아님. A장3화음의 근음은 언제나 A이지만, 전위에 따라 베이스는 A, C♯, E로 달라질 수 있음.</p>


| 전위 | 근음 | 베이스 |
| --- | --- | --- |
| 기본형 | 근음 | 근음 |
| 제1전위 | 근음 | 3도 |
| 제2전위 | 근음 | 5도 |


### 3화음 전위 구성 방법

<p style="margin: 0 0 1em 0;">화음 기호를 보고 3화음을 전위할 때, 먼저 기본 3화음을 만든 뒤 마지막에 베이스를 바꾸면 됨.</p>

1. 보표에 근음을 적음.
1. 그 위에 3도와 5도를 쌓음.
1. 근음의 장조 조표를 기준으로 필요한 임시표를 표기함.
1. 단·감·증3화음일 경우에는 3도나 5도에 임시표를 표기함.
<p style="margin: 0 0 1em 0;">이렇게 기본형을 만든 다음, 전위에 따라 해당 음을 베이스로 옮기면 전위가 완성됨.</p>

<p style="margin: 0 0 1em 0;">예를 들어 A♭단3화음 제1전위(A♭mi/C♭)는 기본형을 만든 뒤 3도 음을 베이스로 옮기면 됨.</p>

<img src="/images/18.-전위-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

- 먼저 근음 A♭을 보표에 적고, 그 위에 3도와 5도를 쌓으면 C와 E가 됨.
- A♭장조의 조표는 B♭, E♭, A♭, D♭이므로 E에 ♭을 표기하면,  A♭장3화음(A♭–C–E♭)이 완성됨.
- 단조로 바꾸기 위해 3도를 반음 내려 C를 C♭으로 바꾸면 A♭단3화음(A♭–C♭–E♭)이 됨.
- 여기서 3도 음을 베이스로 옮기면 제1전위(A♭mi/C♭)가 구성됨.
<p style="margin: 0 0 1em 0;">이 방식은 정확하고, 단계적으로 전위를 이해할 수 있지만 시간이 걸림. 하지만, 여러 성질의 3화음과 7화음을 반복해서 연습하면 이런 과정 없이도 바로 전위를 파악할 수 있음.</p>


### 전위된 3화음 식별하기

<p style="margin: 0 0 1em 0;">3화음은 근음과 성질, 전위를 기준으로 판단함. 전위된 3화음도 이 세 가지가 식별 기준이며 아래의 단계에 따라 확인하면 쉽게 찾을 수 있음.</p>

- 화음의 근음을 찾아 적음.
- 화음의 성질(장, 단, 감, 증 등)을 확인해 적음.
- 전위 상태를 확인함.
- 슬래시(/)와 베이스 음이름을 적절히 표기함.
<p style="margin: 0 0 1em 0;">아래 악보의 1마디는 전위된 화음이고, 2마디가 기본형임.</p>


<iframe src="https://musescore.com/user/32728834/scores/6839468/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

- 2마디의 기본형을 보고 D가 근음이라는 것을 알 수 있음.
- 화음의 성질은 근음D와 3음F 사이가 반음 3개 거리인 <strong>단3도</strong>이고, D와 5음A 사이가 <strong>완전5도</strong>이므로 단3화음임.
- 3화음의 3음이 베이스에 위치하고 있으므로 이 화음은 제1전위 상태임.
- 화음 기호는 Dmi/F로 표기함.

<iframe src="https://musescore.com/user/32728834/scores/6876514/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 예제도 같은 단계로 풀어보면, </p>

- 2마디의 기본형을 보고 A가 근음이라는 것을 알 수 있음.
- 화음의 성질은 근음A와 3음C♯사이가 <strong>장3도</strong>이고, A와 5음E 사이가 <strong>완전5도</strong>이므로 장3화음임.
- 3화음의 5음이 베이스에 위치하고 있으므로 이 화음은 제2전위 상태임.
- 따라서 화음 기호는 A/E임.
<p style="margin: 0 0 1em 0;">위 예제에서 보여준 2마디(기본형)을 직접 적기보다 머릿속으로 떠올려서 풀다보면 시간을 절약할 수  있음.</p>


### <strong>7화음의 전위</strong>

<p style="margin: 0 0 1em 0;">3화음과 마찬가지로 7화음도 전위가 가능함. 전위로 다른 음이 베이스에 올 수 있음. </p>


<iframe src="https://musescore.com/user/32728834/scores/6843779/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">A7화음의 기본형과 제1전위, 제2전위, 제3전위</p>

<p style="margin: 0 0 1em 0;">7화음은 3화음보다 구성음이 하나 더 많으므로 전위도 하나 더 많음. 이때 7도 음이 베이스에 있으면 제3전위임. </p>

<p style="margin: 0 0 1em 0;">베이스와 근음은 같은 개념이 아님. 위 악보의 A7화음의 근음은 언제나 A이지만, 전위에 따라 베이스는 달라질 수 있음.</p>


| 전위 | 근음 | 베이스 |
| --- | --- | --- |
| 기본형 | 근음 | 근음 |
| 제1전위 | 근음 | 3도 |
| 제2전위 | 근음 | 5도 |
| 제3전위 | 근음 | 7도 |


### 7화음 전위 구성 방법

<p style="margin: 0 0 1em 0;">화음 기호를 보고 7화음을 전위할 때, 먼저 기본 7화음을 만든 뒤 마지막에 베이스를 바꾸면 됨.</p>

1. 보표에 근음을 적음.
1. 그 위에 3도, 5도, 7도를 쌓음.
1. 근음의 장조 조표를 기준으로 필요한 임시표를 표기함.
1. 조표에 해당하는 음에 임시표를 적용해 7화음을 완성함.
<p style="margin: 0 0 1em 0;">이렇게 기본형을 만든 다음, 전위에 따라 해당 음을 베이스로 옮기면 전위가 완성됨.</p>

<img src="/images/18.-전위-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">D장7화음(D major major seventh chord)을 예를 들면,</p>

- 먼저 근음 D를 보표에 적고, 그 위에 3도·5도·7도를 쌓으면 F, A, C가 됨.
- D장조에서 조표는 F♯, C♯이므로 F와 C에 ♯을 표기하면 D장7화음(D–F♯–A–C♯)이 완성됨.
- 여기서 5도 음인 A를 베이스로 옮기면 제2전위(Dmaj7/A)가 됨.
<p style="margin: 0 0 1em 0;">이 방식은 정확하고, 단계적으로 전위를 이해할 수 있지만 시간이 걸림. 하지만 여러 성질의 3화음과 7화음을 반복해서 연습하면 이런 과정 없이도 바로 전위를 파악할 수 있음.</p>


### 전위된 7화음 구성하기

<p style="margin: 0 0 1em 0;">7화음도 3화음과 마찬가지로 근음과 성질, 전위를 기준으로 판단함. 전위된 7화음은 여기에 7음의 성질까지 함께 확인해야 하므로, 아래의 단계에 따라 식별하면 됨.</p>

- 화음의 근음을 찾아 적음.
- 3화음 부분의 성질(장, 단, 감, 증 등)을 확인해 적음.
- 7음의 성질(장7도인지 단7도인지)을 확인해 적음.
- 전위 상태를 확인함.
- 슬래시(/)와 베이스 음이름을 표기함.
<p style="margin: 0 0 1em 0;">아래 악보의 1마디는 전위된 7화음이고, 2마디가 기본형임.</p>


<iframe src="https://musescore.com/user/32728834/scores/6843787/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">악보를 보고 위 단계대로 풀어보면,</p>

- 2마디의 기본형을 보고 E가 근음이라는 것을 알 수 있음.
- 3화음 부분의 성질은 단3화음임.
- 7음의 성질도 단7도임.
- 3음이 베이스에 있으므로 제1전위 상태임.
- 따라서 화음 기호는 Emi7/G임.

<iframe src="https://musescore.com/user/32728834/scores/6876617/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보도 같은 단계로,</p>

- 2마디의 기본형을 보고 G가 근음이라는 것을 알 수 있음.
- 3화음 부분의 성질은 장3화음임.
- 7음의 성질은 단7도임.
- 7음이 베이스에 있으므로 제3전위 상태임.
- 따라서 화음 기호는 G7/F임.
<p style="margin: 0 0 1em 0;">여기서도 2마디(기본형)를 직접 다시 적기보다 머릿속으로 기본형을 떠올려서 판단하면 시간을 절약할 수 있음.</p>

<p style="margin: 0 0 1em 0;">또한, 음의 중복(더블링)이나 개리배치(open spacing)와 관계없이 3화음과 7화음을 판단하는 기준은 변하지 않음.</p>
