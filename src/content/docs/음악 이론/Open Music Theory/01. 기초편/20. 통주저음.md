<p style="margin: 0 0 1em 0;">17세기와 18세기에는 종이와 잉크 값이 매우 비쌌고, 악보를 적는 데 시간이 많이 걸렸음. 그래서 합주곡에서는 화음을 하나하나 모두 적지 않고, 베이스 위에 필요한 정보만 간단히 표시하는 통주저음을 사용했음.</p>

<p style="margin: 0 0 1em 0;">통주저음은 베이스 아래에 숫자와 기호를 적어, 그 위에 어떤 음을 쌓아야 하는지 알려 주는 체계임.</p>


### 3화음 숫자 기호

<p style="margin: 0 0 1em 0;">통주저음에서는 베이스 위 음정을 숫자로 표기함. 연주자는 그 숫자를 보고 베이스 위에 필요한 음정을 쌓아 화음을 연주함.</p>


<iframe src="https://musescore.com/user/32728834/scores/6839460/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">기본 위치 3화음은 베이스 위에 3도와 5도를 쌓음.($\begin{matrix}5\\3\end{matrix}$)
1전위는 3도와 6도를 쌓음.($\begin{matrix}6\\3\end{matrix}$)
2전위는 4도와 6도를 쌓음.($\begin{matrix}6\\3\end{matrix}$)</p>


<iframe src="https://musescore.com/user/32728834/scores/6839461/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">시간과 잉크, 공간을 더 아끼기 위해 숫자를 줄여 적기도 함. 기본 위치는 숫자를 생략하고, 1전위는 6만 표시함.
2전위는 1전위와 구별해야 하므로 $\begin{matrix}6\\4\end{matrix}$를 모두 표기함.</p>

<p style="margin: 0 0 1em 0;">통주저음만 보고 실제 화음을 만들어 연주하거나 악보에 옮기는 과정을 실현이라고 함.</p>


<iframe src="https://musescore.com/user/32728834/scores/6839462/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보의 첫 마디처럼 베이스가 E♭이고 숫자가 없다면 E♭장화음을 기본형으로 읽음. </p>

<p style="margin: 0 0 1em 0;">3마디처럼 베이스가 G이고
($\begin{matrix}6\end{matrix}$)
이 적혀 있으면 G 위에 3도(B♭)와 6도(E♭)를 쌓아 1전위로 읽음.</p>

<p style="margin: 0 0 1em 0;">5마디는 베이스 B♭ 아래에
($\begin{matrix}6\\4\end{matrix}$)
이 적혀 있으므로, B♭ 위에 4도와 6도를 쌓아 읽음. </p>


### 7화음 숫자 기호

<p style="margin: 0 0 1em 0;">7화음도 3화음과 같은 원리로 표기함.</p>


<iframe src="https://musescore.com/user/32728834/scores/6843780/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보의 전체 표기를 보면, 기본 위치 7화음은 베이스 위에 3도·5도·7도를 쌓음.</p>

<p style="margin: 0 0 1em 0;">1전위는 3도·5도·6도,
2전위는 3도·4도·6도,
3전위는 2도·4도·6도를 쌓음.</p>


<iframe src="https://musescore.com/user/32728834/scores/6843782/s/mesgHK/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보는 현대음악에서 사용하는 7화음의 약식 통주저음 표기임.
기본 위치는 $7$, 1전위는 $\begin{matrix}6\\5\end{matrix}$, 2전위는$\begin{matrix}4\\3\end{matrix}$, 3전위는
$\begin{matrix}4\\2\end{matrix}$
또는 $2$로도 표기함.</p>

<p style="margin: 0 0 1em 0;">전위된 7화음도 3화음과 같은 방식으로 읽음. 베이스 위에 표시된 음정을 그대로 쌓으면 되고, 전체 숫자는 보통 생략한다는 점만 주의하면 됨.</p>


<iframe src="https://musescore.com/user/32728834/scores/26579989/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보의 첫 마디에 베이스 A 아래에 $7$이 적혀 있으므로, A 위에 3도·5도·7도를 쌓아 읽음. A 딸림7화음 기본형임. </p>

<p style="margin: 0 0 1em 0;">3마디에서는 베이스 C♯ 아래에$\begin{matrix}6\\5\end{matrix}$
이 적혀 있으므로 3도·5도·6도를 순서대로 쌓아 읽음. A 딸림7화음 1전위임.</p>

<p style="margin: 0 0 1em 0;">5마디에서는 베이스 E 아래에$\begin{matrix}4\\3\end{matrix}$
이 적혀 있으므로 E 위에 3도·4도·6도를 쌓아 읽음. A 딸림7화음 2전위임.</p>

<p style="margin: 0 0 1em 0;">7마디는 베이스 G 아래에 $\begin{matrix}4\\2\end{matrix}$
이 적혀 있으니, G 위에 2도·4도·6도를 쌓아 읽음. A 딸림7화음 3전위임.</p>


### 기타 통주저음 기호

<p style="margin: 0 0 1em 0;">음을 반음 올리거나 내릴 때는 해당 숫자 앞에 ♭, ♯, ♮를 붙임. 아래 악보에 임시표가 붙은 통주저음과 그 실현 예를 정리함.</p>


<iframe src="https://musescore.com/user/32728834/scores/6877976/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">아래 악보 처럼 숫자에 사선을 긋거나 숫자 앞에 +를 붙이면 그 음을 반음 올려서 읽음.</p>


<iframe src="https://musescore.com/user/32728834/scores/6878013/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">임시표가 숫자 없이 단독으로 표기되어 있으면 베이스 위 3도에 적용함. 아래 악보에 단독 임시표의 실현 예 참고</p>


<iframe src="https://musescore.com/user/32728834/scores/6878023/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">가로선은 베이스 위 음의 변화나 화성 변화를 나타냄.</p>


<iframe src="https://musescore.com/user/32728834/scores/26580793/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 악보의 첫 마디는 베이스 위 5도가 6도로 바뀌는 진행임.</p>

<p style="margin: 0 0 1em 0;">3–4마디는 계류 진행으로, 4도가 3도로 해결되는 형태임.</p>

<p style="margin: 0 0 1em 0;">5–6마디는 종지적 $\begin{matrix}6\\4\end{matrix}$ 진행임.
$\begin{matrix}6\\4\end{matrix}$ → $\begin{matrix}5\\3\end{matrix}$ 로 내려감.</p>

<p style="margin: 0 0 1em 0;">7–8마디는 베이스 이동에 따른 화성 변화임.</p>


### 로마 숫자와 숫자 기호

<p style="margin: 0 0 1em 0;">로마 숫자와 통주저음 숫자를 함께 적어 기능과 전위를 동시에 표시하기도 함. $I^{6}$ 은 으뜸화음 1전위, $V^{\begin{matrix}4\\3\end{matrix}}$는 딸림7화음 2전위임.
이 방식은 교재에 따라 사용 여부가 다름.</p>
