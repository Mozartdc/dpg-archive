### <strong>7화음 Seventh Chords</strong>

<p style="margin: 0 0 1em 0;">7화음은 3화음 위에 3도를 하나 더 쌓은 화음임.</p>

<p style="margin: 0 0 1em 0;">전부 3도씩 올라가니까 시작이 줄이면 전부 줄, 칸이면 전부 칸에 놓임.</p>


<iframe src="https://musescore.com/user/32728834/scores/6843767/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;"><strong>선율형과 화성형으로 그려진 7화음</strong></p>

<p style="margin: 0 0 1em 0;">3화음과 마찬가지로 7화음도 맨 아래 음이 근음임. 나머지 음은 근음에서 몇 도 위인지에 따라 이름 붙임.</p>


### 7화음의 성질과 이름

<p style="margin: 0 0 1em 0;">자주 쓰는 7화음은 다섯 가지임.</p>

<p style="margin: 0 0 1em 0;">장7화음, 속7화음(장·단), 단7화음, 반감7화음, 감7화음.</p>

<p style="margin: 0 0 1em 0;">구분은</p>

1. 근음·3음·5음이 만드는 3화음이 장인지 단인지 감인지
1. 근음에서 7음까지가 장7도인지 단7도인지 감7도인지에 따라 나뉨

| <strong>7화음 명칭</strong> | <strong>3화음</strong> | <strong>7음</strong> | <strong>코드 기호</strong> |
| --- | --- | --- | --- |
| 장-장 7화음 (장7화음) | 장3화음 | 장7도 | Fma7 |
| 장-단 7화음 (딸림 7화음) | 장3화음 | 단7도 | F7 |
| 단-단 7화음 (단7화음) | 단3화음 | 단7도 | Fmi7 |
| 반감 7화음 | 감3화음 | 단7도 | F∅7 |
| 완전감 7화음 (감7화음) | 감3화음 | 감7도 | Fo7 |

<p style="margin: 0 0 1em 0;">위 표에서 7화음의 명칭이 두 가지로 적힌 것은, 이론적 이름과 관용적 이름이 함께 쓰이기 때문임.</p>

<p style="margin: 0 0 1em 0;">처음 세 가지는 앞부분이 3화음의 성질, 뒷부분이 7도의 성질을 뜻함.</p>

- 장-장: 장3화음 + 장7도
- 장-단: 장3화음 + 단7도
- 단-단: 단3화음 + 단7도
<p style="margin: 0 0 1em 0;">반감 7화음과 감 7화음은 모두 <strong>감 3화음</strong>을 기반으로 하지만, <strong>7음의 간격</strong>이 다름.</p>

- 반감: 감3화음 + 단7도
- 감: 감3화음 + 감7도

<iframe src="https://musescore.com/user/32728834/scores/6850641/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">코드 기호는 근음 이름 뒤에 화음의 종류를 표기함.</p>

- 근음에 ♯이나 ♭이 있으면 코드명에도 그대로 표기함. 예: B♭m7, G♯ma7
- 가장 아래 음이 근음이 아니면, 슬래시(/) 뒤에 그 음을 표기함.

### 7화음 청음 <strong>Listening to Seventh Chords</strong>

<p style="margin: 0 0 1em 0;">Example 4를 잘 들어보면 7화음마다 느낌이 다름. 그래서 소리랑 이미지를 같이 떠올려 두면 기억하기 쉬움.</p>

- <strong>장 7화음 (M7):</strong> 밝은 재즈 느낌
- <strong>속 7화음 (Dom7):</strong> 해결되지 않은 불안정한 느낌, 다른 코드로 가고 싶은 강한 성질 
- <strong>단 7화음 (m7):</strong> 어두운 재즈 느낌
- <strong>반감 7화음 (ϕ7):</strong> 음산하고 긴장된 느낌
- <strong>감 7화음 (∘7):</strong> 매우 강한 긴장감

### <strong>장조와 단조에서의 7화음 Qualities in Major and Minor</strong>

<p style="margin: 0 0 1em 0;"><strong>장조</strong></p>

<p style="margin: 0 0 1em 0;">장조의 모든 구성음은 7화음의 근음이 될 수 있음. 조가 바뀌어도 각 자리의 7화음 성질은 같음.</p>

- <strong>장 7화음:</strong> 도(I), 파(IV)
- <strong>딸림 7화음:</strong> 솔(V) → 장 3화음 + 단 7도
- <strong>단 7화음:</strong> 레(ii), 미(iii), 라(vi)
- <strong>반감 7화음:</strong> 시(viiϕ) → 감 3화음 + 단 7도

<iframe src="https://musescore.com/user/32728834/scores/8514077/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;"><strong>단조</strong></p>

<p style="margin: 0 0 1em 0;">단조에서는 7음을 반음 올려 이끔음으로 쓰는 경우가 많아서, 같은 조 안에서도 V와 vii 자리의 화음 성질이 달라짐.</p>

- <strong>단 7화음:</strong> 도(i), 파(iv)
- <strong>장 7화음:</strong> 메(III), 르(VI)
- <strong>딸림 7화음:</strong> 솔(V) → 이끄는음이 반음 올라가 장 3화음이 됨.
- <strong>반감 7화음:</strong> 레(iiϕ)
- <strong>감 7화음:</strong> 티(vii∘) → 이끔음이 반음 올라가 감 7화음(감 3화음 + 감 7도)이 됨.

### <strong>7화음 구성하기 Spelling Seventh Chords</strong>

<p style="margin: 0 0 1em 0;">코드 기호를 보고 7화음을 적는 순서는 3화음과 비슷함.</p>

<p style="margin: 0 0 1em 0;"><strong>예제 : D장7화음(Dmaj7)</strong></p>

1. 오선에 근음 D를 적음.
1. 그 위에 3도, 5도, 7도를 차례로 쌓음.
1. D장조의 조표(F♯, C♯)를 적용함.
1. 조표에 따라 F와 C에 ♯을 붙임.
<p style="margin: 0 0 1em 0;">→ D–F♯–A–C♯</p>

<img src="/images/17.-7화음-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">감7화음처럼 변화가 많은 경우에는 장7화음을 먼저 만든 뒤 필요한 부분을 고치는 방식이 이해하기 쉬움.</p>

<p style="margin: 0 0 1em 0;"><strong>예제 : A♭감7화음(A♭°7)</strong></p>

1. 오선에 근음 A♭을 적음.
1. 3도, 5도, 7도를 쌓아 C, E, G를 놓음.
1. A♭장조 조표(B♭, E♭, A♭, D♭)를 적용함.
<p style="margin: 0 0 1em 0;">→ A♭–C–E♭–G (장7화음 상태)</p>

<p style="margin: 0 0 1em 0;">이제 감7화음으로 바꿈.</p>

1. 3화음을 감으로 만들기 위해
  <p style="margin: 0 0 1em 0;">C → C♭</p>

  <p style="margin: 0 0 1em 0;">E♭ → E♭♭</p>

1. 7도도 감7도로 만들기 위해
  <p style="margin: 0 0 1em 0;">G → G♭♭</p>

<p style="margin: 0 0 1em 0;">최종: A♭–C♭–E♭♭–G♭♭</p>

### <strong>7화음의 식별 Identifying Seventh Chords, Doubling, and Spacing</strong>

<p style="margin: 0 0 1em 0;">7화음도 3화음과 마찬가지로 근음, 성질, 전위에 따라 이름을 붙임. 전위는 다음 장에서 다루므로, 여기서는 기본형을 찾는 방법만 설명함.</p>

- 먼저 근음을 찾음.
- 그 근음의 장조를 기준으로 생각함.
- 3화음의 성질을 확인함.
- 7도의 성질을 확인함.

<iframe src="https://musescore.com/user/32728834/scores/6843785/s/Ml4Mxs/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">7화음의 이름을 찾는 순서는, 구성할 때의 과정을 거꾸로 따라가면 됨. </p>

1. 가장 아래 음을 보고 근음을 정함. → C♯
1. C♯장조를 기준으로 생각함.
1. 3화음의 성질을 확인함.
<p style="margin: 0 0 1em 0;">악보의 E와 G는 내추럴임.</p>

<p style="margin: 0 0 1em 0;">C♯장조라면 E♯, G♯이어야 하므로 반음 낮아진 상태임.</p>

<p style="margin: 0 0 1em 0;">따라서, 감3화음임.</p>

1. 7도의 성질을 확인함.
<p style="margin: 0 0 1em 0;">C♯장조에서는 B♯이 와야 하는데, B 내추럴이므로 반음 낮아진 상태임.</p>

<p style="margin: 0 0 1em 0;">따라서, 장7도에서 반음 줄어 단7도임.</p>

<p style="margin: 0 0 1em 0;">감3화음 + 단7도 → C♯반감7화음 (C♯ϕ7)</p>
