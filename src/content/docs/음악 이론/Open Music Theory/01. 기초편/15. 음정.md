<p style="margin: 0 0 1em 0;">음악에서 두 음을 나란히 놓으면, 우리는 자연스럽게 “가깝다” 또는, “멀다” 같은 감각을 먼저 느끼게 됨. 이 감각을 정리해서 다루기 위해 만든 개념이 음정(Interval)임.
그런데 악보를 볼 때 느끼는 거리와, 실제로 소리로 들을 때 느끼는 거리는 다를 수 있어서 이 ‘거리’가 항상 하나의 기준으로만 정해지는 건 아님.
어떤 음정은 악보상으로는 넓어 보이지만 소리는 비교적 가깝게 들리고, 반대로 악보상으로는 가까워 보여도 강한 긴장으로 들리기도 함. 그래서 음정은 처음부터 두 가지 관점으로 나눠서 생각해야 함.
하나는 악보에서 두 음이 얼마나 떨어져 있는지를 표시하는 <strong>도수 (Size)</strong>이고, 다른 하나는 실제로 들었을 때 두 음 사이가 어떻게 느껴지는지를 나타내 <strong>성질(Qualities)</strong>임.
이 두 가지를 함께 생각하지 않으면, 음정은 그냥 숫자나 기호에 불과한 추상적인 개념이 됨. 음악적으로 음정을 다루려면, 항상 ‘적혀 있는 것’과 ‘들리는 것’을 동시에 염두에 둬야 함.</p>

<p style="margin: 0 0 1em 0;"><strong>도수 (Size)</strong></p>

<p style="margin: 0 0 1em 0;">두 음이 어떻게 울리느냐에 따라 음정은 성격이 달라짐. 두 음을 동시에 치거나 부르면 화성적 음정이 되고, 차례로 이어서 치거나 부르면 선율적 음정이 됨.</p>


<iframe src="https://musescore.com/user/32728834/scores/8418647/s/H5aOaq/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">모든 음정에는 두 가지 정보가 함께 따라붙음.
하나는 두 음이 악보에서 얼마나 떨어져 있는지를 말해 주는 도수이고, 다른 하나는 그 거리가 어떤 성격을 가지는지를 나타내는 성질임.
이 가운데 도수는 오선보 위에서 두 음 사이에 몇 개의 줄과 칸이 있는지를 기준으로 정해짐. 표기할 때는 2, 3, 4 같은 숫자를 쓰지만, 실제로 부를 때는 2도, 3도, 4도처럼 서수로 읽음.
도수를 셀 때는 항상 아래 음을 1로 두고 시작함.</p>

<p style="margin: 0 0 1em 0;">예외로, 같은 줄이나 같은 칸에 있는 두 음 사이의 간격은 1도라고 하지 않고 유니즌이라고 부르며, 8도는 옥타브라고 부름.</p>


<iframe src="https://musescore.com/user/32728834/scores/8418638/s/e4jxEI/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">도수는 줄과 칸의 개수만으로 정해지는 일반적(Generic)인 기준이라 음에 어떤 임시표가 붙든 도수 자체는 달라지지 않음.
아래 악보처럼 임시표가 제각각이라도, 두 음 사이의 줄과 칸이 세 개라면 이들은 모두 3도가 됨.</p>


<iframe src="https://musescore.com/user/32728834/scores/8418665/s/n71TMI/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;"><strong>성질(Qualities) - 완전, 장, 단</strong></p>

<p style="margin: 0 0 1em 0;">같은 도수라도 실제로 들리는 느낌은 달라질 수 있기 때문에 도수(size)만으로는 음정을 충분히 설명할 수 없음. 이 차이를 구분하기 위해 도수에 붙는 기준이 성질임.
성질은 악보에 적힌 두 음 사이의 거리를 좀 더 세밀하게 구분하고, 도수와 결합해 그 음정이 어떻게 들리는지를 설명해 줌.</p>

<p style="margin: 0 0 1em 0;">다섯 가지 음정의 성질.</p>

- 증 (Augmented, A 또는 +)
- 장 (Major, ma)
- 완전 (Perfect, P)
- 단 (Minor, mi)
- 감 (Diminished, d 또는 o)
<p style="margin: 0 0 1em 0;">음정을 말하거나 적을 때는 성질을 먼저 쓰고, 그다음에 도수를 써서 완전 4도, 단 3도, 증 2도와 같이 표기함.</p>

<p style="margin: 0 0 1em 0;"> 
이번 챕터에서는 이 가운데 완전, 장, 단 세 가지만 먼저 다룸. 이 세 성질은 현재 가장 기본적인 기준으로 쓰이고 있음.
학자나 시대에 따라 다른 방식으로 음정의 성질을 분류하기도 했지만, 지금은 2도, 3도, 6도, 7도는 장이나 단으로 나뉘고, 유니즌, 4도, 5도, 옥타브는 완전 음정으로 취급함.</p>


| Major / Minor | Perfect |
| --- | --- |
| 2nds | Unisons |
| 3rds | 4ths |
| 6ths | 5ths |
| 7ths | Octaves |

<p style="margin: 0 0 1em 0;"><strong>장음계를 활용한 음정 성질 구분 방법</strong></p>

<p style="margin: 0 0 1em 0;">음정의 성질을 파악 할 때 흔히 반음의 개수를 일일이 세는데, 이 방식은 시간도 오래 걸리고 계산 실수가 잦아 권장하지 않음. 대신 우리가 이미 익숙한 <strong>장음계(Major Scale)의 규칙</strong>을 활용하면, 훨씬 빠르고 정확하게 음정 성질 구분 할 수 있음.
</p>

1. <strong>도수 결정:</strong> 두 음 사이의 줄과 칸을 세어 도수를 먼저 확정함.
1. <strong>으뜸음 가정:</strong> 아래쪽 음이 장음계의 으뜸음(Tonic)이라고 가정함.
1. <strong>포함 여부 확인:</strong> 위쪽 음이 가정된 장음계의 구성음에 포함되는지에 따라 성질을 결정함.
- <strong>포함된다면:</strong>
  - 1, 4, 5, 8도는 <strong>완전(Perfect)</strong> 음정임.
  - 2, 3, 6, 7도는 <strong>장(Major)</strong> 음정임.
- <strong>포함되지 않는다면:</strong>
  - 기준보다 반음 낮다면 <strong>단(Minor)</strong> 음정임 (2, 3, 6, 7도의 경우).
  - 그 외에는 다음 섹션에서 다룰<strong> 증(Augmented)</strong>이나 <strong>감(Diminished)</strong> 음정임.

<iframe src="https://musescore.com/user/32728834/scores/6836733/s/wud8Ld/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">위 a 악보의 F와 C를 장음계 방법으로 음정을 찾아보면, </p>

1. <strong>도수 확인:</strong> 먼저, 이 음정은 <strong>5도</strong>임. (F부터 세면 F=1, G=2, A=3, B=4, C=5).
1. <strong>장음계 대입:</strong> C음은 <strong>F 장조</strong>(플랫이 하나, B♭)의 조성 안에 포함됨.
1. 따라서, 이 음정은 <strong>완전 5도</strong>임.
<p style="margin: 0 0 1em 0;">이제 <strong>b </strong>악보의 E♭과 C♭에 이 과정을 똑같이 적용하면,</p>

1. <strong>도수 확인:</strong> 먼저, 이 음정은 <strong>6도</strong>임 (E♭부터 세면 E♭=1, F=2, G=3, A=4, B=5, C=6).
1. <strong>장음계 대입:</strong> C♭음은 <strong>E♭ 장조</strong>(플랫이 셋: B♭, E♭, A♭)의 조성 안에 <strong>없음</strong>.
1. 따라서, 이 음정은 <strong>단 6도</strong>임. <em>참고:</em> E♭ 장조에서 장 6도가 되려면 C♭ 대신 C♮이어야 함.
<p style="margin: 0 0 1em 0;"><strong>증·감 성질</strong></p>

<p style="margin: 0 0 1em 0;">앞에서 본 것처럼, 음정의 성질은 증·장·완전·단·감, 이렇게 다섯 가지가 있음. 여기서는 기준에서 반음 벗어나는 경우를 살펴봄.</p>

<p style="margin: 0 0 1em 0;"><strong>증음정(Augmented) : </strong>완전 음정이나 장음정보다 간격이 반음 하나 더 넓은 경우임.</p>

<p style="margin: 0 0 1em 0;">아래 악보의 a의 첫 번째 마디, F와 C는 C가 F장조의 구성음이기 때문에 <strong>완전 5도</strong>임. 여기서 위쪽 음 C를 반음 올려 C♯으로 바꾸면, 두 음 사이 간격이 반음 더 넓어져 F–C♯은 <strong>증 5도</strong>가 됨(A5 또는 +5).</p>

<p style="margin: 0 0 1em 0;">다음 마디에서는 G와 E는 E가 G장조의 구성음이므로 <strong>장 6도</strong>를 이룸. 이때 위쪽 음 E를 반음 올려서 E♯으로 만들면, 이 음정은 <strong>증 6도</strong>가 됨(A6 또는 +6).</p>

<p style="margin: 0 0 1em 0;">악보 b는 아래쪽 음을 바꾼 경우로 원리는 같음. 첫 번째 마디에서 완전 5도인 F–C에서 아래쪽 음 F를 반음 내려 F♭으로 바꾸면, 간격이 반음 더 넓어져 <strong>증 5도</strong>가 됨.</p>

<p style="margin: 0 0 1em 0;">다음 마디에서도 장 6도인 G–E에서 아래쪽 음 G를 반음 내려 G♭으로 바꾸면, <strong>증 6도</strong>가 됨.</p>


<iframe src="https://musescore.com/user/32728834/scores/8418677/s/-x_9Yz/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;"><strong>감음정 (Diminished)</strong> : 완전 음정이나 단음정보다 간격이 반음 하나 더 좁은 경우임.</p>

<p style="margin: 0 0 1em 0;">아래 악보 a의 첫 번째 마디, 완전 5도인 F–C의 위쪽 음을 C♭으로 반음 내리면, 간격이 반음 좁아져 <strong>감 5도</strong>가 됨(트라이톤이라 부르며 d5 또는 o5로 씀).</p>

<p style="margin: 0 0 1em 0;">다음 마디의 장 6도인 G와 E에서 위쪽 음을 반음 내리면서 먼저 단 6도가 되고, 여기서 E𝄫(더블 플랫)으로 한 번 더 내려 <strong>감 6도</strong>가 됨.</p>

<p style="margin: 0 0 1em 0;">완전음정과 단음정은 간격이 반음 좁아지면 바로 감음정이 되지만, <strong>장음정은 반음 좁아지면 단음정이 되기 때문임.</strong></p>

<p style="margin: 0 0 1em 0;">악보 b는 아래쪽 음을 바꾼 경우임. 첫 번째 마디의 완전 5도 F–C에서 아래쪽 음을 F♯으로 반음 올리면, 위쪽과 간격이 줄어들어 <strong>감 5도</strong>가 됨.</p>

<p style="margin: 0 0 1em 0;">두 번째 마디의 장 6도 G–E 역시 아래쪽 음을 G♯으로 반음 올리면 먼저 단 6도가 되고, 세 번째 마디에서 G𝄪(더블 샵)으로 반음 더 좁혀야 <strong>감 6도</strong>가 됨.</p>


<iframe src="https://musescore.com/user/32728834/scores/6836745/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">아래 악보는 음정의 상대적인 크기를 반음 단위로 배열한 것임. 각 괄호는 인접한 괄호와 반음 차이임.</p>

<p style="margin: 0 0 1em 0;">악보 a에서는 <strong>위쪽 음의</strong> 임시표 변화에 따라 음정의 성질이 어떻게 바뀌는지를 보여줌.</p>

<p style="margin: 0 0 1em 0;">완전 음정이나 장음정보다 반음 더 넓어지면 증음정이 되고,</p>

<p style="margin: 0 0 1em 0;">장음정보다 반음 좁아지면 단음정이 되며,</p>

<p style="margin: 0 0 1em 0;">완전 음정이나 단음정보다 반음 좁아지면 감음정이 됨.</p>

<p style="margin: 0 0 1em 0;">악보 b는 같은 내용을 <strong>아래쪽 음</strong>의 변화로 바꿔서 정리한 것임.</p>

<p style="margin: 0 0 1em 0;">위쪽 음 대신 아래쪽 음에 임시표를 붙였을 뿐, 음정의 성질이 변하는 원리는 악보 a와 완전히 같음.</p>


<iframe src="https://musescore.com/user/32728834/scores/8418728/s/SPtiOU/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">악보만 보고 임시표로 인한 음정의 성질 변화를 떠올리기 어렵다면, 감각을 통해 떠올려보는 방식(kinesthetic)으로 생각해도 됨.</p>

<p style="margin: 0 0 1em 0;"><strong>악보 a</strong>처럼 위쪽 음이 바뀌는 경우에는, 두 손을 위아래로 나란히 두고 <strong>위쪽 손</strong>을 위아래로 움직여 성질이 어떻게 변하는지를 머릿속에서 대응시킬 수 있음. <strong>악보 b</strong>에서는 반대로, 위쪽 손은 그대로 두고 <strong>아래쪽 손</strong>을 위아래로 움직이면 됨. 어느 음을 바꾸느냐에 따라 움직이는 손만 달라질 뿐, 원리는 같음.</p>

<p style="margin: 0 0 1em 0;">이때 손 사이의 거리가 가까워지거나 멀어지더라도 음정의 <strong>도수(generic size)는 변하지 않음</strong>. 손의 움직임은 임시표가 더해지거나 바뀔 때 생기는 <strong>성질의 변화를 연상하기 위한 도구</strong>임.</p>
