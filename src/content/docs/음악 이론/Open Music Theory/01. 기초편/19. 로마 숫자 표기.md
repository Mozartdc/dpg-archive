### 로마 숫자

<p style="margin: 0 0 1em 0;">로마 숫자는 조성 안에서 화음을 표시하는 방법임. 각 숫자는 화음 근음의 도수를 뜻함. 특정 음이름이 아니라 도수를 기준으로 하기 때문에, 조성이 달라도 어떤 기능인지 바로 알 수 있음.</p>


| 음계 도수 | 대문자 | 소문자 |
| --- | --- | --- |
| 1 | I | i |
| 2 | II | ii |
| 3 | III | iii |
| 4 | IV | iv |
| 5 | V | v |
| 6 | VI | vi |
| 7 | VII | vii |


### 로마 숫자 표시 방법

<p style="margin: 0 0 1em 0;">대문자 로마 숫자는 라틴 알파벳 I와 V를 대문자로 쓰고, 소문자는 i와 v를 쓰면 됨.
손으로 쓸 때는 대문자를 소문자 로마 숫자와 확실하게 구분하기 위해 위 아래에 가로선을 그림.</p>

### 로마 숫자와 화음의 성질

<p style="margin: 0 0 1em 0;">로마 숫자는 화음의 근음만 나타내는 것이 아니라, 그 화음의 성질까지 함께 표시함. 대문자 로마 숫자는 장3화음, 소문자 로마 숫자는 단3화음을 뜻함.</p>

<p style="margin: 0 0 1em 0;">장조에서 ˆ1(도, do) 위에 쌓은 화음은 “I”로, ˆ2(레, re) 위에 쌓은 화음은 소문자 로마 숫자 “ii”로 적음. 이런 식으로 각 도수에 따라 화음의 성질을 구분해 표기함.</p>

<p style="margin: 0 0 1em 0;">소문자 로마 숫자 뒤에 위첨자 °를 붙이면(예: vii°) 감3화음을 뜻하고, 대문자 로마 숫자 뒤에 + 기호를 붙이면(예: V+) 증3화음을 뜻함.</p>


| 3화음의 성질 | 표기 규칙 | 예시 |
| --- | --- | --- |
| 장3화음 | 대문자 | V |
| 단3화음 | 소문자 | ii |
| 증3화음 | 대문자 + 기호 | V+ |
| 감3화음 | 소문자 + 위첨자 ° | vii° |

<p style="margin: 0 0 1em 0;">조성이 달라져도 스케일 도수와 솔페주는 바뀌지 않듯, 로마 숫자도 마찬가지임. 아래 악보에 G장조와 G단조의 다이어토닉 3화음의 로마 숫자를 보면, 으뜸음이 무엇이든 사용하는 로마 숫자는 같다는 것을 알 수 있음.
또한 화음의 근음에 해당하는 솔페주와 도수를 로마 숫자와 함께 보면 이해가 더 쉬움.</p>

<p style="margin: 0 0 1em 0;">단조에서 이끔음을 반음 올리면 화음의 성질이 바뀌고, 그에 따라 로마 숫자도 달라짐.
원래 단조의 5도 화음은 v(단화음)이지만, 이끔음을 올리면 V(장화음)가 됨.
또한 7도 위에 쌓은 화음도 이끔음을 올리면 감화음 vii°로 바뀜.</p>


| 7화음의 성질 | 표기 방식 | 예시 |
| --- | --- | --- |
| 장-장7화음<br>(메이저7화음) | 대문자 | IV7 |
| 장-단7화음<br>(도미넌트7화음) | 대문자 | V7 |
| 단-단7화음<br>(마이너7화음) | 소문자 | ii7 |
| 반감7화음 | 소문자 + ∅ | ii∅7 |
| 완전감7화음<br>(감7화음) | 소문자 + ° | vii°7 |

<p style="margin: 0 0 1em 0;">장7화음과 도미넌트7화음은 둘 다 대문자 로마 숫자에 ⁷을 붙여 적기 때문에 구분이 어려움. 따라서 장7화음인지 도미넌트7화음인지는 조성 안에서의 위치를 봐야 알 수 있음.</p>

<p style="margin: 0 0 1em 0;">도미넌트7화음은 V⁷(단조에서는 VII⁷)에만 쓰임. 그 외의 자리에서 V⁷이 나오면 장7화음임.</p>

<p style="margin: 0 0 1em 0;">G장조와 G단조의 7화음을 코드 기호와 로마 숫자(파란색)로 함께 보면, 조가 바뀌어도 로마 숫자는 같음.</p>

### 로마 숫자 분석

<p style="margin: 0 0 1em 0;">분석 할 때는 가장 먼저 조성을 파악해서 적어 둠. 코드 기호와 마찬가지로, 음이름을 대문자로만 쓰면 장조를 뜻하고, 단조일 경우에는 “mi”를 붙이거나(예: B♭mi), 음이름을 소문자로 적기도 함(예: b♭).</p>

<p style="margin: 0 0 1em 0;">다음으로 화음의 근음을 찾아야 함. 로마 숫자는 그 화음의 근음의 도수에 따라 정해지는데, 화음이 전위되어 있을 수 있으므로, 베이스 음이 항상 근음인 것은 아님. 필요하면 화음을 근음 위치로 다시 쌓아 보되, 가능하면 머릿속으로 정리함.</p>

<p style="margin: 0 0 1em 0;">로마 숫자는 화음의 성질에 따라 대문자나 소문자로 적음. 한 조성 안에서 각 도수의 화음 성질은 일정하므로, 장조와 단조에서의 화음 성질을 익혀 두면 로마 숫자를 대문자로 쓸지 소문자로 쓸지, 또 필요한 기호를 덧붙일지 판단할 수 있음.</p>

<p style="margin: 0 0 1em 0;">정리하면 다음과 같은 순서로 진행함.</p>

1. 조성을 확인함(조표의 올림표나 내림표를 빠뜨리지 않음).
1. 화음의 근음을 찾고, 그 도수를 확인함. 이 도수가 해당 로마 숫자에 해당함.
1. 화음의 성질에 맞게 대문자·소문자와 필요한 기호를 사용해 로마 숫자를 적음.
<p style="margin: 0 0 1em 0;">예시 7에서는 Louise Reichardt의 “Die Wiese”(1811)에 대한 로마 숫자 분석을 보여 줌. 독일어 가사는 다음과 같은 뜻임.</p>

<p style="margin: 0 0 1em 0;">“어느 봄날, 모든 것이 아름답고 즐거웠던 날, 나는 외딴 여름 별장에 이르렀고, 그곳에서 한 소녀가 나와 울고 있었네.”</p>

<p style="margin: 0 0 1em 0;">이 분석에서 눈여겨볼 점은 다음과 같음.</p>

- 조성은 분석의 시작 부분에 적혀 있음. 로마 숫자는 오선 아래에 표시함.
- 화음이 바뀌지 않으면 로마 숫자를 반복해서 쓰지 않음.
- 이 곡은 단조이므로, 때로는 올린 이끔음(G♯, A단조 기준)을 사용함. 그 결과 V와 V⁷이 장화음 또는 도미넌트7화음으로 나타남.
- 올리지 않은 G♮(버금이끎음)는 해당 음 위에 쌓인 도미넌트7화음을 만듦.[3]
- 4마디 성부에 있는 A는 장식음으로, VII⁷ 화음에 포함되지 않으며 로마 숫자에도 영향을 주지 않음.
- 화음의 5도가 빠져 있어도 여전히 V⁷으로 봄.
- 전위는 로마 숫자를 바꾸지 않음. 필요한 경우 전위 표시는 숫자저음(Figured Bass) 방식으로 따로 적음.
<p style="margin: 0 0 1em 0;">예시 8에서도 또 다른 로마 숫자 분석을 보여 줌. 괄호 안의 음은 장식음으로, 화음에 포함되지 않음.</p>

<p style="margin: 0 0 1em 0;">예시 7과 8에서 볼 수 있듯, 다음과 같은 요소들은 로마 숫자에 영향을 주지 않음.</p>

- 화음 배치: 음의 옥타브 위치, 중복, 일부 음(특히 5도)의 생략 등
- 전위나 베이스 음: 전위는 숫자저음으로 표시할 수 있지만, 로마 숫자 자체는 변하지 않음
- 장식음: 모든 음이 화성에 속하는 것은 아니며, 장식음은 로마 숫자를 바꾸지 않음
- 음높이 이외의 요소: 리듬, 음색, 아티큘레이션, 셈여림 등은 대체로 로마 숫자 표기에 영향을 주지 않음
<p style="margin: 0 0 1em 0;">따라서 한 곡을 온전히 분석하려면 로마 숫자로 나타나는 화성뿐 아니라, 그 밖의 음악적 요소도 함께 고려해야 함. 로마 숫자는 화성을 설명하는 데 유용한 도구이지만, 곡 전체를 모두 설명해 주는 체계는 아님.</p>
