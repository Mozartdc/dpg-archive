<p style="margin: 0 0 1em 0;">시창(Sight-singing)은 처음 보는 악보를 보고 즉석에서 노래할 수 있는 능력으로, 음악을 배우는 모든 사람에게 중요한 능력임.</p>

<p style="margin: 0 0 1em 0;">여기에는 처음 보는 리듬을 정확한 박자로 세며 읽는 기술도 포함됨(이를 sight counting이라고 함).</p>

<p style="margin: 0 0 1em 0;"><strong>시창과 리듬 읽기를 위한 연습 방법</strong></p>

<p style="margin: 0 0 1em 0;">시창이나 리듬 읽기를 연습할 때는 올바른 자세가 중요함.</p>

<p style="margin: 0 0 1em 0;">의자 앞쪽에 걸터앉아 허리를 곧게 세우고, 허벅지가 바닥과 평행이 되도록 하며, 횡격막을 중심으로 호흡하는 것이 좋음.</p>

<p style="margin: 0 0 1em 0;">이때 노래하거나 박자를 읽을 때는 음절을 또렷하게 발음해야 정확한 리듬 감각을 기를 수 있음.</p>

<p style="margin: 0 0 1em 0;">Proper singing posture sitting down.</p>

<p style="margin: 0 0 1em 0;">시창 리듬을 연습할 때는 리듬 카운팅을 사용하면 도움이 됨.</p>

<p style="margin: 0 0 1em 0;">Open Music Theory에서는 미국식 카운팅 방식을 기준으로 설명하지만, 반드시 그 방식만 써야 하는 것은 아님. 리듬을 정확하게 읽을 수 있다면 다른 체계도 충분히 효과적임.</p>

<p style="margin: 0 0 1em 0;">시창 리듬 연습에서 유의할 점은 다음과 같음.</p>

- 카운트를 악보에 미리 적어 두지 않는 것이 좋음.
<p style="margin: 0 0 1em 0;">눈으로 바로 리듬을 읽고 반응하는 연습이 되어야 실제 시창 능력이 늘기 때문임.</p>

- 악보를 처음 보면 박자표부터 확인해야 함.
<p style="margin: 0 0 1em 0;">한 마디에 몇 박이 들어가는지, 어떤 음표가 한 박을 기준으로 하는지를 먼저 파악해야 리듬을 안정적으로 읽을 수 있음.</p>

- 연습할 때는 지휘를 활용하는 것도 도움이 됨.
<p style="margin: 0 0 1em 0;">손으로 박을 그리면 템포를 유지하기 쉽고, 지금 어느 박에 있는지도 놓치지 않게 됨. 지휘가 부담스럽다면 손이나 발로 박자를 두드려도 괜찮음.</p>

- 템포가 흔들린다면 도구를 써도 됨.
<p style="margin: 0 0 1em 0;">메트로놈 앱을 켜고 천천히 연습하거나, 특정 리듬에서 계속 헷갈린다면 리듬을 더 잘게 나눠서 연습하는 것이 효과적임.</p>

<p style="margin: 0 0 1em 0;"><strong>시창 방법</strong></p>

<p style="margin: 0 0 1em 0;">시창할 때는 도수나 계이름 같은 선율 시창 체계를 사용하는 것이 기본임. 특정 방법이 정답인 것은 아니며, 하나를 정해 꾸준히 쓰는 것이 중요함.</p>

- 계이름 적지 말 것: 악보에 계이름이나 숫자를 적기 시작하면 읽는 능력이 잘 늘지 않음.
- 기본 정보 확인: 음자리표, 박자표, 조표를 먼저 보고 조성을 파악해야 함.
- 선율의 윤곽 파악: 음이 올라가는지, 내려가는지, 같은 음에서 머무는지 큰 흐름만 잡아도 시창이 훨씬 수월해짐.
- 분리 연습: 선율이 어렵다면 템포를 늦추거나 구간을 나눔. 리듬이 문제일 때는 음정 없이 리듬만 읽고, 음정이 문제일 때는 리듬을 단순화해 음정만 불러봄.
<p style="margin: 0 0 1em 0;">대학 음악 교육과정에서도 여러 학기에 걸쳐 단계적으로 다루는 어려운 테크닉이므로, 단기간에 익히려 서두르기보다 올바른 방법으로 꾸준히 연습하는 것이 중요함. 연습 과정에서 막히는 부분이 있다면 혼자 넘기려 하지 말고 강사의 도움을 받는 것이 좋음.</p>

<p style="margin: 0 0 1em 0;">청음 (Dictation) 전략</p>

<p style="margin: 0 0 1em 0;">청음은 리듬, 선율, 화성 진행 등 처음 듣는 소리를 듣고 악보로 옮겨 적는 과정임.</p>

<p style="margin: 0 0 1em 0;">리듬 청음</p>

<p style="margin: 0 0 1em 0;">A rhythm for dictation.</p>

<p style="margin: 0 0 1em 0;">리듬 청음 방법 중 하나는 박자와 마디 위치에 맞춰 점을 찍으면서 시간의 기준이 되는 그리드를 만드는 것임.</p>

<img src="/images/14.-시창과-청음의-기초-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">A dot grid for four measures in common time.</p>

<p style="margin: 0 0 1em 0;">이렇게 점으로 기준을 잡은 뒤, 소리가 나는 지점에는 슬래시(/), 소리가 이어지는 구간에는 대시(—), 쉼표에는 동그라미(O)를 표시함. 이 방식을 슬래시 표기법이라 하며, 이후 이를 오선보로 옮겨 적음.</p>


<iframe src="https://musescore.com/user/32728834/scores/8455859/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">Slash notation and staff notation of Example 2.</p>

<p style="margin: 0 0 1em 0;">리듬을 들으면서 지휘를 하거나 박자를 손으로 두드리면, 음이 박자 위에 있는지, 엇박인지, 또는 쉼표인지 구분하는 데 도움이 됨.</p>

<p style="margin: 0 0 1em 0;"><strong>선율 청음</strong></p>

<p style="margin: 0 0 1em 0;">A melody with the same rhythm as Example 2.</p>

<p style="margin: 0 0 1em 0;">선율 청음은 리듬을 먼저 적는 뒤, 앞에서 설명한 리듬 청음 방식을 적용한 후, 그 위에 음높이 정보를 추가하면 됨.</p>

- 선율 윤곽 표시: 음이 올라가는지, 내려가는지, 유지되는지를 선으로 표시함.
- 도약 표시: 음이 크게 뛰는 구간에 별표 등으로 따로 표시함.
- 계이름 기록: 들리는 계이름 음절을 간단히 적어 음정을 정리함.

<p style="margin: 0 0 1em 0;">Examples 5–6 in staff notation.</p>

<p style="margin: 0 0 1em 0;"><strong>프로토노테이션 (Protonotation)</strong></p>

<p style="margin: 0 0 1em 0;">프로토노테이션은 리듬과 선율을 받아적는 기초 표기 방식으로, 게리 카핀스키(Gary Karpinski)가 저서 Manual for Ear Training and Sight Singing에서 소개함. 이후 많은 학생들이 리듬·선율 청음의 보조 도구로 활용함.</p>


<iframe src="https://musescore.com/user/32728834/scores/8455748/embed" style="width:100%; height:450px !important; border:none; display: block;" frameborder="0" allowfullscreen allow="autoplay; fullscreen"></iframe>

<p style="margin: 0 0 1em 0;">A melody in protonotation and staff notation.</p>

<p style="margin: 0 0 1em 0;">프로토노테이션은 박자의 정확한 길이나 조성 정보를 기록하기보다는, 음의 높낮이와 리듬의 기본 형태를 빠르게 정리하기 위해서 사용함.</p>

<p style="margin: 0 0 1em 0;">프로토노테이션의 구성</p>

<p style="margin: 0 0 1em 0;">먼저 박자에 맞춰 그리드를 만듦. 그리드의 형태는 박자 유형에 따라 달라짐.</p>

- 2박자 / 3박자: 강박은 긴 수직선, 약박은 짧은 수직선으로 표시함.
- 4박자: 1박은 긴 선, 3박은 중간 길이 선, 2박과 4박은 짧은 선으로 표시함.
<img src="/images/14.-시창과-청음의-기초-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">두 박자 계열</p>

<img src="/images/14.-시창과-청음의-기초-5.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">세 박자 계열</p>

<img src="/images/14.-시창과-청음의-기초-6.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">네 박자 계열</p>

<p style="margin: 0 0 1em 0;">음은 리듬의 길이를 나타내는 수평선과 이동도법 계이름(또는 음계 차수)으로 적음. 선율이 크게 도약하는 부분은 화살표로 표시함. 쉼표는 빈칸이나 X로 적어, 아직 기록하지 않은 부분과 쉼표를 구분함.</p>

<p style="margin: 0 0 1em 0;"><strong>프로토노테이션을 오선보로 옮기기</strong>
프로토노테이션은 음자리표, 으뜸음, 그리고 박자 단위(또는 박자표의 분모)를 알고 있으면 오선보로 옮길 수 있음.</p>

- 먼저 주어진 정보에 따라 음자리표와 조표, 박자표를 적음.
- 프로토노테이션에서 가장 긴 수직선은 오선보의 마디줄에 옮겨 적음.
- 그다음 음역과 계이름, 으뜸음을 고려해 각 음의 높이를 정함.
- 마지막으로 박자표를 기준 삼아 프로토노테이션의 리듬을 실제 음표 길이로 바꿈.
예를 들어 4/4박자에서는 2박이 2분음표가 되고, 2/2박자에서는 온음표가 됨.
<p style="margin: 0 0 1em 0;">음자리표나 으뜸음, 박자 정보가 없으면 같은 프로토노테이션이 여러 가지 악보로 해석될 수 있음.</p>

<p style="margin: 0 0 1em 0;">프로토노테이션으로 표기한 세 가지 버전</p>

<p style="margin: 0 0 1em 0;">시창이나 리듬 읽기와 마찬가지로 청음도 익숙해지기까지 시간이 걸림. 서두르지 말고 올바른 방식으로 계속 연습하는 것이 중요하며, 막히는 부분이 있으면 혼자 끌고 가지 말고 도움을 받아야 함.</p>
