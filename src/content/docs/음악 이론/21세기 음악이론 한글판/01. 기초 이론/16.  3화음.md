### <strong>3화음 (Triads)</strong>

<p style="margin: 0 0 1em 0;">3화음은 세 음을 3도씩 차례로 쌓아 만든 화음임. 그래서 악보에 적으면 세 음이 모두 줄이거나 모두 칸에 놓임.
이 세 음을 3도씩 바로 위에 겹쳐 적으면 아래에서 위로 포개진 모양이 되어 눈사람처럼 보임. 아래에서부터 차례로 아래음·가운데음·위음이라 부름.</p>

<p style="margin: 0 0 1em 0;"><strong>[Example 1. 악보: 선율적/화성적으로 배치된 두 개의 3화음]</strong></p>

<p style="margin: 0 0 1em 0;"><strong>[Example 2. 그림: 눈사람 모양의 비유]</strong></p>

<p style="margin: 0 0 1em 0;">3화음이 이처럼 3도씩 겹쳐 쌓인 상태를 기본 위치(root position)라 함. (더 자세한 내용은 전위에서 다룸.) 
그림과 같이 화음을 구성하는 각 음에는 고유한 이름이 있음.
근음: 가장 아래에 있는 음
3음: 근음에서 3도 위에 있는 음
5음: 근음에서 5도 위에 있는 음
[예제 3. 근음·3음·5음이 표시된 3화음]</p>


### <strong>3화음의 성질과 청음 (Triadic Qualities and Listening)</strong>

<p style="margin: 0 0 1em 0;">3화음의 성질은 장·단·감·증의 네 가지로 나뉨. 근음에서 3음까지, 그리고 근음에서 5음까지의 <strong>음정 관계</strong>에 따라 결정됨.
장 3화음과 단 3화음은 3음의 <strong>성질</strong>에 따라 구분함. 두 화음 모두 5음은 완전5도임.
감 3화음과 증 3화음은 5음의 <strong>성질</strong>에 따라 구분함. 감 3화음은 단3도와 감5도로 이루어지고, 증 3화음은 장3도와 증5도로 이루어짐.</p>

<p style="margin: 0 0 1em 0;">소리를 들을 때 각 성질에 감정을 대입해 보면 구별하기 쉬움.</p>

<p style="margin: 0 0 1em 0;">장 3화음은 밝은 느낌, 단 3화음은 어두운 느낌, 감 3화음은 긴장된 느낌, 증 3화음은 불안정하고 떠 있는 느낌으로 들림.</p>


### <strong>코드 기호 (Chord Symbols)</strong>

<p style="margin: 0 0 1em 0;">3화음의 코드 기호는 <strong>근음의 알파벳</strong>과 <strong>성질</strong>을 조합하여 표기함. 이 교재에서는 다음과 같은 약어를 사용함.</p>

- <strong>장 3화음:</strong> 근음만 표기 (예: <strong>C</strong>)
- <strong>단 3화음:</strong> 소문자 <strong>mi</strong> (예: <strong>Cmi</strong>)
- <strong>감 3화음:</strong> 위첨자 원 <strong>°</strong> (예: <strong>C°</strong>)
- <strong>증 3화음:</strong> 더하기 기호 <strong>+</strong> (예: <strong>C+</strong>)
<p style="margin: 0 0 1em 0;">만약 근음에 샵이나 플랫이 있다면 기호에도 표시해야 함. (예: <strong>B♭mi</strong>, <strong>F♯°</strong>)</p>

<p style="margin: 0 0 1em 0;">경우에 따라 베이스 음(최저음)을 따로 표시하기도 하는데, 슬래시(/)를 긋고 그 베이스 음을 적음. 아래 악보의 C 단 3화음의 베이스가 E♭일 때와 <strong>Cmi/E♭</strong>, G일 때 <strong>Cmi/G</strong>로 표기 참고.</p>

<p style="margin: 0 0 1em 0;"><strong>[Example 5. 악보: 코드 기호와 슬래시 코드 예시]</strong></p>


### <strong>장조와 단조에서의 3화음 (Triad Qualities in Major and Minor)</strong>

<p style="margin: 0 0 1em 0;">장음계나 단음계의 모든 음 위에는 3화음을 쌓을 수 있음.</p>

<p style="margin: 0 0 1em 0;"><strong>장음계 (Major Scale)</strong> </p>

<p style="margin: 0 0 1em 0;">아래 악보는 G 장조의 예시임. 장조에서는 음계의 각 음 위에 다음과 같은 성질의 3화음이 놓이고, 이 패턴은 모든 장조에서 같음</p>

- <strong>장 3화음:</strong> 도, 파, 솔 (I,IV,V)
- <strong>단 3화음:</strong> 레, 미, 라 (ii,iii,vi)
- <strong>감 3화음:</strong> 시 (vii∘)
<p style="margin: 0 0 1em 0;"><strong>[Example 6. 악보: 장조의 3화음 패턴]</strong></p>

<p style="margin: 0 0 1em 0;"><strong>단음계 (Minor Scale)</strong> </p>

<p style="margin: 0 0 1em 0;">아래 악보는 G단조의 예시임. 단조는 이끄는음을 올리는 경우가 있으므로 자연단음계와 화성단음계를 함께 보아야 함.</p>

- <strong>단 3화음:</strong> 도, 파, 솔(이끔음 미적용 시) (i,iv,v)
- <strong>장 3화음:</strong> 메, 르, 테(이끔음 미적용 시), 그리고 <strong>솔</strong>(이끄는음 적용 시) (III,VI,VII,V)
- <strong>감 3화음:</strong> 레, 그리고 <strong>티</strong>(이끔음 적용 시) (ii∘,vii∘)
<p style="margin: 0 0 1em 0;"><strong>[Example 7. 악보: 단조의 3화음 패턴 (자연/화성 단음계 포함)]</strong></p>


### 3화음 구성하기

<p style="margin: 0 0 1em 0;">코드 기호를 보고 3화음을 적는 순서.</p>

1. 근음을 오선에 적음.
1. 근음 위에 3도와 5도를 쌓음.
1. 근음에 해당하는 장조의 조표를 확인.
1. 조표를 적용하여 장 3화음을 완성.
1. 단·감·증 3화음일 경우, 장 3화음을 기준으로 3음과 5음을 반음 조정함.
<p style="margin: 0 0 1em 0;"><strong>예시 1: D 장 3화음</strong></p>

<p style="margin: 0 0 1em 0;">근음 D를 적고, 위에 F와 A를 쌓음.</p>

<p style="margin: 0 0 1em 0;">D장조의 조표는 F♯, C♯이므로 F에 ♯을 붙임.</p>

<p style="margin: 0 0 1em 0;">D–F♯–A가 됨.</p>

<p style="margin: 0 0 1em 0;"><strong>예시 2: A♭ 단 3화음</strong></p>

<p style="margin: 0 0 1em 0;">근음 A♭을 적고, 위에 C와 E를 쌓음.</p>

<p style="margin: 0 0 1em 0;">A♭장조의 조표를 적용하면 A♭–C–E♭이 됨.</p>

<p style="margin: 0 0 1em 0;">단 3화음이므로 3음 C를 반음 내려 C♭으로 고침.</p>

<p style="margin: 0 0 1em 0;">A♭–C♭–E♭이 됨.</p>

- 3화음을 적을 때 감 3화음은 장 3화음에서 3음과 5음을 반음 내리고, 증 3화음은 5음을 반음 올림.
