<p style="margin: 0 0 1em 0;">21세기 음악이론 한글판 <span class="notion-text-gray">
(dcinside digital piano gallery)</span></p>

<p style="margin: 0 0 1em 0;">같은 소리 내는데 악보에선 다르게 표기하는 음을 이명동음(enharmonic notes)이라 함.</p>

<img src="/images/1.5-이명동음-0.svg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">C♯과 D♭은 피아노에서 같은 건반이지만, 악보엔 다르게 쓰는 것처럼 소리는 같고 표기만 다른 음임.</p>

<p style="margin: 0 0 1em 0;">피아노 검은 건반은 모두 이명동음 관계임</p>

- C♯ = D♭
- D♯ = E♭
- F♯ = G♭
- G♯ = A♭
- A♯ = B♭
<p style="margin: 0 0 1em 0;">흰 건반도 이명동음인 경우가 있음.</p>

- C = B♯
- E = F♭
- D = C𝄪
