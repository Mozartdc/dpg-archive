<p style="margin: 0 0 1em 0;"><span class="notion-text-gray">by Robert Hutchinson </span></p>

---

<p style="margin: 0 0 1em 0;"><span class="notion-text-purple">21세기 음악이론 한글판 </span><span class="notion-text-gray">
(dcinside digital piano gallery)</span></p>

<p style="margin: 0 0 1em 0;">장음계는 온음(W)과 반음(H)이 일정한 순서로 늘어선 음계임. 테트라코드, 즉 4음짜리 덩어리 두 개로 끊어서 보면 구조가 한눈에 들어옴.</p>

<p style="margin: 0 0 1em 0;">하위 테트라코드는 온음–온음–반음으로 가고, 상위 테트라코드도 똑같이 온음–온음–반음으로 감. 둘 사이는 온음 하나로 이어짐.</p>

<p style="margin: 0 0 1em 0;">정리하면, <strong>W–W–H ／ W ／ W–W–H </strong></p>

<img src="/images/2.2-장음계-1.svg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">음계를 구성할 때는 음 이름을 순서대로 써야 함. 같은 음이 두 번 나오거나 중간 음을 생략하면 틀림. (ex. D–E–G–G → X)</p>

<p style="margin: 0 0 1em 0;">모든 장조의 스케일은 이 구조를 따라라므로, 테트라코드 개념을 알면 다른 조도 쉽게 만들 수 있음.</p>
