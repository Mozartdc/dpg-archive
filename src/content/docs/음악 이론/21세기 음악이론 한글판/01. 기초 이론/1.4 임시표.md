---

<p style="margin: 0 0 1em 0;">21세기 음악이론 한글판 <span class="notion-text-gray">
(dcinside digital piano gallery)</span></p>

<p style="margin: 0 0 1em 0;">임시표는 음의 높낮이를 일시적으로 바꾸는 기호임. </p>

- 샵(♯): 음을 반음 올림
- 플랫(♭): 음을 반음 내림
- 더블샵(?): 음을 온음(두 반음) 올림
- 더블 플랫(?): 음을 온음(두 반음) 내림
- 내추럴(♮): 앞에 나온 임시표 효과 취소하고 원래 음으로 돌림
<p style="margin: 0 0 1em 0;">임시표는 보통 한 마디 안에서만 적용되고, 같은 높이의 음에만 영향 줌.</p>

<p style="margin: 0 0 1em 0;">조표는 임시표와 달리 곡 전체에 영향을 줌</p>
