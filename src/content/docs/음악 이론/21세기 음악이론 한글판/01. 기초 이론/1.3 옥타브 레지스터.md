<h3 class="notion-text-gray">Music Theory for the 21st-Century
Classroom</h3>

<p style="margin: 0 0 1em 0;"><span class="notion-text-gray">by Robert Hutchinson </span></p>

---

<p style="margin: 0 0 1em 0;">21세기 음악이론 한글판 <span class="notion-text-gray">
(dcinside digital piano gallery)</span></p>

<p style="margin: 0 0 1em 0;">음악에서 사용하는 음 이름은 A, B, C, D, E, F, G의 7개임. 이를 음악적 알파벳이라고 부름. G 다음에는 다시 A가 오고, 이 순서는 계속 반복됨.</p>


| A | B | C | D | E | F | G | A |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 (octave) |

<p style="margin: 0 0 1em 0;">같은 이름의 음 사이의 거리를 옥타브라고 함. 예를 들어 어떤 C에서 다음 C까지가 한 옥타브임. 옥타브는 음 이름 기준으로 8도 차이가 나기 때문에 줄여서 “<em><strong>8ve</strong></em>”라고도 씀.</p>

<p style="margin: 0 0 1em 0;">
피아노 건반에는 88개의 건반이 있지만, 음 이름은 7개뿐임. 이는 음악적 알파벳이 여러 옥타브에 걸쳐 반복되기 때문임. 이 반복되는 각 옥타브 구간을 옥타브 레지스터라고 부름. 각 레지스터는 7개의 자연음과 그 사이에 있는 5개의 반음계 음으로 구성됨.</p>

<img src="/images/1.3-옥타브-레지스터-0.svg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">피아노의 음역을 설명할 때는 옥타브 레지스터 번호를 사용함. 이때 기준이 되는 음은 미들 C임. 미들 C를 C4라고 부르고, 이를 기준으로 위쪽은 C5, C6처럼 올라가고 아래쪽은 C3, C2처럼 내려감. 레지스터 번호는 B 다음에 오는 C에서 바뀜.</p>

<p style="margin: 0 0 1em 0;">
높은음자리표에서는 미들 C가 오선 아래 덧줄에 있고, 낮은음자리표에서는 오선 위 덧줄에 있음. </p>

<p style="margin: 0 0 1em 0;">알토 음자리표와 테너 음자리표는 C 음자리표를 사용하며, 기호의 위치가 미들 C의 위치가 됨.</p>

<p style="margin: 0 0 1em 0;">피아노에서 사용하는 대보표는 높은음자리표와 낮은음자리표를 중괄호로 묶은 표기법임. </p>

<img src="/images/1.3-옥타브-레지스터-1.svg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">미들 C는 항상 위나 아래 오선에 덧줄로 표기되며, 두 오선 사이의 빈 공간에 떠서 표기되지는 않음.
</p>
