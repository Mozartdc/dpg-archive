<p style="margin: 0 0 1em 0;">건반 위에 손을 올릴 때는 불필요한 근육의 개입을 최소화하는 것이 관건임. 허벅지 위에 손을 편안하게 둔 상태에서 팔뚝(Forearm)의 힘으로만 들어 올리되, 손목 아래로는 힘을 완전히 빼야 함. 이때 손등이 검지 마디(Knuckle)보다 높게 들리지 않도록 주의하면, 나머지 손가락은 중력에 의해 자연스럽게 아래로 매달리는 형태가 됨.</p>

<p style="margin: 0 0 1em 0;">팔뚝을 들어 올리는 주된 동력은 상완요골근(Brachioradialis)에서 나옴. 이 근육은 팔꿈치 뒤쪽에서 시작해 손목 근처까지 이어져 하중을 정교하게 조절함. 연주 전 이미 신체적·심리적 긴장이 높은 경우가 많으므로, 팔을 들어 올릴 때 어깨나 손목에 불필요한 힘이 들어가는지 점검함으로써 현재의 긴장 상태를 즉각 파악할 수 있음.</p>

<img src="/images/레슨-(2)-피아노-연주-자세-1.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">손의 회전은 손 자체가 아닌 팔뚝 내부의 골격 구조에서 시작됨. 팔뚝을 구성하는 요골(Radius)이 고정된 척골(Ulna)을 축으로 삼아 그 위를 교차하며 회전하는 것이 정상적인 기전임. 만약 축이 되어야 할 척골이 함께 흔들리면 비효율적인 움직임이 발생하므로 주의해야 함.</p>

<img src="/images/레슨-(2)-피아노-연주-자세-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">팔뚝의 회전은 팔꿈치 안쪽에서 요골을 당겨주는 원회내근(Pronator Teres)과 손목 부근의 방회내근(Pronator Quadratus)에 의해 이루어짐. 특히 방회내근은 잘못된 자세로 연습할 경우 약해지거나 짧아지기 쉬운 부위임. 만약 회전 반경이 충분히 확보되지 않으면 약지(4번)와 새끼손가락(5번)의 독립적인 움직임이 제한되어 원활한 연주를 방해함.</p>

<img src="/images/레슨-(2)-피아노-연주-자세-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">회전 각도가 좁으면 이를 보충하기 위해 어깨가 위로 들리거나 팔꿈치가 몸 바깥으로 벌어지는 보상 작용이 나타남. 이러한 부자연스러운 자세는 어깨뼈 주변과 등 근육까지 긴장 상태로 끌어들여 전신의 경직을 초래함. 따라서 무리하게 근육을 단련하기보다, 안정적인 회전 한계를 유지하며 편안하게 가동 범위를 넓히는 연습이 필요함.</p>

<p style="margin: 0 0 1em 0;">피아노 연주 시의 신체 움직임은 촉각과 결합된 감각 기억으로 뇌에 각인됨. 손가락이 건반에 닿는 순간의 자세가 무의식적으로 고정되므로, 잘못된 습관이 형성된 후에는 올바른 자세로 교정하려 해도 신체가 이를 거부하게 됨. 초기부터 올바른 감각 기억을 형성하는 것이 중요한 이유도 여기에 있음.</p>

<p style="margin: 0 0 1em 0;">쇼팽의 교수법은 신체의 자연스러운 물리적 구조를 따르는 데 목적이 있음. 손은 자체의 하중만 유지한 채 불필요한 긴장을 배제하며, 팔뚝이 주된 동작과 회전을 담당하여 균형을 잡는 것이 핵심임. 이러한 방식은 단기적으로 기교를 향상할 뿐 아니라, 장기적으로는 손의 피로를 줄이고 부상을 예방하는 가장 확실한 길임.</p>
