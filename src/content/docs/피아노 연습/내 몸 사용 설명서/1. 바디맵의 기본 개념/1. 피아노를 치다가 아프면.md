<p style="margin: 0 0 1em 0;">내가 건반을 누르는 힘만 쓰는지, 아니면 버티는 힘도 같이 쓰는지 생각해봐야 함.</p>

<p style="margin: 0 0 1em 0;">그렇게 세게 누르지 않아도 건반을 누를 수 있고, 손가락을 올려두고 있으면 건반은 안 올라옴.</p>

<p style="margin: 0 0 1em 0;">그런데, 움직이지 않으려고 버티는데 힘이 많이 쓰기도 함.</p>

<p style="margin: 0 0 1em 0;">버티는 힘은 불필요한데다 손목, 팔, 어깨, 등까지 힘이 들어감.</p>

<p style="margin: 0 0 1em 0;">피아노는 반복 운동이긴 하지만, 자전거 페달링이나 다른 운동처럼</p>

<p style="margin: 0 0 1em 0;">반복 자체는 문제가 없음.</p>

<p style="margin: 0 0 1em 0;">피아노를 치다가 아프면, 많이 쳐서 아픈게 아니라 잘 못 쳐서 아픈것임.</p>

<p style="margin: 0 0 1em 0;">피아노를 80넘어서도 계속 치려면, 손가락과 팔을 더 강하게 만드는 연습이 아니라, 쓸데 없는 움직임을 덜어내는 법을 배워야 함.</p>
