<p style="margin: 0 0 1em 0;">바디맵핑은 신체구조 정보를 몸의 감각으로 바꾸어 움직임에 연결하는 작업임. </p>

<p style="margin: 0 0 1em 0;">바디맵은 그냥 아는 데서 끝나는 정보가 아니라, 몸에서 직접 느껴지고 연주로 이어져야 함.</p>

<p style="margin: 0 0 1em 0;">Taubman 역시 원리를 이해하는 데서 그치지 않고, 손과 팔이 그 원리를 따라 자연스럽게 움직여야 한다고 강조함. </p>

<p style="margin: 0 0 1em 0;">원리가 몸에 익으면 관절에 무리가 덜 가고, 힘도 한곳에 몰리지 않게 됨. 그러면 불필요한 피로가 줄고, 어려운 패시지도 훨씬 편하게 풀어낼 수 있음.</p>


### 무게를 지지하고 전달하기

<p style="margin: 0 0 1em 0;">머리와 척추가 중력선에 맞게 균형을 이루면, 뼈대가 대부분의 체중을 지지하게 됨.</p>

<p style="margin: 0 0 1em 0;">이럴 땐 서 있든 앉아 있든 “억지로 자세를 만드는 느낌”이 아니라, 그냥 있어도 편안하고 호흡과 손 움직임이 자연스러움.</p>

<p style="margin: 0 0 1em 0;">반대로 머리가 앞으로 쏠리거나, 골반이 뒤로 말리거나, 척추 정렬이 무너지면, 뼈대가 아니라 근육이 구조를 지지하게 됨.</p>

<p style="margin: 0 0 1em 0;">그러면 목, 어깨, 허리, 팔 근육이 계속 잡고 버텨야 해서 긴장, 통증, 피로가 생기고 손가락 움직임도 무겁고 거칠어짐.</p>



<img src="/images/01.-바디매핑-1.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">그래서 좋은 자세를 억지로 유지한다기보다,</p>

1. 골반, 좌골, 척추, 머리의 위치를 해부학적으로 정확히 느끼고,
1. 그 뼈대 위에 체중을 맡기면, 근육이 버텨야 할 부담이 줄어 불필요한 긴장이 줄어듦.
<p style="margin: 0 0 1em 0;">피아노에서는 이걸 구체적으로</p>

- 좌골로 의자에 안정적으로 앉고,
- 척추를 위로 길게 세우되 힘주지 않고,
- 팔의 무게를 어깨와 쇄골에서 손가락 끝까지 걸어 두는 느낌으로 쓰는 것을 말함.
<p style="margin: 0 0 1em 0;">이렇게 뼈대의 지지에 의존하면서 손, 팔, 어깨, 척추 전체를 포괄적으로 느끼도록 주의하면, 손가락 힘을 덜 써도 소리가 잘 나고 빠른 움직임도 훨씬 수월해짐.</p>

<p style="margin: 0 0 1em 0;">구조적 균형이 잡히면 근육을 버티는 데 덜 쓰고 움직이는 데 더 쓸 수 있게 됨.</p>

<p style="margin: 0 0 1em 0;">정리하면,</p>

- 자세를 억지로 만드는 게 아니라, 구조적 균형이 잡히면 좋은 자세가 저절로 잡히는 듯 느껴짐.
- 구조가 무너지면 근육이 대신 버티게 되어 긴장됨.
- 뼈대와 그 지지를 감각적으로 인지하고 거기에 몸무게를 맡기면 근육 긴장이 줄고, 움직임과 피아노 연주도 실제로 더 수월해질 수 있음.
