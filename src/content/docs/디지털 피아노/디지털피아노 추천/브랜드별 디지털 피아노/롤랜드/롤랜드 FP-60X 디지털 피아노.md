<p style="margin: 0 0 1em 0;">2021년 1월 11일 Roland가 FP-X 시리즈를 공식 발표했고, 글로벌 출시는 2021년 2월부터 시작됨. 국내에서는 코스모스악기가 2021년 02월 22일 FP-60X를 정식 수입함.</p>


### <strong>건반 / 사운드</strong>

<p style="margin: 0 0 1em 0;">FP-60X는 FP-30X와 동일한 PHA-4 Standard 액션을 사용함. PHA-4 건반은 타 브랜드 동급 기종 중 비교적 무거운 타건감을 가지며, 복원력이 안정적이고 이스케이프먼트 구조를 구현해 실제 피아노에 가까운 반응을 제공함. 건반을 끝까지 눌렀을 때 그랜드 피아노에서 느껴지는 미세한 노치감도 표현됨.</p>

<p style="margin: 0 0 1em 0;">다만 FP-60X는 FP-30X와 같은 건반 액션을 사용하지만, 하우징 구조가 다름. FP-30X는 비교적 가벼운 슬랩형 섀시를 사용하는 반면 FP-60X는 더 단단한 상위 섀시 구조를 사용하며 전면 스피커 구조와 보강된 프레임을 갖고 있음. 이 때문에 기본 액션은 동일하지만 실제 연주 시 체감되는 안정감과 타건 반응은 다소 차이가 있을 수 있음. 일반적으로 FP-60X가 더 단단하고 안정적인 타건감을 준다는 평가가 많음.</p>

<p style="margin: 0 0 1em 0;"><strong>벨로시티 커브는 FP-30X가 5가지 프리셋 방식인 것과 달리, FP-60X는 100단계 조절이 가능해 연주 반응을 훨씬 세밀하게 설정할 수 있음. 여기에 Hammer Response 10단계를 지원해 연주 반응을 훨씬 세밀하게 조절할 수 있으므로, 실제 체감 타건감은 같다고 보기 어려움</strong></p>

<p style="margin: 0 0 1em 0;">사운드는 Roland의 SuperNATURAL 음원 엔진을 사용함. 샘플링과 물리 모델링을 결합한 방식으로 연주 강약에 따라 자연스럽게 반응하는 것이 특징임. FP-60X는 새로운 사운드 칩셋이 적용되어 전작보다 음색 밀도와 밸런스가 개선되었으며, 동시발음도 256음으로 증가함.</p>

<p style="margin: 0 0 1em 0;">실제 피아노 음색은 FP-30X와 매우 유사한 계열로 평가되는 경우가 많으며, 헤드폰 환경에서는 두 모델의 차이를 크게 느끼기 어렵다는 의견도 있음. 다만 FP-60X는 더 큰 하우징과 전면 스피커 구조를 사용하기 때문에 스피커로 연주할 때는 공간감과 음압에서 차이가 발생함.</p>


### <strong>디자인 및 조작부</strong>

<p style="margin: 0 0 1em 0;">외형은 FP-90X의 하우징을 사용함. 전면 스피커 구조와 보강된 섀시를 적용해 전체적인 구조 강성이 높음. 색상은 검정색과 흰색 두 가지로 출시됨.</p>

<p style="margin: 0 0 1em 0;">본체 마감은 무광 플라스틱 중심이지만 전체 구조가 단단하고 내부 프레임 강성이 높아 안정감 있는 느낌을 줌.</p>

<p style="margin: 0 0 1em 0;">FP-60X은 대부분의 기능에 전용 버튼이 있고, 다이브가 필요한 기능도 LCD 화면 덕분에 앱 없이도 쓰기 편함. </p>

<p style="margin: 0 0 1em 0;">버튼 조작감은 탄성 코일 스프링을 넣어서 눌렀을 때 반응이 좋고, 버튼 아래 라벨도 고대비 색상과 읽기 쉬운 글꼴을 써서 가독성이 좋음.</p>

<p style="margin: 0 0 1em 0;">페이더와 슬라이더도 잘 만들어졌음. 적당한 저항감이 있어서 미세 조정이 가능하고, 특히 레이어 음색 간 밸런스를 잡을 때 배치가 잘 되어 있음.</p>

<p style="margin: 0 0 1em 0;">FP-30X에서 특히 싫었던 부분이 버튼식 볼륨 조절이었는데, FP-60X는 전용 볼륨 페이더가 있어서 훨씬 낫게 느껴짐.</p>

<p style="margin: 0 0 1em 0;">좌측 상단의 3밴드 EQ 페이더도 훌륭함. 이런 기능은 더 많은 회사가 넣어 줬으면 좋겠음. EQ는 소리의 주파수 밸런스를 바로 만질 수 있게 해 주는데, 메뉴 깊숙이 들어가지 않고 전면에서 바로 조절할 수 있다는 점이 실전에서는 꽤 큼.음.</p>

<p style="margin: 0 0 1em 0;">보면대는 기본 포함이며 전용 스탠드(KSC-72)를 별도로 구매할 수 있음. 전용 스탠드와 함께 사용할 경우 3페달 유닛(KPD-90)을 장착해 Damper, Sostenuto, Soft 페달을 모두 사용할 수 있음.</p>

<p style="margin: 0 0 1em 0;">후면에는 전원 어댑터 단자와 페달 단자가 있음. 기본 구성은 DP-10 댐퍼 페달이며 하프 페달을 지원함. 추가로 Sostenuto와 Soft 페달 단자가 있어 3페달 구성이 가능함.</p>

<p style="margin: 0 0 1em 0;">연결 단자는 스테레오 라인 출력(6.3mm L/Mono, R), USB Computer(Type B), USB Memory(Type A), 오디오 입력(3.5mm), 헤드폰 단자 2개(6.3mm, 3.5mm), 마이크 입력 단자를 제공함. Bluetooth MIDI와 Bluetooth Audio도 지원함.</p>
