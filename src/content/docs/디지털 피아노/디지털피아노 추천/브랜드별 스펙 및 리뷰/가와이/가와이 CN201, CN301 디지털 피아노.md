CN201과 CN301은 2022년 6월 1일에 나온 모델인데, 전작인 CN29와CN39의 후속기라기 보다, Onkyo 관련 부품 공급 문제로 사운드 시스템을 새로 잡으면서 모델명이 바뀐 경우임.
원래 CN29와 39를 1~2년 정도 더 이어갈 수도 있었지만, 가와이의 주요 사운드 칩 공급처였던 Onkyo의 파산으로 인해 관련한 부품을 새로 정리한 모델임.

건반 / 사운드
CN201과 CN301은 모두 RHIII,(Responsive Hammer III) 건반을 사용함. 88건반 그레이드 해머 구조이고, Ivory Touch 흰 건반 표면, 렛오프 시뮬레이션, 트리플 센서, 카운터웨이트가 들어감. CN29와 CN39도 같은 RHIII 건반을 사용했음.



<img src="/images/가와이-CN201,-CN301-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

가와이의 해머액션은 RHIII부터 렛오프 기능이 들어감.

CN201, CN301으로 모델명을 바꾸면서 사운드 구성, 스피커, 녹음, 연결성, 기능이 바뀌었는데, 스피커와 헤드폰 처리 변경으로 특히 헤드폰에서 체감이 좋아졌다는 평이 있음. 

하지만, Onkyo 협업 오디오 설계에서 가와이 자체 설계로 넘어가서 일부 오디오 회로 쪽은 다운그레이드라고 생각됨. 물론, Bluetooth Audio, 앱 연동, 새로으운 조작부, 센서 처리는 출시 시기에 맞게 업그레이 됨.

사운드는 둘 다 Progressive Harmonic Imaging, PHI 기반임. CN201은 SK-EX, EX, K-60을 중심으로 하고, CN201기준 동시발음은 192음, 음색 수는 19개임.

CN301은 201에 비교해 SK-EX, SK-5, EX, K-60 음원이 추가되고, 최대 동시발음도 256음, 음색 수는 45개임.

CN201은 12cm 풀레인지 스피커 2개에 40W 출력이고, CN301도 총출력은 같지만, 13cm 저중역 스피커 2개와 8 x 12cm 중고역 디퓨저 스피커 2개를 넣은 4스피커 구조임. 출력은 같아도, CN301이 음역 분리와 소리 확산에서 더 향상된 구성임.

디자인 / 조작부
CN201은 폭 136cm, 깊이 40.5cm, 높이 86cm, 무게 43kg임. CN301은 폭 141cm, 깊이 43.5cm, 높이 87cm, 무게 47.5kg임. 4스피커 운용으로 301이 캐비닛이 더 큼.

조작부는 두 모델 모두 OLED 디스플레이를 사용함. (조작부는 29,39와 비교해 큰 변화는 없음)

CN201은 메트로놈 박자를 1/4, 2/4, 3/4, 4/4, 5/4, 3/8, 6/8 7종만 지원하지만, 301은 7/8, 9/8, 12/8을 더해 10종 지원이고 드럼 리듬도 100종 추가 됨.

내장곡도 CN201은 데모 19곡, Concert Magic 40곡, 레슨 6권 구성인데, CN301은 데모 33곡, Concert Magic 176곡, 레슨 12권 구성임.

두 모델 모두 Bluetooth MIDI와 Bluetooth Audio를 모두 지원하고, USB TO HOST로 컴퓨터나 스마트 기기와 연결할 수 있음. 여기에 CN301은 USB 메모리 단자, Line In, Line Out이 추가 됨

경쟁모델

Yamaha YDP-165
