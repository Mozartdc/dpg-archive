ES520은 2020년 9월 9일 ES920과 함께 발표한 모델로,  기존 ES 라인에 새롭게 추가된 중급 기종임. 롤랜드의 FP60x처럼 앤트리 라인의 액션을 상급기 섀시에 창착한 제품임.



<img src="/images/가와이-ES520-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

건반 / 사운드

ES520은 Responsive Hammer Compact II 건반을 사용함. 88건반 그레이드 해머 구조에 트리플 센서를 넣었고, ES120의 RHC에 트리플 센서 해머 디텍션 추가한 액션임. 건반 구조는 여전히 컴팩트 계열이지만, 빠른 반복 타건과 약한 터치 반응이 더 정교해짐



<img src="/images/가와이-ES520-디지털-피아노-2.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

사운드는 SK-EX와 EX 콘서트 그랜드가 메인이며  Progressive Harmonic Imaging 음원을 쓰고, 88건반 스테레오 샘플링과 최대 192음 동시발음을 지원함. ES120보다 상위 음원으로 올라가고, 피아노 음색의 밀도와 표현 폭도 더 넓어짐.

스피커는 Onkyo와 협업한 오디오 설계로 20W x 2, 총 40W 출력임.

디자인 / 조작부

ES520은 폭 1340mm, 깊이 375mm, 높이 145mm, 무게 14.5kg의 포터블 모델임. ES920과 같은 큰 새시를 사용하고, 중앙 OLED 디스플레이와 물리 버튼 중심 조작부를 부분적으로 공유함. 전체적인 디자인도 지나치게 번잡하지 않으면서도 중급기다운 완성감이 느껴짐.



<img src="/images/가와이-ES520-디지털-피아노-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/가와이-ES520-디지털-피아노-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

OLED 디스플레이를 통해 본체에서 음색과 메뉴를 바로 확인할 수 있음. 화면 유무는 기능을 자주 바꾸거나 빠르게 조작할 때 체감 차이가 큼. 



<img src="/images/가와이-ES520-디지털-피아노-5.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

Dual, Split, Four Hands를 지원하고, 34음색, 100종 리듬, 등록 메모리, 내장 녹음 기능을 갖춤. Bluetooth MIDI와 Bluetooth Audio도 지원하고, USB TO HOST, MIDI IN/OUT, Line In, Line Out까지 들어감. 

타사 비교 모델

Roland FP-60X
