P-525는 2023년 10월 5일 Yamaha가 발표한 P 시리즈 최상위 모델로, 기존 P-515의 후속기임. 국내에서는 야마하뮤직코리아가 2024년 2월 16일 출시함.



<img src="/images/야마하-P-525-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### 건반 / 사운드

P-525는 GrandTouch-S 건반을 사용함. 흰 건반에 목재를 쓰고, synthetic ebony ivory key tops와 escapement를 적용한 구조로, P-515의 NWX보다 센서와 전체 응답성이 개선되었음. 해머 무게 배분과 반발 구조, 스톱퍼를 최적화해 더 자연스러운 그레이드 감각과 안정된 바닥 타건감을 구현한 액션임. 흰 건반과 검은 건반의 피벗 위치도 다르게 설계되었고, 반복 타건 반응을 빠르고 안정적으로 구현함.



<img src="/images/야마하-P-525-디지털-피아노-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

터치 감도는 Hard2, Hard1, Medium, Soft1, Soft2, Fixed로 총 6단계임. 전작보다 조절 폭이 넓고, 연주감도 더 안정적이고 세밀한 컨트롤이 가능해짐. 다만 P-515의 NWX는 같은 시기의 디지털 피아노 중에서도 가장 무거운 축에 속했던 반면, GrandTouch-S는 더 경쾌한 타건감이라 P-515처럼 무거운 건반을 기대했던 사람들에겐 혹평도 있음.

사운드는 Yamaha CFX와 Bösendorfer Imperial 두 콘서트 그랜드를 중심으로 44개 메인 음색, 18개 Drum SFX Kit, 480개 XG 음색을 지원함. 여기에 VRM, Grand Expression Modeling, binaural sampling(CFX Grand 한정)이 들어가서, 단순 샘플 재생보다 공명과 터치 반응을 더 풍부하게 만들어 줌. 최대 동시발음은 256음임.

Grand Expression Modeling은 건반을 누르는 속도와 깊이, 릴리즈 차이에 따라 음색 반응을 바꾸는 기능이고, VRM은 damper resonance, string resonance, duplex scale, body resonance 같은 공명 요소를 시뮬레이션하는 엔진임. 


### 디자인 및 조작부

P-525는 P-515와 크기와 무게가 거의 같음. 공식 크기는 1336 x 376 x 145mm, 무게는 22kg임. P 시리즈라서 휴대형으로 분류되지만, P-225와 비교하면 훨씬 무겁고 사실상 세미 포터블에 가까움. 대신 그만큼 상위 건반, 더 큰 스피커, 더 많은 기능을 담은 구조임.

외형도 P-515 계열 디자인과 거의 비슷해서 디자인은 약간 구세대 느낌이 남지만, 대부분 기능을 버튼 한 번으로 접근할 수 있어서 실사용은 더 편한 편임. 198 x 100 도트 LCD가 들어가고, 화살표 키로 음색, 기능, 녹음, 설정 메뉴를 조작하는 배치임.



<img src="/images/야마하-P-525-디지털-피아노-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

스피커는 좌우 각각 Oval 12cm x 6cm 메인 드라이버와 2.5cm 돔 트위터를 사용하고, 출력은 20W + 6W 구성으로 양쪽에 배치됨. 전체 출력이 52W라 포터블 치고는 꽤 여유가 있고, 가정에서 쓰기에는 여유 있는 출력임.



<img src="/images/야마하-P-525-디지털-피아노-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

P-525는 Dual, Split, Duo를 모두 지원하고, Split에서 왼손 영역에 아무 음색이나 배정할 수 있음. 리듬은 40종으로, 드럼과 베이스 자동 반주가 따라붙는 간단한 arranger 기능도 들어감. 등록 메모리는 6버튼 4뱅크 구조라 자주 쓰는 설정을 저장하고 바로 불러오기에 편함. 피아노룸 기능도 지원해 피아노 음색 관련 설정을 한곳에서 따로 조절할 수 있음.

내장 녹음은 MIDI 250곡, 16트랙까지 가능하고, USB 메모리로 WAV 오디오 녹음도 지원함. USB TO HOST는 Type C이고, USB 오디오 인터페이스도 내장이라 44.1kHz 24bit stereo로 바로 DAW에 연결 가능함. 



<img src="/images/야마하-P-525-디지털-피아노-5.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

연결 단자는 헤드폰 2개, 5핀 MIDI IN OUT, AUX IN, AUX OUT, USB TO DEVICE, USB TO HOST, DC IN, 서스테인 페달, 전용 페달 유닛 단자를 제공함. 기본 페달은 FC3A이고 하프페달을 지원함. 전용 스탠드는 L-515, 3페달 유닛은 LP-1 또는 FC35를 사용함. Bluetooth는 Audio MIDI 둘 다 지원하고 Smart Pianist 앱도 연동 가능함.
