YDP-145와 YDP-S35는 2022년 4월에 나온 Yamaha Arius 시리즈 모델로, 내부 사양은 사실상 같고 캐비닛 형태만 다른 모델임. YDP-145는 일반 콘솔형, YDP-S35는 깊이를 줄인 슬림형으로 보면 됨. 
또, P-125를 캐비닛 형태로 만든 것이라고 생각해도 됨.

국내 기준, YDP-145는 전국 트레이더스에서 매년 1회에서 2회 정도 10만원대 할인 행사가 자주 있고, YDP-S35는 전국 코스트코에서 비슷한 수준의 할인 행사가 있음.



<img src="/images/야마하-YDP-145-S35-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### 건반 / 사운드

YDP-145와 YDP-S35는 GHS 건반을 사용함. 88건 풀웨이티드 구조에 matte black keytops가 들어가고, 터치 감도는 Hard, Medium, Soft, Fixed 4단계임. Yamaha 입문형에서 오래 써 온 액션이라 전체적으로 무난하고 적응이 쉬운 편임. 경쟁모델에 비교해 건반이 가볍고, 올라오는 감각이 약간 물렁한 편이고, 피벗 길이도 아주 긴 구조는 아니라 건반 안쪽을 눌렀을 때는 불편할 수 있지만, 입문 단계에서 크게 문제 되는 수준은 아님.

사운드는 Yamaha CFX를 기반으로 함. 기본적으로 CFX 콘서트 그랜드의 공간감 있는 샘플이 들어가고, key-off sample과 VRM Lite도 적용되어 전작인 YDP-144와 S34보다 공명 표현이 조금 더 좋아진 편임. 음색 수는 총 10개로 많지는 않지만, 피아노 3종, 전자피아노 2종, 오르간 2종, 하프시코드, 비브라폰, 스트링으로 기본 구성은 갖추고 있음. 최대 동시발음은 192음임.

타 브랜드와 비교 할 때 가장 큰 장점은 CFX 기본 음색임. 고음은 부드럽고 저음도 과하게 부풀지 않아 클래식이든 팝이든 무난하게 쓰기 좋고, 공간감도 잘 살아 있음. 다만 모델링 관련 세부 항목은 조절할 수 없어서, 음색을 세밀하게 조정하는 것은 한계가 있음.


### 디자인 및 조작부

YDP-145는 전형적인 콘솔형 가정용 디지털피아노 디자인임. 크기는 1357 x 422 x 815mm, 무게는 38kg이고, 가구형 피아노 느낌임. 



<img src="/images/야마하-YDP-145-S35-디지털-피아노-2.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

YDP-S35는 슬림형 모델임. 크기는 1353 x 296 x 792mm, 무게는 37kg으로 더 얇고 깊이가 짧아서 좁은 공간에서 활용도가 좋음. 뚜껑과 보면대를 하나로 합친 구조라 닫아 두면 훨씬 단정하게 보이고, 디자인은 업라이트형 가구형 보다는 모던한 느낌임. 



<img src="/images/야마하-YDP-145-S35-디지털-피아노-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

내부와 모든 기능은 YDP-S35, YDP-145가 동일하므로 선택 기준은 외형과 공간 활용성만 생각하면 됨.

두 모델 모두 3페달 일체형이고, Damper는 하프페달을 지원함. 금속 페달이라 기본적인 밟는 감각도 너무 가볍지 않은 편임. 다만 조작부는 상당히 미니멀해서 기능 버튼 수가 적고, Smart Pianist 앱으로 보완해서 써야 함. (매뉴얼을 보면서 버튼과 건반 조합으로도 조작 가능함)

스피커는 8W x 2, 12cm x 2 구성임. 스펙만 보면 약해 보일 수 있지만, 실제로는 가정 연습용으로는 충분한 편이고 방 안에서 연습하기엔 음량도 충분함. 

Dual과 Duo는 지원하지만 Split은 없음. 그래서 왼손 베이스 연습 같은 분할 연습은 어렵고, 음색 수도 적어서 피아노 기능 중심의 입문기 성격이 강함. 녹음은 1곡 2트랙 MIDI 녹음만 가능함. 대신 USB to Host를 통한 USB Audio 기능이 있어서, 컴퓨터에 연결하면 외장 오디오 인터페이스 없이도 피아노 소리를 바로 녹음할 수 있음. 이건 같은 가격대에서 꽤 괜찮은 장점임.

Smart Pianist와 연결은 가능하지만 Bluetooth는 내장되지 않음. USB to Host로 연결해야 함. 다만, 포트 위치가 건반 아래쪽이라 케이블 연결이 불편함.



<img src="/images/야마하-YDP-145-S35-디지털-피아노-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

기본으로 제공되는 HPH-50헤드폰과 BC-108피아노 의자.

Roland RP701, F701, Casio PX-870
