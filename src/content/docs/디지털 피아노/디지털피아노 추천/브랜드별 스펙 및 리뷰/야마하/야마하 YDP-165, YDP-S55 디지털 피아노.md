YDP-165와 YDP-S55는 2022년 4월에 나온 Yamaha Arius 시리즈 모델로, 내부 사양은 사실상 같고 캐비닛 형태만 다른 모델임. YDP-165는 일반 콘솔형, YDP-S55는 슬림형으로 보면 됨. 두 모델 모두 2019년형 YDP-164와 YDP-S54의 후속기이며, 핵심 변화는 VRM Lite와 Volume Limiter 추가 정도라 큰 변화는 없음.



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-1.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### 건반 사운드

YDP-165와 YDP-S55는 GH3 건반을 사용함. 88건 풀웨이티드 구조에 synthetic ebony ivory keytops가 들어가고, 터치 감도는 Hard, Medium, Soft, Fixed 4단계임. GH3는 Yamaha 중급형 이상에서 오래 써 온 액션이라 기본기가 꽤 탄탄한 편이고, 같은 가격대에서는 비교적 사실적인 건반으로 평가받는 편임. 

트리플 센서 구조라 빠른 반복 타건에서 GHS보다 확실히 유리하고, 건반 표면 질감도 손에 땀이 나는 사람에게는 꽤 도움이 됨. 다만 전반적인 무게감은 다소 무거운 편이라, 가볍고 빠른 터치를 좋아하는 사람에게는 답답하게 느껴질 수 있음. 그래도 YDP-145와 S35의 GHS와 비교하면 확실히 한 단계 위 액션이라고 체감 할 수 있음.



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

사운드는 Yamaha CFX를 기반으로 함. 기본적으로 CFX 콘서트 그랜드의 개방감 있는 샘플이 들어가고, key-off sample과 VRM Lite, Smooth Release도 적용되어 전작인 YDP-164와 S54보다 공명감과 음의 연결감이 조금 더 좋아진 편임. 음색 수는 총 10개로 많지는 않지만, 피아노 3종, 전자피아노 2종, 오르간 2종, 하프시코드, 비브라폰, 스트링으로 기본 구성은 갖추고 있음. 최대 동시발음은 192음임.


### 디자인 및 조작부

YDP-165는 전형적인 콘솔형 가정용 디지털피아노 디자인임. 크기는 1357 x 422 x 849mm, 무게는 42kg이고, 가구형 피아노임. 깊이가 있어서 보면대 구조도 안정적이고, 전체적으로 어쿠스틱 피아노와 비슷한 익숙한 인상을 줌. 색상도 YDP-S55보다 더 다양함.



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

YDP-S55는 슬림형 모델임. 크기는 1353 x 309 x 792mm, 무게는 40kg으로 더 얇고 깊이가 짧아서 좁은 공간에서 활용도가 좋음. 뚜껑과 보면대를 하나로 합친 구조라 닫아 두면 훨씬 단정하게 보이고, 디자인은 전통적인 가구형보다는 좀 더 모던한 느낌임. 



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

내부와 모든 기능은 YDP-165와 YDP-S55가 동일하므로 선택 기준은 외형과 공간 활용성만 생각하면 됨. 두 모델 모두 3페달 일체형이고 Damper는 하프페달을 지원함. 금속 페달이라 밟는 느낌도 너무 가볍지 않은 편임. 



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-5.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-6.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

조작부는 상당히 미니멀해서 기능 버튼 수가 적고, Smart Pianist 앱으로 보완해서 쓰는 편이 훨씬 편함. 물론 매뉴얼을 보면서 버튼과 건반 조합으로도 조작은 가능하지만, 세부 기능을 자주 만지는 사람에게는 불편함.



<img src="/images/야마하-YDP-165,-YDP-S55-디지털-피아노-7.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

스피커는 20W x 2, 12cm x 2 구성임. YDP-145와 S35의 8W x 2보다 확실히 여유가 있고, 실제로도 음량과 저역 밸런스가 한 단계 더 낫다는 평가가 많음. 방 안에서 연습하기엔 충분히 여유가 있고, 작은 공간에서는 외부 앰프 없이도 꽤 만족스럽게 들을 수 있는 수준임. 전작과 같은 스피커 구성이긴 하지만, VRM Lite가 추가된 음원 쪽 변화 덕분에 체감상은 조금 더 풍부하게 느껴짐.

Dual과 Duo는 지원하지만 Split은 없음. 그래서 왼손 베이스 연습 같은 분할 연습은 어렵고, 음색 수도 적어서 피아노 연주 중심의 구성임. 녹음은 1곡 2트랙 MIDI 녹음만 가능함. 대신 USB to Host를 통한 USB Audio 기능이 있어서, 컴퓨터에 연결하면 외장 오디오 인터페이스 없이도 피아노 소리를 바로 녹음할 수 있음. 이건 같은 가격대에서 꽤 괜찮은 장점임. Smart Pianist와 연결은 가능하지만 Bluetooth는 내장되지 않음. USB to Host로 연결해야 하고, 무선으로 쓰려면 별도 어댑터를 따로 써야 함.
