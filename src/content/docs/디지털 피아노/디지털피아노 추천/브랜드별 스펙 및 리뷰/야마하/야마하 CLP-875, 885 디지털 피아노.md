CLP-875, 885는 CLP-800 시리즈의 상위 라인임. 845까지 사용하던 GrandTouch-S 건반에서 875와 885는 GrandTouch 건반으로 올라가면서 건반 길이, 페달 반응, 스피커 구성, 캐비닛 체급까지 모두 상위 기종임. 



### 건반 / 사운드

두 모델 모두 GrandTouch 건반을 쓰고, 흰 건반에 목재를 넣고, synthetic ebony ivory keytops와 escapement를 적용한 구조임. 845의 목재 GrandTouch-S보다 상위 건반이고, 이름에서도 S가 빠진 GrandTouch 계열이라 같은 목재 건반으로 묶어 보기엔 체급 차이가 있음.



<img src="/images/야마하-CLP-875,-885-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

CLP-875와 CLP-885는 모두 88-key Linear Graded Hammers를 넣었고, GrandTouch Pedals에 GP Response Damper까지 들어감. 페달도 845 이하처럼 단순히 3페달과 하프페달만 구현 한 것이 아니라, 댐퍼 페달의 연속적인 깊이 변화를 더 세밀하게 받는 상급 구조임. 공식 사양상 



<img src="/images/야마하-CLP-875,-885-디지털-피아노-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<strong>CLP-885에는 카운터웨이트가 들어가지만, CLP-875에는 들어가지 않음.</strong> 그래서 두 모델은 같은 GrandTouch 건반처럼 보이지만, 반복 타건 반응, 건반 복귀감, 아주 약한 터치에서의 안정감은 885와 875의 차이가 있음.

885의 카운터웨이트가 건반 반응뿐 아니라, 875보다 885가 고음부 선명도와 다이내믹 레인지가 자연스럽고, 스피커로 들을 때 자연스러운 울림도 차이점임.



<img src="/images/야마하-CLP-875,-885-디지털-피아노-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

사운드는 두 모델 모두 Yamaha CFX와 Bösendorfer Imperial을 중심으로 함. 최대 동시발음은 256음, VRM과 Grand Expression Modeling도 공통으로 들어감. CFX Grand와 Bösendorfer에는 바이노럴 샘플링을 지원하고, 내장 스피커에서만 차이가 있음. 


### 디자인 / 조작부

CLP-875는 폭 1450mm, 깊이 465mm, 높이 967mm, 무게 71kg이고, 유광 모델은 74kg임. CLP-885는 폭 1461mm, 깊이 494mm, 높이 1027mm, 무게 87kg이고, 유광 모델은 90kg임. 875도 이미 중대형 가구형 피아노 체급인데, 885는 더 크고 무거운 상급 업라이트형 캐비닛수준임.



<img src="/images/야마하-CLP-875,-885-디지털-피아노-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

두 모델 모두 Full Dots LCD와 터치 센서 조작부를 넣음. 835와 845는 버튼식 조작부인데, 875와 885는 터치 센서 방식이라 전면이 더 깔끔함. 기능 접근성 자체는 두 모델 모두 좋고, Smart Pianist를 쓰지 않아도 본체에서 주요 설정을 다루기 편함. 건반 덮개는 875가 슬라이딩 방식이고, 885는 폴딩 방식임. 이 부분도 두 모델의 외관 차이를 나눔.



<img src="/images/야마하-CLP-875,-885-디지털-피아노-5.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/야마하-CLP-875,-885-디지털-피아노-6.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

CLP-875는 38음색 구성임. 835와 845의 38음색 구성과 같고, 리버브 7종, 코러스 3종, 브릴리언스 7단계 플러스 유저 설정, 인서션 이펙트 12종, Dual, Split, Duo, Piano Room, 리듬 20종, 250곡 16트랙 MIDI 녹음, USB 메모리 WAV 오디오 녹음까지 들어감.

CLP-885는 음색 수가 53개로 늘고, 여기에 480 XG 음색과 14 Drum/SFX 키트까지 들어감. 단순 피아노 연주만 보면 875도 충분하지만, 내장 음색과 MIDI/XG 활용까지 생각하면 885 쪽이 확실히 넓음. 내장곡도 875는 Voice Demo 23곡, 885는 Voice Demo 27곡으로 차이가 있음.

스피커 차이도 큼. CLP-875는 45W + 25W + 40W 조합을 좌우로 넣은 3웨이 구조이고, 16cm, 8cm 디퓨저, 2.5cm 돔 트위터와 양방향 혼을 조합함. 

CLP-885는 45W + 30W + 40W 조합으로 중역 쪽 출력이 더 크고, 16cm 디퓨저, 8cm 디퓨저, 2.5cm 돔 트위터와 양방향 혼에 Spruce Cone Speaker까지 들어감. 그래서 875와 885의 가격 차이는 건반의 카운터웨이트, 캐비닛 체급, 스피커 질감, 음색 확장성 차이라고 보면 됨.
