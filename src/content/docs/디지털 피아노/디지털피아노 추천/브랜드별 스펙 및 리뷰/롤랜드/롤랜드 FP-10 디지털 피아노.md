<img src="/images/롤랜드-FP-10-디지털-피아노-1.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### 피아노 터치와 사운드

FP-10은  Roland의 PHA-4 건반이 적용됨. 저음은 무겁고 고음으로 갈수록 가벼워지는 그레이드 해머 액션 구조임. 건반 표면은 인조 상아 질감이고, 터치 감도는 5단계로 조절 가능함.

상위 기종인 FP-30x, FP-60x와 같은 건반을 공유하고, FP-30x과는 기본적인 프레임까지 공유함.

PHA-4 건반은 타브랜드의 동급 기종중 타건감이 가장 무겁지만 복원력이 안정적이고, 이스케이프먼트 구조를 구현해 무거운 타건감을 원하는 사람들에게 인기가 좋음.

사운드는 Roland의 SuperNATURAL 음원 엔진을 사용함. 샘플링과 물리 모델링을 결합한 방식임. 


### 디자인 및 조작부

FP-10은 스피커가 본체 하단에 배치되어 소리가 아래쪽으로 출력되며, 이로 인해 본체 높이가 다르누기종에 비교해 약간 높은 편임.

전면 패널 구성은 FP-30x에 비교해 많은 부분을 축소해 단순함. 볼륨 상태를 표시하는 LED와 왼쪽에 네 개의 전원, 볼륨 조절, 기능 버튼으로 구성되며, 기능 버튼은 건반과 함께 눌러 각종 설정을 변경하는 데 사용함.



<img src="/images/롤랜드-FP-10-디지털-피아노-2.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

보면대는 기본 포함이고, 전용 스탠드를 별도로 구매할 수 있음.

FP-10 대부분의 기능 조작은 기능버튼+건반을 눌러 설정할 수 있으나, 본체에 표기되어 있지 않아 설명서를 참고해야 함.

후면에는 전원 어댑터 단자와 서스테인 페달 단자가 있음. 기본 구성은 DP-2 페달 스위치이며, 하프 페달을 지원하는 DP-10도 사용 가능함. 

오디오 출력은 3.5mm 스테레오 헤드폰 단자 1개임. 외부 오디오 시스템에 연결하려면 변환 케이블이 필요함.

헤드폰 단자 사용 시 내장 스피커 차단 여부를 설정할 수 있음. 외부 앰프를 사용하면서 본체 스피커를 동시에 활용하는 것도 가능함. 



<img src="/images/롤랜드-FP-10-디지털-피아노-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

후면에는 컴퓨터 연결용 USB Type B 단자와 업데이트용 USB Type A 단자가 있음. USB A는 FP-30과 동일한 플랫폼을 공유해 물리적 단자만 유지된 것으로 단자를 통해 데이터 저장은 지원하지 않음.

후면 단자 표시를 검정색 바탕에 검정색 글자로 인쇄해 식별이 불편함. 


### <strong>FP-10 및 FP-30x비교</strong>

두 모델 모두 건반은 동일하지만, FP-10은 옵션으로Roland DP-10을 연결해도 연속 값이 아닌 3단계(on/half/off)로 처리됨.

Roland FP-10은 Bluetooth MIDI를 지원함(오디오 스트리밍은 아님, MIDI 데이터만) — 즉 스마트폰/태블릿과 무선으로 MIDI 연결이 가능함.

### Roland Piano App과 FP-10

Roland Piano App은 Bluetooth 또는 USB MIDI로 FP-10과 연결해 다음과 같은 기능을 제공함.

- 기능 제어

- 트랜스포즈(조옮김) 설정

- 스플릿/레이어 설정

- 건반 터치 감도(키 터치) 설정

- Ambience/Brilliance 이펙트 조절

- 사운드 선택 및 탐색

- 본체에서 바로 선택하기 어려운 음색을 앱으로 빠르게 탐색 가능

- 피아노 외 비피아노 음색까지 접근 가능

- 시각 메트로놈

- 시각화된 메트로놈으로 리듬 연습 지원

- 녹음/재생 기능

- MIDI 방식으로 연주 녹음 및 재생

- 자신 연주를 검토‧연습에 활용 가능

- 내장곡 악보 표시

- FP-10 내장곡을 악보로 화면에 표시하고 재생

- 한 손만 듣거나, 반주와 함께 연습 기능 제공

- 청음/학습 기능

- Flash card 같은 청음 연습, 음악 읽기 보조 기능

- One Week Master 기능으로 7일 학습 코스 제공

- 반주 기능

- 자동 반주(자동 맥락 생성) 등을 Left-hand fingering으로 사용 가능

- 확장 악보 연결

- Roland Cloud 유료 구독을 통해 수백 곡 수준 악보 잠금 해제 가능

<strong>주의점</strong>

- FP-10은 Bluetooth MIDI 연결만 지원하므로, 앱과 연결할 때 장치 Bluetooth 설정 화면이 아니라 앱 내부에서 페어링해야 함.

- 일부 Android 기기에서는 Bluetooth 연결이 원활하지 않을 수 있음(기기/OS 버전에 따라 다름).
