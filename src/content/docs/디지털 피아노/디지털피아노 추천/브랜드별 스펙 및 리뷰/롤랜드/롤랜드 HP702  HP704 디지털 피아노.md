HP702와 HP704는 2018년 Roland 출시한 모델로 같은 음원과 기능 구성을 공유하지만 건반 액션, 스피커 시스템, 캐비닛 구성에서 차이가 남. 구조적으로는 세대가 오래된 편이지만, 2026년 기준으로 계속 생산중인 모델임.


### 건반 / 사운드

HP702는 PHA-4 Standard 액션을 사용하고, HP704는 상위 모델인 PHA-50 하이브리드 액션을 사용함. HP704의 PHA-50은 흰 건반 측면에 목재를 쓰고 내부는 플라스틱 구조로 설계되어, 키 밸런스와 움직임이 더 안정적이고 조용한 편임. 반면 HP702의 PHA-4 Standard는 전반적인 반응은 좋지만 타건감이 다소 단단하고, 세게 칠 때 바닥 타건 소음이 조금 더 느껴질 수 있음. 두 모델 모두 Escapement를 지원하며, 아주 천천히 누를 때 미세한 노치감을 구현함.



<img src="/images/롤랜드-HP702-HP704-디지털-피아노-1.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

사운드는 두 모델 모두 Roland의 Physical Modeling 기반 피아노 음원을 사용함. 

음색은 두 모델 모두 동일함. 콘서트 피아노, 발라드, 멜로우, 브라이트 계열의 4가지 주요 피아노 톤을 제공하며, Piano Designer 기능을 통해 Brilliance, Touch Sensitivity, Lid, Key Off Noise, Soundboard Type, Cabinet Resonance 등 세부 항목을 조절할 수 있음. 

다만 피아노 음색을 연주 도중 다른 피아노 톤이나 악기 음색으로 바꿀 때 기존 음이 바로 끊기는 all notes off 현상이 있어, 라이브 연주 중 자연스러운 음색 전환은 안됨. 피아노만 모델링이고 나머지는 샘플링이라 자연스러운 전환은 불가능함.

폴리포니는 피아노 음색에서 무제한이고, 다른 샘플링 음색에서는 모노 기준으로 최대 384음임. 실제 연주에서는 폴리포니 부족을 느낄 가능성은 거의 없고, 페달 반응도 전반적으로 좋고 하프 페달도 지원함.


### 디자인 및 조작부



<img src="/images/롤랜드-HP702-HP704-디지털-피아노-2.jpg" alt="HP 704" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/롤랜드-HP702-HP704-디지털-피아노-3.jpg" alt="HP 702" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

HP704는 상위모델의 캐비닛 구조와 출력이 큰 내장 스피커를 사용하고, 유광 블랙 마감 옵션도 제공함. <strong>구 LX 모델</strong>과 <strong>디자인컨셉과 구조를 공유함.</strong>

HP702는 하위 모델의 캐비닛이며 유광 마감 옵션은 없음. <strong>RP701의 하부 스탠드/플랫폼을 약간 개선하여 사용함.</strong>

두 모델 모두 슬라이딩 키 커버와 보면대, 중앙 집중형 조작 패널 구성이고, 조작 패널은 정면 중앙에OLED 디스플레이와 함께 배치됨.

버튼과 노브는 크고 평평하게 매립된 형태이며, 조명도 들어와 현재 상태를 확인할 수 있음. 악기 음색 선택도 4개의 톤 그룹 버튼을 중심으로 직관적으로 이루어져, 대형 라이브러리를 가진 것치고는 선택하기 쉬운편임. 다만 기능이 많다 보니 설명서나 영상 없이 버튼만 보고 모든 기능을 바로 익히는 건 불가능함.

HP702는 2스피커, 총 28W 구성이라 볼륨은 충분하지만 저역과 울림이 부족하고, HP704는 총 60W, 4앰프, 6스피커 구조를 사용해 훨씬 더 풍성하고 자연스러운 울림을 제공함.



<img src="/images/롤랜드-HP702-HP704-디지털-피아노-4.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

헤드폰만 쓰면 두 모델의 차이점이 없지만, 내장 스피커로 들으면 큰 차이가 남. 결국 HP704의 추가 비용은 건반 액션과 내장 스피커 / 캐비닛이라고 보면 됨.

두 모델 모두 Bluetooth Audio와 MIDI를 지원하고, USB 메모리, USB MIDI, 스테레오 오디오 입력, 오디오 출력, 2개의 헤드폰 단자, 외부 기기 음악을 피아노 스피커로 무선 재생할 수 있고, 태블릿 앱과 연동해 악보 보기, 페이지 넘기기 듵 거의 모든 디지털 기능이 같음. 특히 이 가격대에서 앱을 통한 자동 반주 스타일 지원은 타 브랜드 대비 장점임.


### 타 브랜드 비교 모델

HP702 기준으로는 Yamaha YDP184, Yamaha CLP-825, Kawai CN201이 언급되며, 

HP704 기준으로는 Kawai CA401, Yamaha CLP-835, Korg G1 Air, Casio AP-550이 경쟁 모델임.
