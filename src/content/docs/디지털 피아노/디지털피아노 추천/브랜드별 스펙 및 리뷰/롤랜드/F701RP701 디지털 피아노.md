F701과 RP701은 각각 F-140R과 RP501R의 후속 모델이며, 2021년 2월 출시됨. 


### 건반 / 사운드

F701과 RP701은 모두 FP-30X와 FP-60X에 들어간 PHA-4 Standard 액션을 사용함. PHA-4 건반은 타 브랜드 동급 기종 중 비교적 무거운 타건감을 가지며, 복원력이 안정적이고 이스케이프먼트 구조를 구현해 실제 피아노에 가까운 반응을 제공함. 건반을 끝까지 눌렀을 때 그랜드 피아노에서 느껴지는 미세한 노치감도 느껴짐.

플라스틱 구조이지만, 표면은 인조 상아 질감을 적용해 촉감이 좋고 손에 땀이 있는 경우에도 미끄러짐이 적음. 기계식 웨이트와 트리플 센서를 사용하며, 저음은 더 무겁고 고음으로 갈수록 가벼워지는 그레이드 구조도 적용됨.

벨로시티 반응은 100단계 조절이 가능하고 Off도 지원함. 기본 반응도 자연스러운 편이라 별도 적응 없이 다양한 다이내믹을 다루기 쉬움. 다만 세게 칠 때 PHA-4 Standard 특유의 바닥 타건 소음이 느껴질 수 있음. 그래도 같은 급 액션 중에서는 비교적 조용한 편임.

사운드는 Roland의 SuperNATURAL Piano 음원 엔진을 사용함. 샘플링과 물리 모델링을 결합한 방식으로, 이번 세대에서는 Behavior Modeling Core 칩이 적용되어 전작보다 음색 밀도와 밸런스가 개선됨. 이전 세대의 SuperNATURAL 피아노가 밝고 얇다는 평가를 받았다면, F701과 RP701은 전반적으로 더 꽉 차고 풍성하게 바뀜.

총 324음색을 제공하지만, 메인 음색은 피아노 26종, 스트링 16종, 오르간 12종, Voice 1종, Do Re Mi 4종, 드럼 9종으로 구성됨. 나머지 255개 음색은 퀄리티 좋은 메인 음색이 아니라 GM2라는 표준 음색 세트로 음질이나 디테일이 떨어지고 개별 설정도 안 됨.

Roland 특유의 뭉툭한 음색은 Brilliance로 어느 정도 고역을 조절할 수는 있지만, 완전히 다른 성향의 톤으로 바꾸는 데에는 한계가 있음.


### 디자인 및 조작부



<img src="/images/F701RP701-디지털-피아노-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/F701RP701-디지털-피아노-2.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

전작은 2010년대 초반 디지털 피아노 특유의 버튼 많은 스타일이었다면, 새 디자인은 더 현대적이고 조금 더 고급스럽게 보임. 실제로 F701은 조작계와 외형 디자인으로 Red Dot Award를 받았고, RP701은 같은 설계를 캐비닛 형태만 다른 모델임.



<img src="/images/F701RP701-디지털-피아노-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

F701은 더 슬림하고 낮아 공간을 덜 차지하는 구조고, 이전 시리즈처럼 건반 덮개가 그대로 보면대 역할도 겸함. 

RP701은 좀 더 전통적인 콘솔형 디지털 피아노임. 업라이트 피아노에서 상부를 잘라낸 듯한 형태에 가깝고, 건반 뒤쪽이 더 길게 빠져 있으며 전용 보면대도 따로 있음. 덮개를 전면 조작부 상단만 가리는 위치에서 멈춰 둘 수 있어 시야를 정리하기 좋음. 

두 모델 모두 본체는 목재 중심 구조에 반광 마감을 사용함. 크기는 다르지만, <strong>내부 설계와 조작부와 기능은 거의 같음.</strong> RP701이 더 큰 캐비닛을 사용하지만, 스피커 출력은 F701과 동일함.



<img src="/images/F701RP701-디지털-피아노-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

조작부의 버튼은 FP-90X 계열과 같은 구조를 사용하며, 탄성 코일 스프링으로 촉감이 또렷하고, 버튼 주변에는 백라이트가 있어 어떤 기능이 켜져 있는지 바로 확인 가능함. OLED 화면은 128 x 64 해상도로, 이전 세대의 LED 표시보다 정보량도 많고 가독성도 좋음.

음색 수가 많지만, 엔코더 노브로 쉽게 찾을 수 있고, 단계마다 클릭감이 있고 눌러서 Enter처럼도 사용할 수 있음. 전체 조작 구조는 미니멀하고 정리된 편이지만, 설정을 자주 만지려면 여러 페이지를 넘겨야 해서 세부 조정을 자주 하는 사람에게는 약간 번거로울 수 있음.

가장 아쉬운 부분은 볼륨 조절 방식임. 엔코더 노브로는 조절되지 않고, 볼륨 업 버튼과 다운 버튼, 5개의 표시등으로 100단계 볼륨을 표현해야 해서 현재 값이 정확히 어디인지 알기 어렵고, 이건 FP-30X와 같은 구조라 불편함.

스피커는 두 모델 모두 동일함. 12W 스피커 두 개를 사용하며, 12cm 유닛 구성임. 하방 배치 방식이고, 음량은 꽤 큰 편이라 작은 방 정도는 외부 앰프 없이도 어느 정도 채울 수 있음. 다만 6.3mm 스테레오 라인아웃이 없어서, 외부 앰프를 쓰려면 변환이 필요함.

헤드폰 단자는 3.5mm와 6.3mm 두 개를 제공하며, 동시에 사용 가능함. Audio In으로 3.5mm stereo mini 입력도 지원하고, USB Type A는 저장용, USB Type B는 MIDI와 Audio 모두 지원해 DAW에 직접 녹음도 가능함. Bluetooth Audio와 MIDI도 지원함.

두 모델 모두 스탠드와 3페달 구성이 포함된 일체형 구조임.
