## 3. 각 부 명칭


### 전면 패널



<img src="/images/E2x2-OTG-한글-매뉴얼-2.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

1. IN1&IN2
  <p style="margin: 0 0 1em 0;">마이크, 악기, 라인 레벨 기기를 연결하는 입력 단자임. XLR과 6.35mm 밸런스/언밸런스 콤보 플러그를 모두 지원함. 마이크는 XLR 플러그, 악기는 6.35mm TS 플러그, 라인 레벨 기기는 6.35mm TRS 플러그로 연결.</p>

1. INST
  <p style="margin: 0 0 1em 0;">입력 게인과 입력 임피던스를 바꾸는 버튼. 라인 입력과 악기 입력을 여기서 전환함. 표시등이 꺼져 있으면 라인 입력 상태임. 표시등이 켜져 있으면 악기 입력 상태임.</p>

1. 48V
  <p style="margin: 0 0 1em 0;">마이크 입력용 팬텀 전원 스위치임. 표시등이 켜지면 해당 XLR 입력에 48V 팬텀 전원이 공급됨.</p>

<p style="margin: 0 0 1em 0;"><strong>! 주의</strong></p>

<p style="margin: 0 0 1em 0;">팬텀 전원은 콘덴서 마이크에만 필요함. 필요하지 않은 장비에 공급하면 손상될 수 있으므로 필요할 때만 켜야 함. 팬텀 전원을 켜거나 끄기 전에는 볼륨을 최소로 낮춰야 함.</p>

1. 입력 게인 노브
  <p style="margin: 0 0 1em 0;">입력 라인의 게인을 조절함. 반시계 방향으로 돌리면 게인이 줄고, 시계 방향으로 돌리면 게인이 커짐. 조정할 때는 입력 미터를 확인해야 함. CLIP 표시가 켜지면 게인을 낮춰야 함.</p>

1. MON
  <p style="margin: 0 0 1em 0;">직접 모니터링을 켜는 버튼. 모노 입력 신호를 헤드폰 출력의 좌우 채널로 직접 보내므로 듀얼 모노 모니터링이 가능함.</p>

1. LED 미터
  <p style="margin: 0 0 1em 0;">IN1&2는 입력 미터. OUT1&2는 출력 미터임. 신호가 빨간색 CLIP 지점에 도달하면 클리핑 상태이므로 신호 레벨을 낮춰야 함.</p>

1. 전원 표시등
  <p style="margin: 0 0 1em 0;">계속 켜져 있으면 동작 상태, 꺼져 있으면 전원이 꺼진 상태임. 천천히 깜박이면 대기 상태임.</p>

<p style="margin: 0 0 1em 0;">자동 대기 기능이 켜져 있을 때 USB 신호 없이 전원만 감지되면, IN1과 IN2에 신호가 들어와도 1분 뒤 대기 상태로 들어감. USB 신호가 감지되면 자동으로 다시 동작 상태로 돌아감. 자동 대기 기능은 TOPPING Professional Control Center에서 톱니바퀴 모양의 설정에서 제어.</p>

1. OTG 표시등
  <p style="margin: 0 0 1em 0;">계속 켜져 있으면 OTG 포트에 기기가 연결된 상태, 꺼져 있으면 OTG 포트에 연결된 기기가 없는 상태임.</p>

1. 모니터 믹스
  <p style="margin: 0 0 1em 0;">MON 버튼을 누른 뒤 이 노브로 실시간 입력 신호와 컴퓨터 재생 신호 사이의 비율을 조절함. 반시계 방향으로 돌리면 입력 신호 비율이 커짐. 시계 방향으로 돌리면 재생 신호 비율이 커짐. 이 노브는 입력 신호의 녹음 레벨에는 영향을 주지 않음.</p>

1. 헤드폰 볼륨 조절
  <p style="margin: 0 0 1em 0;">헤드폰 출력 볼륨을 조절함. 반시계 방향으로 돌리면 줄고, 시계 방향으로 돌리면 커짐.</p>

1. 6.35mm 헤드폰 출력 잭
  <p style="margin: 0 0 1em 0;">6.35mm 헤드폰을 연결하는 단자.</p>

1. GAIN
  <p style="margin: 0 0 1em 0;">헤드폰 게인 설정 버튼임. 표시등이 꺼져 있으면 Low GAIN. 표시등이 켜져 있으면 High GAIN.</p>

1. MONITOR
  <p style="margin: 0 0 1em 0;">모니터 출력의 LINE OUT 레벨과 AUX OUT 레벨을 조절함. 반시계 방향으로 돌리면 줄고, 시계 방향으로 돌리면 커짐.</p>


### 후면 패널



<img src="/images/E2x2-OTG-한글-매뉴얼-3.jpg" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

1. LINE OUT 
  <p style="margin: 0 0 1em 0;">6.35mm TRS 밸런스 출력 잭임. 액티브 스피커나 앰프를 연결함.</p>

1. AUX OUT
  <p style="margin: 0 0 1em 0;">3.5mm 스테레오 출력 단자.</p>

1. OPTICAL OUT
  <p style="margin: 0 0 1em 0;">광 S/PDIF 출력 단자.</p>

1. OTG
  <p style="margin: 0 0 1em 0;">모바일 기기를 연결하는 단자임. 모바일 기기의 오디오를 컴퓨터로 녹음할 때 쓰며, 반대로 오디오를 모바일 기기로 보낼 때도 사용함.</p>

1. USB-C
  <p style="margin: 0 0 1em 0;">동봉된 USB 케이블로 컴퓨터와 연결하는 단자. USB 포트의 전원 공급 상태에 따라 E2x2 OTG에 전원도 함께 공급됨.</p>

1. POWER
  <p style="margin: 0 0 1em 0;">특히 모바일 기기나 태블릿을 연결했을 때 USB 포트 전원이 부족하면 DC 5V 전원을 연결함. 이 경우 옆의 USB-C 포트는 전원 입력에 사용하지 않고 데이터 전송에만 사용함.</p>

1. 전원 스위치
  <p style="margin: 0 0 1em 0;">기기의 전원을 켜고 끄는 스위치임.</p>

### USB 모드 설정

<p style="margin: 0 0 1em 0;">USB-C 포트에는 PC 또는 모바일 기기를 직접 연결할 수 있음. 연결 대상에 맞게 PC 모드 또는 모바일 모드로 설정해야 함.</p>

<p style="margin: 0 0 1em 0;">기기 전원을 켜는 동안 전면의 IN1 48V 버튼을 길게 누르면 설정 모드로 들어감.</p>

<p style="margin: 0 0 1em 0;">48V 버튼을 눌러 모드를 전환함.</p>

- PC 모드에서는 표시등 8개가 켜짐.
- 모바일 모드에서는 표시등 4개가 켜짐.
<p style="margin: 0 0 1em 0;">설정을 변경한 뒤 전원을 끄고 다시 켜야 적용됨.</p>

## 6. DAW에서의 오디오 설정

<p style="margin: 0 0 1em 0;">E2x2 OTG는 Mac에서는 Core Audio, Windows에서는 ASIO를 지원하는 모든 DAW와 호환됨.</p>

<p style="margin: 0 0 1em 0;">E2x2 OTG는 Mac의 Core Audio와 Windows의 ASIO를 지원하는 모든 DAW에서 사용할 수 있음.</p>

<p style="margin: 0 0 1em 0;">DAW의 오디오 설정에서 입력 장치와 출력 장치로 E2x2 OTG를 선택함.</p>

<p style="margin: 0 0 1em 0;">Windows에서는 ASIO 드라이버를, Mac에서는 Core Audio 드라이버를 선택함.</p>

<p style="margin: 0 0 1em 0;">구체적인 설정 방법은 사용하는 각 DAW의 사용 설명서를 참고.</p>

## 7. Troubleshooting


### 전원이 켜지지 않음

<p style="margin: 0 0 1em 0;">후면 전원 스위치가 켜져 있는지 확인함.</p>

<p style="margin: 0 0 1em 0;">전원 공급이 부족할 수 있으므로 POWER 단자에 DC 5V 전원을 연결해 봄.</p>

<p style="margin: 0 0 1em 0;">USB 케이블이 손상되었을 수 있으므로 새 케이블로 바꿔 보고, USB 케이블 길이는 2m 이하를 권장함.</p>

<p style="margin: 0 0 1em 0;">컴퓨터의 다른 USB 포트에 연결해 보거나, 다른 컴퓨터에서도 시험해 봄.</p>


### 재생 소리가 들리지 않음

<p style="margin: 0 0 1em 0;">스피커 연결 상태와 스피커 볼륨을 확인함.</p>

<p style="margin: 0 0 1em 0;">E2x2 OTG의 라인 출력 및 헤드폰 출력 볼륨을 조절함.</p>

<p style="margin: 0 0 1em 0;">ToppingPro 설정을 확인함.</p>

<p style="margin: 0 0 1em 0;">USB 허브를 거치지 말고 컴퓨터에 직접 연결해 봄.</p>

<p style="margin: 0 0 1em 0;">사용하지 않는 다른 USB 장치를 제거한 뒤 다시 확인함.</p>

<p style="margin: 0 0 1em 0;">실행 중인 응용 프로그램을 모두 종료한 뒤 다시 확인함.</p>


### 녹음 소리가 너무 크거나 너무 작거나 아예 없음

<p style="margin: 0 0 1em 0;">E2x2 OTG의 입력 게인 레벨을 조절함.</p>

<p style="margin: 0 0 1em 0;">콘덴서 마이크에 48V 팬텀 전원이 필요하면 팬텀 전원을 켬.</p>

<p style="margin: 0 0 1em 0;">ToppingPro 설정을 확인함.</p>


### 소리가 끊김

<p style="margin: 0 0 1em 0;">오디오 응용 프로그램 또는 ToppingPro에서 버퍼 크기와 레이턴시를 더 크게 설정함. Windows 전용 항목임.</p>

<p style="margin: 0 0 1em 0;">다른 컴퓨터에서도 test 해 봄.</p>


### 입력 단자에 연결된 기기의 소리가 왜곡됨

<p style="margin: 0 0 1em 0;">입력 미터를 확인함. 클리핑 표시가 켜지면 입력 게인을 낮춤.</p>


### 재생 또는 녹음이 되지 않음

<p style="margin: 0 0 1em 0;">사용 중인 소프트웨어에서 E2x2 OTG가 입력 및 출력 장치로 설정되어 있는지 확인함.</p>

<p style="margin: 0 0 1em 0;">E2x2 OTG가 컴퓨터에 제대로 연결되어 있는지 확인함.</p>

<p style="margin: 0 0 1em 0;">E2x2 OTG를 사용하는 소프트웨어를 모두 종료한 뒤 USB 케이블을 뽑았다가 다시 연결함.</p>


### 모바일에서 이 기기를 인식하지 못함

<p style="margin: 0 0 1em 0;">일부 스마트 폰은 먼저 OTG 기능을 활성화해야 함.</p>

<p style="margin: 0 0 1em 0;">사용 중인 USB 케이블이 OTG 기능을 지원해야 함.</p>

<p style="margin: 0 0 1em 0;">전원 공급이 부족할 수 있으므로 POWER 단자에 DC 5V 전원을 연결해 봄.</p>

<p style="margin: 0 0 1em 0;">스마트 폰을 USB-C 포트에 연결했다면 기기를 모바일 모드로 설정해야 함.</p>

## 8. 주의사항

<p style="margin: 0 0 1em 0;">고온다습한 환경과 기기에 강한 충격을 피할 것.
제품을 임의로 분해하면 보증이 무효가 됨.
실내용으로만 사용해야 함.
E2x2 OTG의 고장으로 직간접적으로 발생한 손실이나 손해에 대해 Topping은 책임을 지지 않음.
제품 개선을 위해 사양은 예고 없이 바뀔 수 있음.</p>

## 9. 주요 사양 (@24bit/96kHz)


### 마이크 입력

<p style="margin: 0 0 1em 0;">입력 환산 잡음 @A-wt, 150 Ohm: -130.5dBu</p>

<p style="margin: 0 0 1em 0;">THD+N @A-wt: -110dB (0.0003%)</p>

<p style="margin: 0 0 1em 0;">다이내믹 레인지 @A-wt: 115dB</p>

<p style="margin: 0 0 1em 0;">SNR @A-wt: 115dB</p>

<p style="margin: 0 0 1em 0;">크로스토크 @1kHz: -140dB</p>

<p style="margin: 0 0 1em 0;">주파수 응답: 20Hz~40kHz (±0.2dB)</p>
