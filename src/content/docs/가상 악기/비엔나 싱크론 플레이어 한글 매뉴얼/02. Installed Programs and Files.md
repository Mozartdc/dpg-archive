<p style="margin: 0 0 1em 0;">설치를 진행하면 Synchron Player 플러그인이 플러그인 폴더에 설치됨.</p>

<p style="margin: 0 0 1em 0;">Synchron Library Installer는 Applications(또는 Programs) 폴더에 설치되며, 라이브러리를 수동으로 설치할 때 사용함.</p>


### Apple Silicon Mac 사용 시

<p style="margin: 0 0 1em 0;">Apple Silicon이 탑재된 Mac에서는 DAW를 Native 모드로 실행해야 함.</p>

<p style="margin: 0 0 1em 0;">Finder에서 DAW 앱을 선택한 뒤 Command+I로 정보 창을 열고, “Open in Rosetta Mode”가 체크되어 있지 않은지 확인해야 함. 해당 항목이 해제되어 있어야 Synchron Player가 최적의 성능으로 작동함.</p>

<p style="margin: 0 0 1em 0;"><strong>macOS</strong></p>

<img src="/images/02.-Installed-Programs-and-Files-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Vienna Synchron Player는 단독 실행이 가능하고, 시퀀서 없이도 바로 사용할 수 있는 프로그램임.</p>

<p style="margin: 0 0 1em 0;">VST, AU, AAX 플러그인은 macOS의 플러그인 폴더(/Library/Audio/Plug-ins)와 Pro Tools 폴더에 설치됨.</p>

<p style="margin: 0 0 1em 0;">또한 MIDI Dump 기능을 위한 가상 MIDI 드라이버도 함께 설치됨.</p>

<p style="margin: 0 0 1em 0;"><strong>Windows</strong></p>

<img src="/images/02.-Installed-Programs-and-Files-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Vienna Synchron Player는 단독 실행 프로그램으로 설치됨. VST, VST3, AAX 플러그인은 사용자가 지정한 위치에 설치됨.</p>

<p style="margin: 0 0 1em 0;">MIDI Dump 기능을 위한 가상 MIDI 드라이버도 함께 설치됨.</p>


### <strong>Content Installation</strong>

<p style="margin: 0 0 1em 0;">샘플 콘텐츠는 Vienna Assistant로 다운로드하고 설치함.</p>

<p style="margin: 0 0 1em 0;">설치 과정에서 파일을 저장할 위치를 선택해야 하며, 충분한 여유 공간이 있는 드라이브를 지정해야 함.</p>

<p style="margin: 0 0 1em 0;">다운로드할 마이크 포지션도 선택할 수 있음. 더 많이 설치할수록 믹싱 시 선택할 수 있는 옵션이 늘어남.</p>

<p style="margin: 0 0 1em 0;">택한 드라이브의 공간이 부족하면 경고 메시지가 표시됨. 이 경우 설치를 취소한 뒤 다른 드라이브를 선택해 다시 진행해야 함.</p>


### <strong>Vienna Hard Drive Installation</strong>

<p style="margin: 0 0 1em 0;">Vienna 전용 하드 드라이브를 사용하는 경우 Vienna Assistant가 이를 자동으로 인식함. 이때는 다운로드 없이 바로 설치가 진행됨.</p>
