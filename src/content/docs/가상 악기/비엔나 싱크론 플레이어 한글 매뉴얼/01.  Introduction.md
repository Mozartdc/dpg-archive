### <strong>About Vienna Synchron Player</strong>

<p style="margin: 0 0 1em 0;">Synchron Player는 멀티 마이크로 녹음된 사운드를 효율적이고 직관적으로 다룰 수 있도록 설계된 엔진임.</p>

<p style="margin: 0 0 1em 0;">사용 가능한 모든 어티큘레이션은 트리 뷰에서 한눈에 확인할 수 있고, 탭을 통해 바로 편집할 수 있음. 또한 내장 믹서를 이용해 각 마이크 신호를 조합함으로써, 스테레오부터 서라운드까지 다양한 믹싱 환경에 맞게 설정할 수 있음.</p>

<img src="/images/01.-Introduction-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">인터페이스는 자유롭게 크기를 조절할 수 있으며, 레티나 디스플레이에 최적화되어 어떤 해상도에서도 선명하게 표시됨.
</p>
