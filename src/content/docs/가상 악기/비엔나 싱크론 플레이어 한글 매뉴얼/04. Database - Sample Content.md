<p style="margin: 0 0 1em 0;">샘플 콘텐츠는 Vienna Assistant로 Synchron 시리즈 제품을 설치할 때 함께 설치됨. Vienna Assistant는 콘텐츠의 이동·복사·삭제를 관리하는 도구임.
사용자가 만든 프리셋은 User Presets 폴더에 저장되며, 이 폴더 위치는 원하는 곳으로 지정할 수 있음.</p>

<p style="margin: 0 0 1em 0;">macOS 기본 경로:
/Users/사용자이름/Documents/VSL/Vienna Synchron Player</p>

<p style="margin: 0 0 1em 0;">Windows 기본 경로:
C:\Documents and Settings\All Users\Documents\VSL\Vienna Synchron Player</p>

<img src="/images/04.-Database---Sample-Content-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">샘플 콘텐츠를 다른 위치로 옮긴 경우에는 Vienna Synchron Player의 Database 설정에서 샘플 경로를 다시 지정하면 됨. 왼쪽 하단의 +/- 버튼으로 볼륨 스캔 경로를 추가하거나 제거할 수 있음. 경로를 변경한 뒤에는 호스트(DAW)를 재시작해야 함.</p>

<p style="margin: 0 0 1em 0;">연결이 끊긴 드라이브나 이름·경로가 바뀐 폴더는 빨간색으로 표시됨.
드라이브의 읽기 속도는 처음 실행 시와 경로 변경 시 자동으로 측정됨. 이 값을 기준으로 프리로드 크기가 자동 설정됨. 필요하면 사용자가 직접 조정할 수 있음.
선택한 경로의 드라이브 정보는 오른쪽 하단에 표시됨.</p>
