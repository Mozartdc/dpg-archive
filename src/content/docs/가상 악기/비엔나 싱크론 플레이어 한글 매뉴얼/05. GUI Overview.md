<p style="margin: 0 0 1em 0;">Velocity Histogram
키보드에서 입력되는 벨로시티 값을 그래프로 표시함. 현재 표시 상태를 고정하거나 해제하려면 더블클릭.</p>

<p style="margin: 0 0 1em 0;">Volume Knob
Master Volume 오디오 파라미터(MIDI CC7)에 연결되어 전체 볼륨을 조절함. 클릭 후 드래그하거나 더블클릭해 값을 입력할 수 있음. Command/Ctrl+클릭하면 기본값으로 돌아감.</p>

<p style="margin: 0 0 1em 0;">Volume Meter
스테레오 출력 레벨 표시. 클리핑이 발생하면 빨간색으로 표시됨.</p>

<p style="margin: 0 0 1em 0;">Flow View
Flow를 지원하는 컬렉션에서 사용할 수 있음. Flow 아이콘을 클릭하면 해당 뷰로 전환되며, 이 상태에서는 아이콘이 Precision Presets 아이콘으로 바뀜.</p>

<p style="margin: 0 0 1em 0;">Tuning
12음 평균율 외에도 다양한 튜닝 방식을 선택할 수 있음.</p>

<p style="margin: 0 0 1em 0;">GPU Power
GPU 컨볼루션 처리 활성화.</p>

<p style="margin: 0 0 1em 0;">Help
시스템 기본 브라우저에서 온라인 매뉴얼 보기.</p>

<p style="margin: 0 0 1em 0;">MIDI Dump
현재 선택된 슬롯 정보를 MIDI로 전송함. MIDI CC 전송 여부는 Preferences에서 설정함.</p>

<p style="margin: 0 0 1em 0;">MIDI Player
Synchron Power Drums 및 Synchron World Percussion 프리셋에서 사용할 수 있음. 클릭하면 별도의 MIDI Player 창이 열림.</p>

<p style="margin: 0 0 1em 0;">Scale Button
인터페이스 크기를 75%에서 200%까지 단계적으로 조절함.</p>

<p style="margin: 0 0 1em 0;">Key Info
키보드에 할당된 건반 정보를 보여주는 추가 창 열기.</p>

<p style="margin: 0 0 1em 0;">CPU Real-Time Display
현재 CPU 부하를 표시함. 피크 값은 빨간색, 평균 값은 파란색으로 표시됨. 레이턴시가 낮거나 보이스 수가 많을수록 CPU 부하가 증가함.</p>

<p style="margin: 0 0 1em 0;">Settings
인터페이스 및 엔진 설정 조정. Database 탭에서 샘플 콘텐츠 경로 지정 가능함.</p>

<p style="margin: 0 0 1em 0;">Dimension Tree View
현재 프리셋에서 사용 가능한 모든 아티큘레이션 개요 표시.</p>

<p style="margin: 0 0 1em 0;">Tab Selection
커스터마이징 옵션 탭 선택.
∙ PERFORM
연주 관련 주요 파라미터와 컨트롤러 할당 목록 표시. Control 탭의 파라미터를 추가 가능함.
∙ CONTROL
모든 파라미터와 컨트롤러 매핑 목록 표시. 엔진 파라미터 추가 및 매핑 가능함.
∙ EDIT
트리 구조에서 선택한 슬롯의 커스터마이징 옵션 표시.
∙ MIX
사용 가능한 마이크를 위한 통합 믹서.
Tab Display
선택한 탭의 컨트롤 표시.</p>

<p style="margin: 0 0 1em 0;">Browser
사용 가능한 제품의 프리셋과 패치 선택.</p>

<p style="margin: 0 0 1em 0;">Information Window
추가 정보 표시. 마우스를 올리면 관련 정보 확인 가능함.</p>

<p style="margin: 0 0 1em 0;">Virtual Keyboard
선택한 슬롯의 연주 범위 표시. 키스위치는 트리 뷰의 해당 디멘션 색상에 맞춰 구분됨. 건반 상단 라인에서 추가 정보 확인 가능함.</p>

<p style="margin: 0 0 1em 0;">Activate Range Display
선택한 슬롯의 연주 범위를 핸들과 함께 표시. 최대 1옥타브까지 확장 가능함.</p>

<p style="margin: 0 0 1em 0;">Short Mouse-over Information
Information Window 설명의 요약 정보 표시.</p>

<p style="margin: 0 0 1em 0;">Voices
현재 재생 중인 보이스 수 표시.</p>

<p style="margin: 0 0 1em 0;">RAM Load / Samples / Patches
현재 사용 중인 RAM 용량, 샘플 수, 패치 수 표시.</p>


### Flow and Precision Presets

<p style="margin: 0 0 1em 0;">대부분의 라이브러리는 두 종류의 프리셋을 제공함. Synchron Player 상단의 Flow 아이콘 또는 “Precision”을 클릭해 전환함.</p>

<p style="margin: 0 0 1em 0;">Precision Presets
VSL 라이브러리의 기존 구조와 동일한 방식임. 라이브러리의 세부 기능을 직접 조정하며 사용할 수 있음.</p>

<p style="margin: 0 0 1em 0;">Flow Presets
라이브러리 콘텐츠를 한 화면에서 직관적으로 확인하는 구조임. Flow 브라우저에서 프리셋을 선택하면 악기, 아티큘레이션, 사운드 파라미터가 하나의 창에 함께 표시됨.</p>


### MIDI Player

<p style="margin: 0 0 1em 0;">Synchron Power Drums 및 Synchron World Percussion용 드럼 그루브 재생 기능임. 시퀀서의 현재 재생 위치와 템포에 맞춰 동작함. MIDI 파일의 벨로시티와 템포를 조정한 뒤 드래그 앤 드롭으로 시퀀서에 가져올 수 있음.</p>

<p style="margin: 0 0 1em 0;">Power Drums에는 “3 drum sets in 1 MIDI file” 폴더가, World Percussion에는 “Full ensemble for DAW” 폴더가 포함되어 있음. 해당 폴더의 MIDI 파일은 여러 MIDI 채널로 분리되어 있어 각 악기를 동시에 사용할 수 있음.</p>

<p style="margin: 0 0 1em 0;">Factory 폴더의 MIDI 파일은 그루브 요소가 악기별로 정리되어 있고, 원하는 조합으로 그루브를 구성할 수 있음.</p>

<img src="/images/05.-GUI-Overview-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">FACTORY
Synchron Power Drums에서 제공하는 커스텀 루프 선택.</p>

<p style="margin: 0 0 1em 0;">USER
사용자 MIDI 루프 컬렉션 탐색. MIDI Player 기능으로 재생 및 편집 가능함.</p>

<img src="/images/05.-GUI-Overview-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">FAVORITES
폴더를 우클릭해 즐겨찾기로 지정하면 여기에 표시됨.</p>

<p style="margin: 0 0 1em 0;">Browser
Finder/탐색기와 같은 방식으로 폴더 구조 탐색.</p>

<p style="margin: 0 0 1em 0;">Loop Data
선택한 파일의 이름, 원본 템포, 마디 수 표시.</p>

<p style="margin: 0 0 1em 0;">Play/Pause
클릭하면 선택한 루프 재생 시작. 파일 이름을 더블클릭해도 동일하게 동작함.</p>

<p style="margin: 0 0 1em 0;">Host Sync
활성화하면 Play/Pause 버튼이 비활성화되며, 시퀀서 재생과 함께 동기화됨.</p>

<p style="margin: 0 0 1em 0;">Loop
선택한 그루브 반복 재생 활성화.</p>

<p style="margin: 0 0 1em 0;">Velocity
루프 전체 벨로시티 조정. 시퀀서로 드래그할 때 이 값이 함께 적용됨.</p>

<p style="margin: 0 0 1em 0;">Tempo
하프 타임, 일반, 더블 타임 선택. 시퀀서로 드래그할 때 설정 값이 함께 적용됨.</p>

<p style="margin: 0 0 1em 0;">Loop Position
선택한 그루브의 현재 재생 위치 표시.</p>

<p style="margin: 0 0 1em 0;">Loop List
사용 가능한 루프 목록과 현재 선택 항목 표시.</p>


### Key Info

<p style="margin: 0 0 1em 0;">매핑 구조를 확인할 수 있는 이동 가능한 플로팅 창.</p>

<p style="margin: 0 0 1em 0;">키 스위치와 매핑 정보를 함께 표시함. 타악기처럼 여러 아티큘레이션이나 히트가 하나의 패치 안에서 서로 다른 건반에 할당된 경우, 전체 구조를 한눈에 확인할 수 있음.</p>

<img src="/images/05.-GUI-Overview-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Key Descriptions
선택한 슬롯의 매핑 세부 정보 표시.</p>

<p style="margin: 0 0 1em 0;">Keyswitches
키보드와 Key Info 창에 함께 표시됨. 디멘션 트리의 해당 디멘션 색상과 동일하게 구분됨.</p>

<p style="margin: 0 0 1em 0;">Range Display / Adjustment
Range Display를 활성화하면 선택한 패치/슬롯의 연주 범위를 표시함(가장 오른쪽 디멘션 기준). 선택한 슬롯의 연주 범위를 위아래로 최대 1옥타브까지 확장 또는 축소 가능함.</p>
