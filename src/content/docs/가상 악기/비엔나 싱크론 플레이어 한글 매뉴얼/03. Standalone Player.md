<p style="margin: 0 0 1em 0;">Vienna Synchron Player를 독립 실행형 가상 악기로 사용할 때, 스탠드얼론 버전은 시퀀서와 연결되지 않으므로 먼저 오디오와 MIDI를 설정해야 함. 나머지 기능은 플러그인 버전과 동일함.</p>


### <strong>Preferences</strong>

<p style="margin: 0 0 1em 0;">설정은 오른쪽 아래의 렌치 아이콘을 누르면 열림.</p>

<img src="/images/03.-Standalone-Player-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">오디오 탭에서는 사용할 사운드카드를 고를 수 있고, 샘플 레이트와 레이턴시도 여기에서 설정함. 오디오 라우팅 탭에서는 선택한 사운드카드에 맞춰 출력 채널을 지정할 수 있음.</p>

<img src="/images/03.-Standalone-Player-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<img src="/images/03.-Standalone-Player-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">사용할 MIDI 포트는 MIDI 탭에서 선택함.</p>
