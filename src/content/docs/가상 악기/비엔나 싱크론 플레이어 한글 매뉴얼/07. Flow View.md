이건 Vienna Synchron Player의 공통 조작 개념 설명이고, 주로 오케스트라 악기나 샘플 라이브러리 전반을 대상으로 한 내용임. 


피아노 가상악기와 아예 무관한 건 아니지만, Dynamics, Expression, Timbre, Reverb, Output 을 제외하고는 대부분 VSL에서 제공하는 다른 악기들을 제어하는 기능임. 



<img src="/images/07.-Flow-View-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

Flow Preset을 쓰면 Synchron Player의 Flow 화면에서 악기의 주요 설정, 주법, 전체적인 소리 설정을 바로 만질 수 있음. 화면은 항목별로 나뉘어 있고, 아래에는 가상 키보드가 표시됨. 프리셋 목록에서 악기를 고르면 그 악기 화면이 열림.



<img src="/images/07.-Flow-View-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>Instrument / Tone</strong>

Instrument 섹션에는 지금 선택된 악기 그림이 보이고, 위에는 불러온 프리셋 이름이 표시됨. 이 부분을 클릭하면 다른 악기로 바꿀 수 있음. 또 악기 창의 Info Hub 아이콘을 누르면 안내 투어를 시작하거나, 악기 정보를 볼 수 있는 Vienna Academy로 들어갈 수 있음.

악기 아래 있는 Dynamics 페이더는 기본적으로 모드휠(MIDI CC1)이나 건반 세기(벨로시티)에 연결되어 있음. 예를 들어 롱 노트에서는 보통 모드휠을 써서, 소리가 나는 중에도 음량을 자연스럽게 올리거나 내릴 수 있음. 관악기처럼 손을 덜 쓰고 싶으면 브레스 컨트롤러(MIDI CC2)로 바꿔 쓸 수도 있음. 반대로 스타카토는 건반 벨로시티로 설정되는 경우가 많은데, 짧게 끊어 치는 음은 이미 난 뒤에 음량을 바꾸는 게 거의 의미 없기 때문임.



<img src="/images/07.-Flow-View-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

Tone
연주를 더 자연스럽고 살아 있게 만드는 세 가지 설정을 조절할 수 있음.
Humanization(MIDI CC27)
연주 타이밍을 일부러 조금씩 어긋나게 해서, 너무 기계적으로 딱 맞지 않게 만드는 기능임. 쉽게 말해, 자연스러운 흔들림을 더해주는 설정임.
Timbre(MIDI CC8)
악기 음색을 조절하는 기능임. 값을 바꾸면 소리가 더 밝아지거나 더 부드럽고 어둡게 들리게 됨.
Expression(MIDI CC11)
추가 볼륨 조절 역할을 하는 기능임. 전체 음량을 더 세밀하게 다듬을 때 씀.


### <strong>Articulation / Performance</strong>

Articulation
그 악기로 낼 수 있는 주법들을 고를 수 있음. 위쪽에는 선택 가능한 주법 목록이 보이고, 각각에 연결된 키스위치도 함께 표시됨. 이 키스위치는 Synchron Player의 가상 키보드에서도 확인 가능함. 필요하면 해당 칸을 직접 클릭해서 원하는 주법으로 바로 바꿀 수도 있음.



<img src="/images/07.-Flow-View-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

Performance
선택한 주법 안에서 세부 연주 방식들을 조절할 수 있음. 어떤 주법은 옵션이 단순하고, 어떤 주법은 더 다양하게 나뉨. 예를 들어 staccato는 bold와 agile 정도만 고를 수 있고, 이 차이도 연주 속도에 따라 반영됨. 반면 롱 노트는 조절할 수 있는 항목이 더 많고, 악기에 따라 구성도 달라짐.

Legato
MIDI CC3으로 켜고 끌 수 있음. “Universal” 아티큘레이션을 선택하면 MIDI CC3이 단순한 스위치가 아니라 페이더처럼 표시되어, tremolo와 legato를 섞어 조절할 수 있음.

Legato를 켜면 추가 버튼이 나타나고, 서스테인 페달(MIDI CC64) 동작에 따라 음 사이 연결을 legato로 할지 portamento로 할지 바꿀 수 있음.

First note
건반을 누르는 세기, 즉 벨로시티에 따라 결정되며, 음이 처음 시작될 때의 어택 느낌에 영향을 줌.

Release
MIDI CC4로 조절하며, 음을 뗄 때의 마무리를 normal과 soft 사이에서 바꿀 수 있음.

일부 악기, 예를 들어 Prime Edition의 플루트에는 Vibrato 페이더도 있는데, 이건 MIDI CC20에 연결되어 있고, 비브라토 없는 소리와 비브라토 있는 소리 사이를 자연스럽게 섞어 조절할 수 있게 해줌.


### Sound

Sound 창에서는 먼저 Standard나 FX Mixer Preset 중 하나를 골라 기본 믹서 설정을 정할 수 있음. 이 설정이 전체 사운드의 기본 틀을 잡아줌. 

Mic Balance(MIDI CC61)는 가까운 마이크 소리와 공간감이 담긴 마이크 소리의 비율을 조절하는 기능임.

Duality Strings Essentials에서는 이 항목 대신 Stage Balance가 들어감. 여기서는 큰 홀에서 녹음한 Stage A 사운드와, 좀 더 작고 가까운 느낌의 Stage B 사운드를 얼마나 섞을지 정할 수 있음. 즉, 웅장한 느낌과 친밀한 느낌 사이를 원하는 만큼 조절하는 기능임.

FX/Reverb(MIDI CC5)는 이펙트나 리버브가 얼마나 걸릴지를 조절하는 기능임.

마지막으로 Output Volume(MIDI CC63)은 Synchron Player에서 최종적으로 나가는 전체 소리 크기를 조절하는 항목임.
