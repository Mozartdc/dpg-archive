<p style="margin: 0 0 1em 0;">디멘션 트리 뷰는 Synchron Player의 중심 화면입임. 세로 한 줄이 하나의 디멘션이며, 각 디멘션에는 여러 슬롯이 들어 있고, 흰 테두리는 현재 선택된 슬롯임. 각 디멘션에서 선택된 슬롯을 왼쪽부터 차례로 보면, 지금 어떤 아티큘레이션 조합을 쓰고 있는지 알 수 있음.</p>

<p style="margin: 0 0 1em 0;">아직 샘플을 불러오지 않은 슬롯은 회색으로 표시되고, 슬롯 이름 왼쪽 칸을 더블클릭해서 불러오면 활성화됨.</p>



<img src="/images/06.-Dimension-Tree-View-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>Switch Controls</strong>



<img src="/images/06.-Dimension-Tree-View-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">각 디멘션은 세 가지 방식으로 전환할 수 있음.</p>

- None이면 MIDI로 전환하지 않음.
- Note, White Note, Program Change, Sequence, Interval에서는 노드를 단계별로 선택함.
- Dim.Ctrl에서는 노드를 연속으로 전환하거나 크로스페이드함.
<p style="margin: 0 0 1em 0;"><strong>Sequence</strong></p>

<p style="margin: 0 0 1em 0;">Sequence를 쓰면 노드가 차례대로 바뀜. 옆의 두 값은 리셋 기준과 코드 묶음 기준임.</p>



<img src="/images/06.-Dimension-Tree-View-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">왼쪽 값은 리셋 기준임. 한동안 새 노트가 안 들어오면 처음으로 돌아감.</p>

<p style="margin: 0 0 1em 0;">오른쪽 값은 코드로 묶는 기준임. 아주 짧은 간격으로 들어온 노트는 다음 순서로 넘기지 않고 한 덩어리로 처리함.</p>

<p style="margin: 0 0 1em 0;"><strong>Interval</strong></p>

<p style="margin: 0 0 1em 0;">가운데 슬롯을 기준으로, 레가토로 이어진 음 사이 간격에 따라 슬롯이 바뀜. 새로 시작하는 음은 항상 가운데 슬롯에서 시작하고, 그다음부터는 겹쳐 들어오는 음의 간격에 맞춰 다른 슬롯으로 넘어감.</p>
