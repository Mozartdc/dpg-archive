### <strong>Preset Browser</strong>

Synchron 및 Studio Series 제품에는 각 악기 섹션별로 다양한 프리셋이 들어 있음. 전체 구조는 제품마다 일관된 프리셋 설계와 슬롯 구성을 유지하도록 짜여 있음. 악기 사운드를 다듬을 때는 제공되는 믹서 프리셋을 확인하면 됨.



<img src="/images/08.-Browser-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

Factory/User 프리셋 폴더는 기본 상태를 접힘 또는 펼침으로 지정할 수 있음. 폴더 이름을 우클릭하면 관련 옵션이 나타남.



<img src="/images/08.-Browser-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />



<img src="/images/08.-Browser-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>Creating User Presets</strong>

사용자 프리셋을 만들려면 User 폴더나 하위 폴더 위에 마우스를 올리면 됨. 그러면 새 폴더를 만들거나 현재 프리셋을 선택한 폴더에 저장하는 옵션이 나타남.



<img src="/images/08.-Browser-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

User 폴더 위에 마우스를 올리면 하위 폴더 만들기, 프리셋 저장, 폴더 삭제 옵션이 나타남. 폴더를 삭제하면 그 안의 프리셋도 함께 삭제됨. 프리셋 이름 위에 마우스를 올리면 현재 불러온 설정을 선택한 프리셋에 덮어쓰거나 프리셋을 삭제할 수 있음. 삭제할 때는 확인 메시지가 표시됨.


### <strong>Patch Browser</strong>

패치 브라우저에서는 악기별로 녹음된 개별 아티큘레이션을 확인할 수 있음. 이 자료를 바탕으로 직접 프리셋을 만들 수도 있음. 패치 구조 각 Synchron Series 제품의 라이브러리 매뉴얼을 참고.



<img src="/images/08.-Browser-5.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />


### <strong>MixerPreset Browser</strong>



<img src="/images/08.-Browser-6.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

믹서 프리셋 브라우저에서는 악기 프리셋에 저장된 믹서 프리셋을 더 쉽게 불러오고 관리할 수 있음. Factory 프리셋 목록은 MIX 탭과 같지만, 직접 만든 믹서 프리셋은 브라우저에서 저장하는 쪽이 훨씬 편함.

새 믹서 프리셋을 만들면 악기 프리셋도 함께 저장하라는 안내가 뜨는데, User 폴더 안에 원래의 폴더 배치를 따라 저장할지 선택할 수 있음. 이렇게 저장한 사용자 악기 프리셋을 나중에 열면, 거기에 연결된 사용자 믹서 프리셋도 함께 불러와 바로 쓸 수 있음. 악기 프리셋에서 바꾼 다른 설정도 같이 저장됨.
