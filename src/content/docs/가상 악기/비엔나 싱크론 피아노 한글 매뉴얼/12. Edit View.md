<p style="margin: 0 0 1em 0;">이 화면에서는 노트별로 이퀄라이저를 설정할 수 있음.
노트별 EQ는 선택한 노트들의 설정 값이 모두 같을 때만 사용할 수 있음.</p>

<p style="margin: 0 0 1em 0;">EQ Display
선택한 노트의 EQ 그래프를 표시함.</p>

<p style="margin: 0 0 1em 0;">EQ Node
더블클릭하면 해당 노드가 활성화됨. 마우스 휠로 Q값 조절하고 Option 또는 Alt+클릭하면 주파수가 고정됨. 미세 조정은 Shift+드래그로 , 초기화는 Command 또는 Ctrl+클릭.</p>

<p style="margin: 0 0 1em 0;">Gain
EQ Display에서 선택한 노드의 게인 값을 표시함 Shift+드래그로 미세 조정할 수 있고, 클릭 드래그 또는 더블클릭으로 값을 수정함. 초기화는 Command 또는 Ctrl+클릭.</p>

<p style="margin: 0 0 1em 0;">Frequency
EQ Display에서 선택한 노드의 주파수를 표시함 조작 방식은 Gain과 동일함.</p>

<p style="margin: 0 0 1em 0;">Q
EQ Display에서 선택한 노드의 Q 값을 표시함 조작 방식은 Gain과 동일함.</p>

<p style="margin: 0 0 1em 0;">Reset
선택한 노트의 EQ 설정을 초기화함.</p>

<p style="margin: 0 0 1em 0;">Velocity Curve
키보드 벨로시티 매핑을 조절함 벨로시티 히스토그램 함께 표시됨 더블클릭하면 노드가 추가됨. Command 또는 Ctrl+클릭으로 삭제할 수 있고, 클릭 후 드래그하면 여러 노드을 동시에 선택할 수 있음. 우클릭 메뉴에서 프리셋을 로드하거나 저장할 수 있고, Velocity Curve를 초기화 할 수 있음. 또, 다른 프리셋이나 Synchron Pianos 인스턴스로 복사나 붙여넣기 를할 수 있음.</p>

<p style="margin: 0 0 1em 0;">Global Dynamic Range
Play View의 Dynamic Range Controller와 동일하게 동작함.</p>

<p style="margin: 0 0 1em 0;">Global MIDI Sensitivity
Play View의 MIDI Sensitivity Controller와 동일하게 동작함.</p>

<p style="margin: 0 0 1em 0;">Half Pedal Range
연속 Sustain Pedal Controller의 상단 및 하단 페달 위치를 설정함. Settings에서 활성화해야 적용됨.</p>

<img src="/images/12.-Edit-View-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Note Dynamic Range
선택한 키의 다이내믹 레인지를 표시함 Adjustment Area를 Dynamic Range로 바꾸면 동일한 값이 슬라이더 영역에도 함께 표시됨.</p>

<p style="margin: 0 0 1em 0;">Note Volume
선택한 노트의 볼륨 레벨을 표시함 선택한 노트들의 레벨이 모두 같을 때만 표시되고, 선택 상태에 따라 Volume Adjustment Area의 슬라이더와 동일하게 동작함.</p>

<p style="margin: 0 0 1em 0;">Note Tuning
선택한 키의 튜닝 값을 표시함 Adjustment Area를 Tuning으로 바꾸면 동일한 값이 슬라이더 영역에도 함께 나타남.</p>

<p style="margin: 0 0 1em 0;">Note Dynamic Range</p>

<p style="margin: 0 0 1em 0;">선택한 키의 다이내믹 레인지를 표시함 Adjustment Area를 Dynamic Range로 전환하면 동일한 값이 슬라이더 영역에도 함께 표시됨.</p>

<p style="margin: 0 0 1em 0;">Note Volume</p>

<p style="margin: 0 0 1em 0;">선택한 노트의 볼륨 레벨을 표시함 선택한 노트들의 레벨이 모두 같을 때만 표시되고, 선택 상태에 따라 Volume Adjustment Area 슬라이더와 동일하게 작동함.</p>

<p style="margin: 0 0 1em 0;">Note Tuning</p>

<p style="margin: 0 0 1em 0;">선택한 키의 튜닝 값을 표시함. Adjustment Area를 Tuning으로 전환하면 동일한 값이 슬라이더 영역에도 표시됨.</p>

<p style="margin: 0 0 1em 0;">Max Voices/Mic</p>

<p style="margin: 0 0 1em 0;">마이크당 최대 동시 발음 수를 설정함. 값을 낮추면 연주 음이 많을 때도 안정적임. (기본값 128.)</p>

<p style="margin: 0 0 1em 0;">Max Voices/Key
키당 최대 동시 발음 수를 설정함 매우 느린 연주에서는 값을 높이면 자연스러운 잔향을 표현할 수 있음. (기본값 4.)</p>


### <strong>Advanced Settings</strong>

<p style="margin: 0 0 1em 0;">Release Sample 설정과 Repetition Smear 기능으로 Yamaha CFX의 반응을 세밀하게 조정함.</p>

<p style="margin: 0 0 1em 0;">RX Fix
고정 Release 확장 시간을 설정하고, Release Sample에 그대로 더해짐. (기본값 0초)</p>

<p style="margin: 0 0 1em 0;">RX Dyn
노트와 벨로시티에 따라 달라지는 Release 확장 시간 을 조절하고, Note-on Sample에 추가 적용됨. (기본값 0.5초)</p>

<p style="margin: 0 0 1em 0;">RX Dyn.Shape
노트와 벨로시티에 따른 Release 확장 시간의 곡선 형태를 조절함. (기본값 2)</p>

<p style="margin: 0 0 1em 0;">Repetition Smear
Repetition Smear 기능을 켜고 끔.(기본값 켜짐)</p>

<p style="margin: 0 0 1em 0;">RS Level
Release Sample의 볼륨 레벨을 조절함. 잔향이 과할 경우 알고리즘 리버브 값을 조금 낮추면 조절할 수 있음. (기본값 0 dB)</p>

<img src="/images/12.-Edit-View-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">위 그래프는 RX Dyn.Shape 파라미터의 동작 방식을 나타냄 1.0은 주황 영역을, 2.0은 파란 영역, 5.0은 녹색 영역을 의미함.
X축과 Y축에는 MIDI 노트와 벨로시티 값을, Z축에는 RX 배수 값을 계산함. 이 Z축 값이 RX Dyn 값에 곱해져 Sustain Sample에 더해질 Release 시간을 산출함.
가장 낮은 노트와 가장 높은 벨로시티 조합에서는 항상 값이 1.0으로 나오기 때문에 Release 시간은 RX Dyn에 설정한 값이 그대로 적용됨.</p>

<img src="/images/12.-Edit-View-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Volume
하단 17번 Adjustment Area를 Volume 모드로 전환함.</p>

<p style="margin: 0 0 1em 0;">Dynamic Range
하단 17번 Adjustment Area를 Dynamic Range 모드로 전환함.</p>

<p style="margin: 0 0 1em 0;">Tuning
하단 17번 Adjustment Area를 Tuning 모드로 전환함.</p>

<p style="margin: 0 0 1em 0;">Adjustment Area
핸들을 드래그해서 값을 조절함 Command 또는 Ctrl+클릭하면 핸들이 초기화됨.</p>

<p style="margin: 0 0 1em 0;">Selection Area (Keyboard)
Shift로 키를 범위로 선택할 수 있고, Command 또는 Ctrl로 개별 키를 추가하고, Option 또는 Alt 키는 선택을 해제함.</p>


### Adjustment Area

<p style="margin: 0 0 1em 0;">선택 영역 밖을 한 번 클릭하면 해당 노트만 선택됨. 조절 범위는 -10에서 +10 dB이며, 선택한 상태에서 핸들을 드래그하면 선택한 모든 노트 레벨이 함께 변경되고, 우클릭 또는 더블클릭하면 레벨 0으로 초기화됨.</p>

<p style="margin: 0 0 1em 0;">Option 또는 Alt 키를 누른 상태에서 드래그하면 시작 지점과 끝 지점의 값을 기준으로 두 노트 사이 값이 직선 비율로 자동 분배됨. 시작 노트 값과 끝 노트 값 사이가 단계적으로 증가하거나 감소하는 형태로 자동 계산되고, 이 기능은 현재 선택된 노트와 관계없이 드래그한 구간에 바로 적용됨.</p>


### <strong>election Area (Keyboard)</strong>

- 드래그하면 드래그한 범위로 선택 영역이 바뀜.
- Shift+드래그하면 기존 선택에 노트가 추가됨.
- Option 또는 Alt+드래그하면 기존 선택에서 해당 노트가 제외됨.
- Command 또는 Ctrl+드래그하면, 처음 클릭한 노트 상태 기준으로 선택이 반전됨.
- 선택하지 않은 노트에서 시작하면 기존 선택에 노트가 추가됨. Shift 동작과 동일함
- 이미 선택한 노트에서 시작하면 선택이 해제됨. Option 또는 Alt+동작과 동일함.
