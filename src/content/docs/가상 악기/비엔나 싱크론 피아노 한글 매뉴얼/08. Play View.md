<p style="margin: 0 0 1em 0;">Play View는 Synchron Pianos 인터페이스의 메인 화면으로, 피아노의 기본적인 소리 성향과 공간감을 조절하는 곳임. 여기서 볼륨, 리버브, 다이내믹 레인지뿐 아니라 Body, Sympathetic Resonance, Timbre Shift 같은 피아노 고유의 세부 파라미터를 조정할 수 있음.</p>

<p style="margin: 0 0 1em 0;"><strong>Volume</strong></p>

<img src="/images/08.-Play-View-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">악기의 전체 출력 레벨을 조절함. 메인 화면 왼쪽에 있는 Volume 노브와 동일하게 작동하며, 드래그로 값을 바꾸거나 더블 클릭으로 직접 수치를 입력할 수 있음. Command/Ctrl-클릭을 하면 기본값으로 되돌아감.</p>

<p style="margin: 0 0 1em 0;">Reverb Control은 각 마이크 채널에서 리버브로 보내는 양을 정함. 주로 Close나 Mid 마이크에 적용되며, 실제 리버브의 종류나 세부 설정은 화면 오른쪽 컬럼에서 따로 조절함.</p>

<p style="margin: 0 0 1em 0;">Volume Control은 현재 설정된 볼륨 값을 슬라이더 형태로 보여주며, 조작 방식은 Volume 노브와 동일함. 노브와 슬라이더는 같은 파라미터를 서로 다른 방식으로 표시한 것임.</p>

<p style="margin: 0 0 1em 0;">Dynamic Range Control은 연주 강약에 따른 음량 차이를 조절함. 값을 넓히면 약하게 쳤을 때와 세게 쳤을 때의 차이가 커지고, 좁히면 전체적으로 균일한 반응을 보임.</p>

<p style="margin: 0 0 1em 0;"><strong>Pedals</strong></p>

<p style="margin: 0 0 1em 0;">피아노의 각종 페달 동작을 제어하는 영역임. 실제 그랜드 피아노의 페달 동작을 모델링한 것으로, 모델에 따라 특수 페달도 포함됨.</p>

<p style="margin: 0 0 1em 0;">Fourth Pedal은 Fazioli F308 모델에만 있는 특수 페달임. 해머를 현에 더 가깝게 이동시켜 업라이트 피아노에 가까운 터치감을 만들고, 전체 음량을 줄이는 역할을 함. MIDI CC 75에 할당되어 있음.</p>

<p style="margin: 0 0 1em 0;">Soft Pedal은 우나 코르다(una corda) 방식의 소프트 페달임. 건반과 액션이 옆으로 이동하면서 타현 위치가 바뀌어 소리가 더 부드럽고 얇아짐. 우클릭하면 항상 켜진 상태로 유지할 수 있으며, MIDI CC 67을 사용함.</p>

<p style="margin: 0 0 1em 0;">Sost Pedal은 소스테누토 페달로, 페달을 밟은 순간에 눌려 있던 음만 선택적으로 유지함. 우클릭으로 고정할 수 있고, MIDI CC 66에 할당됨.</p>

<p style="margin: 0 0 1em 0;">Sustain Pedal은 댐퍼를 현에서 들어 올려 모든 음을 지속시키는 일반적인 서스테인 페달임. 원형 테두리를 통해 반페달 상태가 시각적으로 표시되며, CC 값 26에서 102 사이 구간에서 하프 페달이 적용됨. 우클릭 시 항상 활성화할 수 있고, MIDI CC 64를 사용함.</p>

<p style="margin: 0 0 1em 0;"><strong>Parameters</strong></p>

<p style="margin: 0 0 1em 0;">피아노의 음색과 반응을 세부적으로 조정하는 영역임.</p>

<p style="margin: 0 0 1em 0;">Body는 각 음의 기본 진동 특성을 바꿔 피아노가 크게 울리는지, 작고 단단하게 들리는지를 조절함. 값을 올리면 악기가 더 큰 공간에서 나는 듯한 인상을 줌.</p>

<p style="margin: 0 0 1em 0;">Sympathetic Resonance는 공명음을 조절함. 소리를 내지 않고 건반을 눌러 둔 상태에서 다른 음을 쳤을 때 함께 울리는 현의 공명으로, 실제 피아노의 공진 현상을 재현한 것임. 근거리 마이크에 잡히는 소리에만 영향을 줌.</p>

<p style="margin: 0 0 1em 0;">Timbre Shift는 샘플 재생 속도를 미세하게 바꿔 음색 성향을 조절함. 값을 낮추면 소리가 더 따뜻해지고, 높이면 밝고 선명해짐. 반음 단위로 움직이며, 특히 약하게 연주할 때 음색 변화가 잘 드러남.</p>

<p style="margin: 0 0 1em 0;">Smooth Attack은 낮은 벨로시티 연주에서 어택을 부드럽게 정리해 줌. 작은 소리로 연주할 때 거친 느낌을 줄이고 싶을 때 사용함.</p>

<img src="/images/08.-Play-View-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">MIDI Sensitivity는 건반을 얼마나 세게 쳤을 때 소리가 얼마나 반응할지를 조절함. 값이 높을수록 작은 힘에도 다이내믹 변화가 크게 나타남.</p>

<p style="margin: 0 0 1em 0;">Half Pedal은 연속 CC 데이터를 보내는 서스테인 페달을 사용할 때 반페달 효과의 강도를 조절함. 실제 피아노처럼 페달을 조금만 밟았을 때의 댐핑 정도를 정하는 설정임.</p>

<p style="margin: 0 0 1em 0;">Pedal Noise는 서스테인 페달을 밟고 뗄 때 나는 기계적 소리의 크기임. 실제 연주감을 살리고 싶을 때 사용함.</p>

<p style="margin: 0 0 1em 0;">Key Release Noise는 건반에서 손을 뗄 때 발생하는 소음의 크기를 조절함.</p>

<p style="margin: 0 0 1em 0;">Tuning은 전체 기준음을 조절하는 항목으로, 기본값은 A=440Hz임.</p>

<p style="margin: 0 0 1em 0;">Octave는 음역을 옥타브 단위로 위아래로 이동시킴.</p>

<p style="margin: 0 0 1em 0;">Semitone은 음역을 반음 단위로 이동시키는 설정임.</p>

<p style="margin: 0 0 1em 0;"><strong>Context Menu</strong></p>

<img src="/images/08.-Play-View-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">컨텍스트 메뉴는 Play View에서 각 컨트롤을 우클릭했을 때 열리는 설정 메뉴임.(페달, 옥타브/세미톤 컨트롤은 제외)</p>

<p style="margin: 0 0 1em 0;">Selected Control에는 지금 우클릭한 컨트롤의 이름이 표시됨.</p>

<p style="margin: 0 0 1em 0;">Assigned Source는 해당 컨트롤에 이미 MIDI나 자동화 소스가 연결돼 있을 때만 나타남. 어떤 입력이 연결돼 있는지 확인하는 용도임.</p>

<p style="margin: 0 0 1em 0;">Learn을 누르면 MIDI Learn 모드가 활성화됨. 이후 컨트롤러에서 노브나 페달을 움직이면 그 입력이 해당 컨트롤에 자동으로 할당됨.</p>

<p style="margin: 0 0 1em 0;">Available Control Sources에서는 MIDI CC, 내부 모듈 등 사용 가능한 입력 소스를 목록에서 직접 선택해 할당할 수 있음.</p>
