### Interface Settings

<img src="/images/14.-Setting-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;"><strong>Middle C </strong>: C3, C4, C5 중 하나를 Middle C로 지정할 수 있음. (기본값은 C4)</p>

<p style="margin: 0 0 1em 0;"><strong>MIDI 덤프 CC</strong> : MIDI 컨트롤러를 통해 MIDI 덤프 신호를 보낼 CC 번호를 할당함.</p>

<p style="margin: 0 0 1em 0;"><strong>페달 CC</strong> : 페달 신호에 대한 MIDI CC 값을 조정함. 기본값은 서스테인(64), 소스테누토(66), 소프트 페달(67)이며, Synchron Fazioli F308은 4번 페달(75)으로 사용함.</p>

<p style="margin: 0 0 1em 0;"><strong>연속 서스테인 페달 활성화 </strong>: 하프 페달 등 연속적인 신호를 보내는 페달을 사용할 때 활섵화함. (기본값은 OFF)</p>


### Engine Settings 

<img src="/images/14.-Setting-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;"><strong>Default Preload Size : </strong>RAM 사용량과 로딩 시간을 관리함. 낮은 설정은 메모리를 절약하고, 높은 설정은 CPU 부담을 낮춤. 드라이브 속도 측정에 따라 보통 3072 샘플로 설정됨.</p>

<p style="margin: 0 0 1em 0;"><strong>Streaming Threads : </strong>곡을 재생할 때 드라이브에서 데이터를 읽어오는 동시 작업 수. (기본값인 4를 권장)</p>

<p style="margin: 0 0 1em 0;"><strong>Loading Threads : </strong>샘플을 RAM으로 불러올 때 드라이브에서 동시에 데이터를 읽어오는 수치. (기본값 4)</p>

<p style="margin: 0 0 1em 0;"><strong>Pan Law : </strong>소리의 위치를 좌우로 조절할 때 신호 세기가 변하는 정도 설정. (기본값은 -3 dB)</p>

<p style="margin: 0 0 1em 0;"><strong>Enable Multicore Processing : </strong>Standalone에서는 항상 켜져 있고, 시퀀서 안에서 플러그인으로 쓸 때는 프로젝트 규모에 따라 활성화 여부를 선택 할 수 있음. 대규모 작업 시에는 사용 중인 DAW의 멀티코어 처리 방식에 따라 끄는 것을 권장함.</p>

<p style="margin: 0 0 1em 0;"><strong>Unload Samples with Mixer Channels : </strong>이 기능을 켜면 특정 믹서 채널을 비활성화할 때 해당 채널에 할당된 샘플도 함께 삭제하여 RAM 용량을 확보할 수 있음.</p>


### Database Settings 

<img src="/images/14.-Setting-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;"><strong>User Preset Folder : </strong>사용자 프리셋이 저장된 경로를 보여줌. 경로설정이 잘못되었거나 해당 위치에 폴더가 없으면 빨간색으로 보임.</p>

- Windows 기본 경로: C: > Users > Public > Documents
- macOS 기본 경로: /Users/Shared
<p style="margin: 0 0 1em 0;"><strong>Preset Folder Assignment Button : </strong>사용자 프리셋을 저장하고 관리할 폴더를 원하는 위치로 직접 지정함. 경로설정에 문제가 있으면 빨간색으로 보임..</p>

<p style="margin: 0 0 1em 0;"><strong>Volumes Scan Paths : </strong>현재 소프트웨어가 인식하고 있는 모든 샘플 라이브러리 경로 목록.  샘플이 여러 하드 드라이브에 나누어져 있어도 모두 표시됨.</p>

<p style="margin: 0 0 1em 0;"><strong>Preload Size : </strong>악기를 불러올 때 RAM에 미리 올려둘 샘플 데이터의 크기. Engine 탭에서 정한 기본값을 따르지만, 여기서 각 라이브러리마다 이 수치를 따로 조절할 수 있음.</p>

<p style="margin: 0 0 1em 0;"><strong>Speed : </strong>드라이브의 데이터 전송 속도를 측정해서 보여줌.</p>

<p style="margin: 0 0 1em 0;"><strong>Drive Information : </strong>드라이브에 대한 전체적인 정보 출력.</p>

<p style="margin: 0 0 1em 0;"><strong>Add/Remove Path : </strong>샘플이 저장된 폴더 경로를 추가하거나 제거할 수 있음. 설정을 변경한 경우에는 DAW를 완전히 종료한 뒤 다시 실행해야 적용됨. Vienna Assistant가 이 경로를 자동으로 관리하므로, 수동으로 수정하는 것은 권하지 않음.</p>

<p style="margin: 0 0 1em 0;"><strong>Reset Preload Sizes : </strong>사용자가 변경한 모든 Preload Size 값을 기본값으로 되돌림.</p>

<p style="margin: 0 0 1em 0;"><strong>Rescan Speeds : </strong>샘플이 저장된 드라이브의 읽기 속도를 다시 측정함. 이 값은 해당 드라이브가 HDD인지 SSD인지 판단하기 위한 참고 수치일 뿐, 실제 성능을 정확하게 판단하는 기준은 아님.</p>
