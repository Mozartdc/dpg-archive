### <strong>Delay</strong>

<p style="margin: 0 0 1em 0;">에코 효과를 만들거나 사운드 디자인용으로 사용하는 딜레이임. 두 개의 내장 필터로 특정 주파수 대역만 딜레이에 걸 수 있음.
</p>

<img src="/images/11.-Channel-FX-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">TIME: 딜레이가 반복되는 간격을 밀리초 단위로 설정함.</p>

<p style="margin: 0 0 1em 0;">SYNC: 딜레이 시간을 템포에 맞춰 음표 값으로 설정함. 일반, 점, 셋잇단음표 선택 가능함. 예로 T16은 점 16분음표 기준으로 작동함.</p>

<p style="margin: 0 0 1em 0;">FEEDBACK: 딜레이된 소리가 다시 입력으로 돌아가는 양을 조절함. 값이 높을수록 반복 횟수 늘어남.</p>

<p style="margin: 0 0 1em 0;">DAMPING: 딜레이 신호의 고역을 줄임. 노브를 낮출수록 고주파가 더 빨리 감쇠됨.</p>

<p style="margin: 0 0 1em 0;">THINNING: 딜레이 신호의 저역을 줄임. 노브를 높일수록 저주파가 더 많이 제거됨.</p>

<p style="margin: 0 0 1em 0;">SPREAD: 딜레이 신호의 스테레오 퍼짐 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;">0%는 일반 스테레오 딜레이, 50%는 모노 딜레이, 100%는 좌우가 번갈아 나오는 핑퐁 딜레이임.</p>

<p style="margin: 0 0 1em 0;">DRY: 원본 신호의 출력 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">WET: 딜레이가 적용된 신호의 출력 레벨을 설정함.</p>


### <strong>Distortion</strong>

<p style="margin: 0 0 1em 0;"><strong>Saturator</strong></p>

<img src="/images/11.-Channel-FX-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">테이프 새추레이션과 테이프 머신 과부하 특유의 질감을 재현하는 왜곡 효과임. 입력 신호를 오버드라이브해 배음을 추가하고, 소리를 더 두껍고 밀도 있게 만듦.</p>

<p style="margin: 0 0 1em 0;">DRIVE: 입력 신호의 증폭량을 설정함. 값을 올릴수록 오버드라이브가 강해지고 왜곡이 증가함.
QUALITY: 새추레이션 알고리즘의 처리 품질을 설정함. 값이 높을수록 계산 정밀도 증가함.
TYPE: 서로 다른 새추레이션 알고리즘 중 선택함. 타입에 따라 왜곡 성향과 질감 달라짐.
MIX: 원본 신호와 새추레이션이 적용된 신호의 비율을 조절함.
ROLL-OFF: 새추레이션 후의 고주파를 감쇠함. 30,000Hz부터 20Hz까지 범위에서 고역을 점진적으로 줄임.
OUTPUT: 효과 적용 후 최종 출력 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>Cabinet</strong></p>

<p style="margin: 0 0 1em 0;">스피커 캐비닛 에뮬레이션 선택 항목임.</p>

<img src="/images/11.-Channel-FX-3.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">Twin, 1960B, Solo412, AC30, ES212 중 선택 가능함.</p>


### <strong>Dynamics</strong>

<p style="margin: 0 0 1em 0;"><strong>Compressor</strong></p>

<p style="margin: 0 0 1em 0;">입력 신호가 설정한 임계값을 넘을 때 레벨을 줄이는 효과임. 큰 소리를 눌러 다이내믹을 정리하고, 왜곡과 피크를 완화함. 어택과 릴리즈 설정에 따라 소리의 어택감과 바디감이 달라짐.</p>

<img src="/images/11.-Channel-FX-4.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">ATTACK: 압축이 시작되는 속도를 설정함. 값이 낮을수록 어택이 빠르게 눌리고, 높을수록 어택이 살아남음. 0~300ms.</p>

<p style="margin: 0 0 1em 0;">RELEASE: 압축이 풀리는 속도를 설정함. 길게 설정하면 바디감이 줄고, 짧게 설정하면 어택이 더 강조됨. 0~3000ms.</p>

<p style="margin: 0 0 1em 0;">CHARACTER: 불필요하게 튀는 피크를 억제해 음색을 부드럽게 만듦.</p>

<p style="margin: 0 0 1em 0;">THRESHOLD: 압축이 시작되는 기준 레벨임. 낮을수록 더 많은 신호가 압축됨.</p>

<p style="margin: 0 0 1em 0;">RATIO: 임계값을 넘은 신호가 얼마나 눌리는지 결정함. 3:1은 3dB 초과 입력이 1dB만 증가해 출력됨을 의미함. 1:1~60:1.</p>

<p style="margin: 0 0 1em 0;">SOFT KNEE: 임계값 근처에서 압축을 점진적으로 적용함. 전환이 더 부드러워짐.</p>

<p style="margin: 0 0 1em 0;">MAKE-UP: 압축으로 줄어든 레벨을 자동으로 보상함.</p>

<p style="margin: 0 0 1em 0;">SIDE-CHAIN: 외부 또는 내부 신호를 기준으로 압축을 트리거함. 켜고 끌 수 있음.</p>

<p style="margin: 0 0 1em 0;"><strong>Expander</strong></p>

<p style="margin: 0 0 1em 0;">다이내믹 범위를 확장하는 효과임. 작은 소리는 더 줄이고, 큰 소리는 상대적으로 강조함. 컴프레서와 반대 개념이며, 낮은 비율에서는 노이즈 억제 용도로 사용됨.</p>

<img src="/images/11.-Channel-FX-5.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">ATTACK: 신호가 임계값을 벗어난 뒤 설정된 비율로 처리되는 속도를 설정함.
RELEASE: 신호가 다시 임계값 아래로 내려갔을 때 효과가 풀리는 속도를 설정함.
CHARACTER: 불쾌한 피크를 억제해 음색을 정리함.
THRESHOLD: 익스팬더가 작동하기 시작하는 기준 레벨임.
RATIO: 임계값 이하 신호가 줄어드는 정도를 설정함. 높은 값은 게이트에 가까운 동작을 함.
SOFT KNEE: 임계값 근처에서 효과 적용을 완만하게 함.
MAKE-UP: 처리 후 신호 레벨을 보정함.
SIDE-CHAIN: 사이드 체인 사용 여부를 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>Limiter</strong></p>

<p style="margin: 0 0 1em 0;">신호가 설정된 상한을 넘지 않도록 제한하면서 평균 레벨을 높이는 효과임. 전체 볼륨을 키우는 용도로 사용되며, 과도하면 소리가 평평해질 수 있음.</p>

<img src="/images/11.-Channel-FX-6.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">ATTACK: 리미팅이 적용되는 속도를 설정함. 0~300ms.
RELEASE: 제한된 신호가 원래 레벨로 돌아오는 속도를 설정함. 0~3000ms.
CHARACTER: 피크를 부드럽게 처리해 거친 느낌을 줄임.
THRESHOLD: 리미터가 작동하는 기준 레벨임. 낮출수록 평균 레벨이 올라감.
SOFT KNEE: 리미팅 전환을 부드럽게 함.
MAKE-UP: 처리로 줄어든 볼륨을 보상함.
SIDE-CHAIN: 사이드 체인 사용 여부를 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>Noisegate</strong></p>

<p style="margin: 0 0 1em 0;">일정 레벨 이하의 신호를 억제하는 효과임. 연주하지 않는 구간의 노이즈를 정리하는 데 사용함.</p>

<img src="/images/11.-Channel-FX-7.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">ATTACK: 임계값 도달 후 게이트가 열리는 속도를 설정함.
RELEASE: 신호가 임계값 아래로 내려간 뒤 게이트가 닫히는 속도를 설정함.
CHARACTER: 게이트 동작 시 발생하는 거친 피크를 완화함.
THRESHOLD: 게이트가 닫히는 기준 레벨임.
SOFT KNEE: 게이트 개폐를 완만하게 만들어 자연스럽게 동작하게 함.
MAKE-UP: 출력 레벨을 보정함.
SIDE-CHAIN: 사이드 체인 사용 여부를 설정함.</p>


### <strong>Filters</strong>

<p style="margin: 0 0 1em 0;">
<strong>Bandpass</strong></p>

<img src="/images/11.-Channel-FX-8.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">설정한 두 주파수 사이의 신호만 남기고, 그 외 영역은 줄이는 필터임.
FREQUENCY: 필터가 중심으로 작동하는 주파수를 설정함.</p>

<p style="margin: 0 0 1em 0;">RESONANCE: 통과 대역의 폭과 가장자리 강조 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>Equalizer</strong></p>

<img src="/images/11.-Channel-FX-9.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">특정 주파수 대역을 줄이거나 증폭하는 필터임. 선택한 곡선 유형에 따라 특정 지점만 조정하거나, 기준 주파수 위·아래 전체를 처리함.</p>

<p style="margin: 0 0 1em 0;">TYPE: 사용할 필터 곡선 유형을 선택함.
GAIN: 선택한 대역의 레벨을 -24dB에서 +24dB까지 조절함.
FREQUENCY: 조정할 주파수 위치를 설정함.
Q-FACTOR: 영향을 받는 주파수 대역의 폭을 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>HighPass 6dB, 12dB, 24dB, 36dB</strong>
설정한 주파수 이하의 저역을 줄이고, 그 이상의 신호만 통과시키는 필터임. dB 값이 높을수록 컷이 가파르게 작동함.</p>

<img src="/images/11.-Channel-FX-13.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">CUTOFF: 저역이 차단되기 시작하는 주파수를 설정함.
RESONANCE: 컷오프 지점의 강조 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>LowPass 6dB, 12dB, 24dB, 36dB</strong>
설정한 주파수 이상의 고역을 줄이고, 그 이하의 신호만 통과시키는 필터임. dB 값이 높을수록 컷이 가파르게 작동함.</p>

<p style="margin: 0 0 1em 0;"><strong>Notch 12dB, 24dB, 36dB</strong>
특정 주파수만 좁게 깎아내는 필터임. 공진이나 특정 노이즈 제거에 사용함.</p>

RESONANCE: 영향을 받는 주파수 대역의 폭을 설정함.</p>


### Modulation

<p style="margin: 0 0 1em 0;"><strong>Auto Gain</strong></p>

<p style="margin: 0 0 1em 0;">스테레오 이미지를 크게 확장하는 변조 효과임. 좌우 신호에 LFO를 적용해 움직임과 공간감을 만듦.</p>

<img src="/images/11.-Channel-FX-26.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">AMOUNT: LFO 변조가 신호에 적용되는 강도를 설정함.</p>

<p style="margin: 0 0 1em 0;">SPREAD: 변조가 좌우 스테레오로 벌어지는 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;">PHASE: 좌우 채널 간 LFO 위상 차이를 설정함. 180도는 좌우가 반대로 움직임.</p>

<p style="margin: 0 0 1em 0;">RATE: LFO 속도를 Hz 단위로 설정함.</p>

<p style="margin: 0 0 1em 0;">SYNC: LFO 속도를 템포 기준 음표 값에 맞춤. 일반, 점, 셋잇단음표 선택 가능함. 예로 N16은 16분음표 기준임.</p>

<p style="margin: 0 0 1em 0;">SHAPE: LFO 파형을 선택함. 총 8가지 파형 제공함.</p>

<p style="margin: 0 0 1em 0;"><strong>Chorus - Flanger</strong></p>

<p style="margin: 0 0 1em 0;">코러스와 플랜저를 결합한 변조 효과임. 신호를 두 개의 딜레이 라인으로 나눠 시간차와 간섭을 만들어 움직임을 형성함.</p>

<img src="/images/11.-Channel-FX-27.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">DELAY: 두 딜레이 라인 사이의 시간 차이를 설정함.</p>

<p style="margin: 0 0 1em 0;">DEPTH: 플랜저 변조 깊이를 설정함. LFO 진폭에 영향을 줌.</p>

<p style="margin: 0 0 1em 0;">RATE: 변조 속도를 설정함.</p>

<p style="margin: 0 0 1em 0;">SPREAD: 효과가 스테레오로 퍼지는 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;">SHAPE: LFO 파형을 선택함.</p>

<p style="margin: 0 0 1em 0;">FEEDBACK: 효과 신호 일부를 다시 입력으로 보내 링잉을 추가함.</p>

<p style="margin: 0 0 1em 0;">DAMPING: 효과 신호의 고역을 줄임. 로우패스 필터로 작동함.</p>

<p style="margin: 0 0 1em 0;">DRY: 원본 신호 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">WET: 효과 신호 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">QUALITY: 알고리즘 처리 품질을 설정함.</p>

<p style="margin: 0 0 1em 0;">SYNC: 변조 속도를 템포 기준으로 동기화함.</p>

<p style="margin: 0 0 1em 0;">INVERT FEEDBACK: 피드백 신호의 위상을 반전시킴.</p>

<p style="margin: 0 0 1em 0;">INVERT WET: 웨트 신호의 위상을 반전시킴.</p>

<p style="margin: 0 0 1em 0;"><strong>Phaser</strong></p>

<p style="margin: 0 0 1em 0;">신호 일부를 여러 개의 올패스 필터로 처리해 위상 변화를 만들고, 원본과 섞어 특유의 스윕 효과를 생성함.</p>

<img src="/images/11.-Channel-FX-28.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">STAGES: 신호가 통과하는 필터 단계 수를 설정함. 값이 높을수록 효과가 진해짐.</p>

<p style="margin: 0 0 1em 0;">LOWER FREQUENCY: 스윕이 시작되는 저역 한계를 설정함. 20~20,000Hz.</p>

<p style="margin: 0 0 1em 0;">UPPER FREQUENCY: 스윕이 도달하는 고역 한계를 설정함. 20~20,000Hz.</p>

<p style="margin: 0 0 1em 0;">RATE: 페이저 스윕 속도를 설정함.</p>

<p style="margin: 0 0 1em 0;">SYNC: 스윕 속도를 템포에 동기화함.</p>

<p style="margin: 0 0 1em 0;">SPREAD: 효과가 스테레오로 벌어지는 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;">SHAPE: LFO 파형을 선택함.</p>

<p style="margin: 0 0 1em 0;">FEEDBACK: 효과 신호를 다시 입력으로 보내 강조함.</p>

<p style="margin: 0 0 1em 0;">INVERT FEEDBACK: 피드백 위상을 반전시킴.</p>

<p style="margin: 0 0 1em 0;">DAMPING: 효과 신호의 고역을 줄임.</p>

<p style="margin: 0 0 1em 0;">DRY: 원본 신호 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">WET: 효과 신호 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">INVERT WET: 웨트 신호 위상을 반전시킴.</p>

<p style="margin: 0 0 1em 0;"><strong>Rotary</strong></p>

<p style="margin: 0 0 1em 0;">회전 스피커에서 발생하는 도플러 효과를 재현함. 회전 속도에 따라 코러스부터 트레몰로까지 변화함.</p>

<img src="/images/11.-Channel-FX-29.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">CHARACTER: 스피커 특성을 반영해 음색과 필터 반응을 조절함.</p>

<p style="margin: 0 0 1em 0;">CROSS-OVER: 저음과 고음 스피커가 나뉘는 주파수를 설정함.</p>

<p style="margin: 0 0 1em 0;">SPEED: 스피커 회전 속도를 설정함.</p>

<p style="margin: 0 0 1em 0;">DEPTH: 피치 변조 깊이를 설정함. 캐비닛 크기에 해당함.</p>

<p style="margin: 0 0 1em 0;">LOW/HIGH MIC ANGLE: 저음/고음 스피커 마이크 각도를 설정함.</p>

<p style="margin: 0 0 1em 0;">HIGH/LOW THROB: 고음/저음 스피커의 볼륨 변조량을 설정함.</p>

<p style="margin: 0 0 1em 0;">DRIVE: 스피커 구동에 따른 왜곡 양을 설정함.</p>


### Reverb

<p style="margin: 0 0 1em 0;">Plate Reverb</p>

<img src="/images/11.-Channel-FX-30.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">금속판 특유의 반사로 만들어진 잔향을 재현하는 리버브임. 공간감보다는 밀도 있고 균일한 잔향이 특징임.</p>

<p style="margin: 0 0 1em 0;">DECAY: 잔향이 사라질 때까지 유지되는 시간을 설정함.</p>

<p style="margin: 0 0 1em 0;">DAMPING: 잔향의 고역을 줄임. 로우패스 필터로 작동하며 감쇠량을 조절함.</p>

<p style="margin: 0 0 1em 0;">DRY/WET: 원본 신호와 리버브 신호의 비율을 설정함.</p>

<p style="margin: 0 0 1em 0;"><strong>Room Reverb</strong></p>

<p style="margin: 0 0 1em 0;">실내 공간에서 반사되는 잔향을 시뮬레이션한 리버브임. 플레이트보다 입체적인 공간감을 형성함.</p>

<img src="/images/11.-Channel-FX-31.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">PREDELAY: 원본 소리 이후 리버브가 시작되기까지의 지연 시간을 설정함. ms.</p>

<p style="margin: 0 0 1em 0;">ER SIZE: 초기 반사음이 퍼지는 범위를 설정함.</p>

<p style="margin: 0 0 1em 0;">ER CROSS: 초기 반사음이 스테레오 공간에 퍼지는 정도를 설정함.</p>

<p style="margin: 0 0 1em 0;">DECAY: 잔향이 감쇠되는 전체 시간을 설정함. 초.</p>

<p style="margin: 0 0 1em 0;">SIZE: 가상 공간의 크기를 설정함.</p>

<p style="margin: 0 0 1em 0;">LOWPASS: 리버브 신호의 고역을 차단함. Hz.</p>

<p style="margin: 0 0 1em 0;">DAMPING: 잔향 피드백 경로의 고주파를 줄임. Hz.</p>

<p style="margin: 0 0 1em 0;">EARLY: 초기 반사음의 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">LATE: 후기 잔향의 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">DRY: 원본 신호 레벨을 설정함.</p>

<p style="margin: 0 0 1em 0;">WET: 리버브 신호 레벨을 설정함.</p>


### Utility

<p style="margin: 0 0 1em 0;">
<strong>Invert</strong></p>

<img src="/images/11.-Channel-FX-32.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">신호의 위상이나 채널 배치를 빠르게 조정하는 도구임.
PHASE INVERT: 입력 신호의 위상을 반전시킴.
CHANNEL SWAP: 좌우 스테레오 채널을 서로 바꿈.
</p>

<p style="margin: 0 0 1em 0;"><strong>Power Pan</strong></p>

<img src="/images/11.-Channel-FX-33.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;">스테레오 이미지를 손실 없이 제어하는 패닝 도구임. 일반 밸런스 페이더와 달리, 폭을 줄여도 한쪽 신호가 사라지지 않음.
CENTER: 패닝 중심 위치를 퍼센트 %로 표시함.
WIDTH: 스테레오 폭을 설정함. 100%는 원본 스테레오 폭 그대로임.
BALANCE: 지정된 패닝 파라미터를 퍼센트 % 기준으로 적용함.
GAIN: 밸런스 조절 시 적용되는 게인을 설정함. 일반적으로 밸런스는 게인 변화로 보정됨.
PHASE INVERT: 입력 신호의 위상을 반전시킴.
PAN LAW: 패닝 위치에 따른 레벨 보정을 설정함. 중앙 패닝 시 좌우 신호 합산으로 음량이 커지는 것을 방지함. 0~6dB 범위에서 조절 가능하며, 0dB는 정전력 패닝을 끔.</p>

<p style="margin: 0 0 1em 0;"><strong>Sample Delay</strong>
신호에 샘플 단위의 지연을 추가하는 도구임. 위상 보정이나 미세한 타이밍 조절에 사용함.</p>
