<p style="margin: 0 0 1em 0;">여기서는 Yamaha CFX를 기준으로 하나의 피아노를 어떤 위치에서, 어떤 마이크로 담았는지를 설명함.
각 피아노마다 마이크 배치는 조금씩 다르고, 자세한 내용은 각 악기별 매뉴얼에서 따로 확인할 수 있음. 
Stage A, Stage B에 수록된 여러 그랜드와 업라이트가 같은 방식으로 녹음돼 있음.</p>

<p style="margin: 0 0 1em 0;"><strong>Close and Mid Microphones</strong></p>

<p style="margin: 0 0 1em 0;">연주자 쪽에서 듣는 소리, 마치 그랜드 피아노 앞에 앉아 있는 느낌을 주는 역할을 하는 가까운 마이크와 중간 거리 마이크</p>

<p style="margin: 0 0 1em 0;"><strong>Close 1, MK 4</strong></p>

<p style="margin: 0 0 1em 0;">Schoeps MK 4 두 개를 ORTF 방식으로 배치함. 해머 근처와 현 바로 위에 최대한 가깝게 둔 세팅임. 요즘 대중음악에서 쓰이는 선명하고 현대적인 피아노 소리를 만들 때 사용하는 방식임.</p>

<p style="margin: 0 0 1em 0;"><strong>Close 2, </strong>SF 24</p>

<p style="margin: 0 0 1em 0;">Royer SF 24 리본 마이크를 블룸라인 방식으로 구성함. 연주자 위치 쪽에 놓여 있어서 반응이 빠르고 자연스러운 소리가 남음. MK 4와는 성격이 다른 클로즈 사운드임.</p>

<p style="margin: 0 0 1em 0;"><strong>Close 3, </strong>M-149</p>

<p style="margin: 0 0 1em 0;">이 세팅은 Steinway & Sons D에만 해당됨. Neumann M-149 튜브 마이크 두 개를 ORTF 방식으로 배치함. 튜브 마이크 특유의 따뜻함과 밀도가 살아 있는 소리를 담는 용도임.</p>

<p style="margin: 0 0 1em 0;"><strong>Mid 1, </strong>MKH 800</p>

<p style="margin: 0 0 1em 0;">Sennheiser MKH 800 두 개를 ORTF 방식으로 배치함. 따뜻하면서도 밝고, 폭이 넓은 소리가 특징임. 이 세트에서는 피아노가 정확히 중앙에 놓임.</p>

<p style="margin: 0 0 1em 0;"><strong>Mid 2, </strong>MKH 8040</p>

<p style="margin: 0 0 1em 0;">같은 ORTF 배치지만 MKH 8040을 사용함. Mid 1보다 소리가 더 단단하고 직접적으로 느껴짐. 마찬가지로 피아노는 중앙에 위치함.</p>

<img src="/images/10.-Microphone-Positions-1.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;"><strong>Room Microphones </strong></p>

<img src="/images/10.-Microphone-Positions-2.png" alt="" style="max-width: 100%; height: auto; border-radius: 4px; display: block; margin: 10px 0;" />

<p style="margin: 0 0 1em 0;"><strong>05 Main/Room Mic – Decca Tree Stereo (Main), DPA 4006</strong></p>

<p style="margin: 0 0 1em 0;"><strong>06 Main/Room Mic – Decca Tree Center, Mono (Main-C), DPA 4006</strong></p>

<p style="margin: 0 0 1em 0;">DPA 4006을 사용한 데카 트리 구성임. Stage A 공간의 디테일까지 왜곡 없이 담기 위해 아주 중립적인 마이크를 사용함. 스테레오에서는 피아노의 위치가 또렷하게 느껴지고, 서라운드 구성에서는 프런트 L/R 스피커 소스로 사용됨.</p>

<p style="margin: 0 0 1em 0;"><strong>07 Main Surround (Surround), DPA 4011</strong></p>

<p style="margin: 0 0 1em 0;">DPA 4011을 카디오이드 마이크로 넓은 AB 배치로 구성한 서라운드 메인 마이크임. 스테레오에서는 공간감을 넓혀주는 보조 마이크로 쓰고, 서라운드에서는 리어 스피커 신호로 사용함.</p>

<p style="margin: 0 0 1em 0;"><strong>08 High Stereo 3D (High), AKG C414</strong></p>

<p style="margin: 0 0 1em 0;"><strong>09 High Surround 3D (High-Surround), AKGC 414</strong></p>

<p style="margin: 0 0 1em 0;">하이마이크로 천장 반사를 포착하기 위해 높은 위치에서 AB 방식으로 AKG C414를 배치함. 스테레오에서는 메인 마이크의 대안으로 쓰거나 더 퍼진 소리를 섞는 용도임. 서라운드나 Auro-3D, Dolby Atmos 환경에서는 천장 스피커 신호로 보내는 마이크임.</p>
